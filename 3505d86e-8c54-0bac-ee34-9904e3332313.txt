<br/>        傅昼景就坐在自己的老板椅上，抬眸直勾勾地盯着季思妍，若有所思地继续皱着眉头。

<br/>        来电显示上是“大哥”。

<br/>        昼景落下手机，万一有什么重要电话，他岂不是会错过？

<br/>        童筱颖只觉这个男人的背影让她觉得有点儿眼熟，但她现在赶时间，无暇顾及这些身外之事，于是跟在了前面这群簇拥的男男女女身后，和那个为首的男人一起进了地利置业的大门。
<br/>
<br/>        耳边，毕恭毕敬的问候“傅总好”，此伏彼起地响起。
<br/>
<br/>        童筱颖见自己已经进了大厦大门，刚准备撤离这支“队伍”，却无意间看到了那个为首的男人侧面。
<br/>
<br/>        居然是他！
<br/>
<br/>        一想到这个那晚被她救了的男人，每次偶遇他，他都会对她做出那些令人脸红心跳的亲密行为，童筱颖几乎是条件反射，本能地转过身去背对着那个男人，试图逃避他。
<br/>
<br/>        傅夜沉走着走着，脑海里忽然闪过“漫雪”的身影，便骤然停下了脚步。
<br/>
<br/>        他一停，在场所有的人都跟着驻足。
<br/>
<br/>        当他回过身去，所有人都静止不动的时候，只有那个令他熟悉得不能再熟悉的背影却在逐渐离他远去。

<br/>        傅夜沉不由地身心一怔，连心跳似乎都慢了半拍。
<br/>
<br/>        好像她这一走，就会永远地失去了一样……

<br/>        一声透着磁性的命令，在整个大厅里响起。

<br/>        明明他紧张着那个快要从自己眼皮子底下逃走的女人，但是，他俊脸上的波澜不惊，让外人看不出他有丝毫内心波动。

<br/>        童筱颖听到这声强势的命令时，还真就心虚地停下了脚步。

<br/>        难道……那个男人认出她呢？

<br/>        傅夜沉下意识地看向正朝自己走过来的弟弟傅昼景，当他再次回神去看那个女孩时，那里早已没有了那个女孩的身影。
