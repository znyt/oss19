　　萧逸终于想起来八龙焚火炉的这般剧烈抖动在哪见过。
<br/>　　正是在东域开启八龙焚火炉的第一道封印之时。
<br/>　　而毫无疑问，现今的这般抖动，来自于第二道封印的冲击。
<br/>　　萧逸边放出感知，边思索着。
<br/>　　半晌，明白了过来。
<br/>　　金光力量，并未对八龙焚火炉产生半分影响。
<br/>　　金光力量，增幅武器威力时，乃是金光力量被武器吸收。
<br/>　　可对于八龙焚火炉，却并未被火炉本身吸收。
<br/>　　也就是说，这些金光力量，相对于八龙焚火炉来说，乃是一股独立的力量。
<br/>　　而这股力量不断涌入炉内火焰空间，甚至装满了整个空间。
<br/>　　这时，再往八龙焚火炉内吸摄入金光力量，无异于‘挤’着这片空间，也便是变相地冲击着第二道封印。
<br/>　　此时，八龙焚火炉，仍旧不断吸摄着金光力量。
<br/>　　不，或许准确来说，是萧逸吸摄着这些金光力量，强行涌入火炉内。
<br/>　　十数分钟后。
<br/>　　嘭…
<br/>　　半空中，八龙焚火炉一声宛若炸炉般的剧烈声响响过。
<br/>　　一股火焰，从八龙焚火炉中，宛如吐息一般，往四面八方溢散。
<br/>　　萧逸脸色一变，连忙仗剑护于身前。
<br/>　　上次八龙焚火炉解开第一道封印时，那股火焰的爆发可是吓了他一跳。
<br/>　　这一次，他自然早便反应过来。
<br/>　　30道星光长河护于身前。
<br/>　　但在这股火焰的冲击下，星光长河被不断焚烧溃散。
<br/>　　萧逸只得不断耗费元力，不断操控星光降临，以抵挡朝他溢散而来的火焰。
<br/>　　半晌，火焰的溢散停止。
<br/>　　萧逸松了口气，散去星光。
<br/>　　萧逸感知了一番八龙焚火炉，脸色一惊。
<br/>　　第二道封印已经被冲破。
<br/>　　而原本的火焰空间，明显范围大了一倍。
<br/>　　“原来如此，折叠空间吗？”萧逸自语一声。
<br/>　　据剑姬前辈说，八龙焚火炉内，封印着八道龙炎吐息，亦有八道封印。
<br/>　　虽然不知道剑姬前辈哪里知晓的这一关于八龙焚火炉的信息。
<br/>　　萧逸也不确定这些火焰是否龙炎。
<br/>　　但，里头有八道强悍封印，这一点萧逸倒是能感知到。
<br/>　　从刚开始得到八龙焚火炉时，那不过是个普通的炼药炉。
<br/>　　直至解开第一道封印，才忽然爆发强悍火焰，且有了那片熊熊火焰的火焰空间。
<br/>　　以前萧逸就感知过，剩下的七道封印，皆在这片空间中。
<br/>　　现今，第二道封印已解开。
<br/>　　而那片火焰空间则大了一倍。
<br/>　　很显然，便是所谓的折叠空间。
<br/>　　八片空间，原本封于八龙焚火炉内。
<br/>　　八道封印，每解开一道，空间恢复，这便有了此刻的火焰空间大了一倍。
<br/>　　只不过，无端端地，封印这些空间做什么？
<br/>　　萧逸皱眉思索着。
<br/>　　而此时，炉内火焰空间增加了一倍，金光力量自然也能继续涌入其内。
<br/>　　萧逸思索着，半晌，再次将感知放入八龙焚火炉内。
<br/>　　在他得的感知中，源源不断的金光力量涌入，填充着这片更加巨大的空间。
<br/>　　而同时，熊熊火焰所产生的温度，似乎比之前高了些。
<br/>　　“是火焰的威力增强了。”萧逸感知着，也思索着。
<br/>　　现在空间内的火焰，似乎比之前更强了。
<br/>　　萧逸注视着这些火焰，半晌，还未等他想明白过来，目光却忽然在火焰中的一物上定格。
<br/>　　“那是…”
<br/>　　萧逸皱着眉，他以前曾经细致感知过这片火焰空间。
<br/>　　空间内，绝对没有任何东西，除了这些火焰外。
<br/>　　而现在，却多了一物。
<br/>　　萧逸看真切些，待得看清楚时，瞳孔陡然一缩。
<br/>　　他终于看清楚那是什么东西。
<br/>　　那是一只黝黑的手臂，手臂上，布满鳞片。
<br/>　　而手掌上，一只只锋利的爪子，散发着惊人寒光。
<br/>　　显然，这不是人类武者的手。
<br/>　　而是妖兽的手，这是一只兽爪。
<br/>　　兽爪，置身于熊熊火焰中，未损分毫，反倒在火焰映衬下，散发出赤红光芒。
<br/>　　“出。”萧逸低喝一声。
<br/>　　直觉告诉他，他能通过八龙焚火炉，拿出这只兽爪。
<br/>　　几乎是他话音落下的瞬间。
<br/>　　八龙焚火炉上，一道赤红光芒一闪而出，稳稳地覆于他掌中。
<br/>　　“嗯？”萧逸眉头紧皱着。
<br/>　　他明显感觉到，此刻他的手掌一阵火热炙痛。
<br/>　　这种痛，不同于正常的火焰灼烧感。
<br/>　　反倒更像是来自于皮肉之下的深层疼痛。
<br/>　　数秒后，赤红光芒消失。
<br/>　　取而代之的是，萧逸的手掌，被一层黝黑的鳞片所覆盖。
<br/>　　“嘶，这…”萧逸倒吸一口凉气，随后脸色大变。
<br/>　　他的手掌，手臂，完全变了。
<br/>　　与刚才火焰空间内的那只兽爪一模一样。
<br/>　　萧逸着实被吓了一跳，但他也反应极快，心头一声低喝，“散”。
<br/>　　他的手掌、手臂，恢复了正常。
<br/>　　再次感知火焰空间，那只兽爪，安静地置身火焰之内。
<br/>　　“呼。”萧逸松了口气。
<br/>　　若是他的手，正的变成了兽爪，那当真是让人难以接受之事。
<br/>　　即便他向来不在乎这些所谓的外表。
<br/>　　“出。”萧逸再次低喝一声。
<br/>　　一道赤红光芒闪烁而过，兽爪，再次覆盖于手臂皮肤。
<br/>　　萧逸甩了甩手臂，倒也没觉有什么不适之处。
<br/>　　反倒是一股惊人的力量，充斥其中。
<br/>　　萧逸只觉整只手臂，充满了使之不尽的力量。
<br/>　　握了握拳头，指尖那锋利的爪子，寒光凛然。
<br/>　　萧逸双眼一眯，取出一把下品圣器的剑类武器。
<br/>　　下一秒。
<br/>　　手中之剑，狠狠劈向自己的另一只手臂。
<br/>　　锵…
<br/>　　锋利的剑身，轰击到手臂上，发出一声锵鸣。
<br/>　　手臂丝毫未损，反倒是剑身缺了个小口。
<br/>　　萧逸将剑握于另一只手掌上。
<br/>　　散发着寒光的爪子，狠狠一捏。
<br/>　　咔…咔…咔…
<br/>　　整把利剑，被轻松捏成齑粉。
<br/>　　“好强。”萧逸脸色一惊。
<br/>　　下品圣器，劈于手臂，丝毫无损。
<br/>　　下品圣器，握于手掌，宛若一把废铜烂铁，瞬间捏碎。
<br/>　　萧逸刚要继续打量这只手臂。
<br/>　　忽然，眼前八龙焚火炉上，再次发出抖动。
<br/>　　萧逸感知了一眼。
<br/>　　里头火焰空间，已被金光力量所填满。
<br/>　　现今，金光力量的继续涌入，将等同继续冲击下一道封印。
<br/>　　“第三道封印。”萧逸嘴角咧过一道期待笑容。
