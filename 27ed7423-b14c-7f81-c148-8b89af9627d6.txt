　　“我记得这个味道，以前应该是经常吃。”陈悦嚼着一口面发呆。
<br/>　　“那就好，说明咱没来错地方。”
<br/>　　“陈姐姐你在贵州呆过？”乌宏志好奇问。
<br/>　　“她是毕节人，这次我们就是来她老家办事的。”不能陈悦开口，谢雨晴接过去说道。
<br/>　　乌宏志还想问下去，被谢雨晴岔开话题，拉着他讨论起旅游来。
<br/>　　谢雨晴难得修个假，而且乌宏志盛情挽留，想让她在贵阳玩几天，但叶少阳着急带着陈悦去办正事，于是商量好，她留在贵阳玩，自己陪陈悦去毕节，等事情办完再来贵阳找她，到时候好好陪她呆几天。
<br/>　　第二天一大早，叶少阳、陈悦和瓜瓜出发，前往毕节。从贵阳过去，高铁就二十分钟，眨眼的工夫就到了，下了火车，叶少阳在滴滴上按照陈悦身份证上的地址叫了辆车，上车后，陈悦一直盯着车窗外面，似乎想要找寻记忆。
<br/>　　“娘，想起来没有？”瓜瓜在一边悄声问道。
<br/>　　“似曾相识。”陈悦说道。
<br/>　　汽车开出市区，进了盘山公路，翻山越岭的，而且这公路边上都没有护栏，有时候一个急转弯，司机是淡定的很，叶少阳坐在副驾驶，一路上心惊胆战的，好在山野间景色不错，分担了注意。
<br/>　　汽车在山里开了有一个小时，来到一个山谷小镇里。
<br/>　　司机问他们具体地址，表示按照他给的地址导航，只能到达这里，之后就不知道往哪里走了。
<br/>　　三人只好下车。
<br/>　　叶少阳通过平台付钱，看到车费有一百多，心疼了一下。
<br/>　　“我说，你难道一点都想不起来，看看这个镇子，认识吗？”
<br/>　　陈悦站在街头，观察了半天，喃喃道：“我记得这个地方，但也只是依稀记得……支离破碎的，串不到一起去。”
<br/>　　“就是说，你不知道该往哪儿走是吧？”
<br/>　　陈悦摇头。
<br/>　　没办法，叶少阳只好去附近超市买水，顺便拿着地址打听。
<br/>　　老板娘是个老大娘，说的是当地方言，在叶少阳听来就像外语一样，半天没听懂，但陈悦听懂了，用略微生涩的方言跟她交流了几句，然后招呼叶少阳离去。
<br/>　　“顺着这条路走三个路口，然后上山，就是了。”陈悦指着一条路说道。
<br/>　　“你会当地话？”
<br/>　　“我……之前完全没有印象，但是听她说，猛然间就记起来一些。”
<br/>　　叶少阳满意说道：“你接触到以往长期接触的东西，会慢慢勾起回忆的。”
<br/>　　山路是铺着石子的小路，下雨天有点泥泞，可能也是下雨的缘故，路上一个人都没有。陈悦走走停停，不断观察附近，若有所思。
<br/>　　走到半山腰一块平地上，路边有一个草庵，里面摆着几个石墩子，看上去像个茶棚，地上有不少瓜子皮和烟蒂，叶少阳猜测，八成是住在山上附近的人修来供上下山的时候歇脚的，刚好看陈悦也有点累，于是招呼她过去坐一会。
<br/>　　“怎么了，我怎么看你有点紧张，你是不是想起什么了？”叶少阳盯着陈悦的脸问道。
<br/>　　陈悦摇摇头，嗫嚅道：“我有点怕。”
<br/>　　“怕什么？”
<br/>　　“怕面对自己的身份吧。”陈悦吐出一口气，笑了笑，道，“我刚一路上来，熟悉的感觉越来越强烈，我们没有走错地方，我很期待知道自己是谁，但是……又有点怕，万一我身份不好，或者什么之类的，总之有些忐忑吧。”
<br/>　　叶少阳想了一下道：“我好像能理解，不过不管怎样，那都是你自己。”
<br/>　　“是啊，所以我只是有点担心，还是很期待的。”
<br/>　　正说着话，有个人打着伞从他们走过的那条山路上来了，是一个农夫打扮的汉子，嘴上叼着一根烟，哼着歌，从这边走过，转头冲叶少阳点了点头，用僵硬的普通话说道：“小伙子，你有打火机没有，我打火机被雨水打湿了。”
<br/>　　“没有。对了大爷，你知道胡家村三组六户，在什么地方？”
<br/>　　“三组哦。你们再往前走，到了岔路口转右，一直走到头就是了。”汉子站着看了他一眼，“那地方偏得很哦，你们来这里做什么？”
<br/>　　目光从叶少阳脸上移到陈悦脸上，眉头微微皱了一下，盯着她不动了。
<br/>　　“你认识她？”叶少阳忙问。
<br/>　　汉子快走两步，歪着头打量她，陈悦也抬头看了他一眼。
<br/>　　汉子面色大变，道：“小尼姑？”
<br/>　　“什么？”
<br/>　　汉子猛然想起什么，脸上的肌肉突然抽搐起来，急忙后退了几步，盯着陈悦，惊恐地说道：“你、你是小尼姑，你不是死了吗，你你你……”
<br/>　　话说到一半，跌跌撞撞地逃了，叶少阳追上去叫了半天汉子还是跑了。
<br/>　　叶少阳转身回到草庵，见陈悦面色苍白，有些难看，安慰道：“他大概认错了，你有头发，怎么会是尼姑，再说，我确定你是活人。”
<br/>　　陈悦什么都没说，但神色一直凝重。瓜瓜拉住她的手安慰了好一会。
<br/>　　“我们走吧。”
<br/>　　陈悦起身，拉着瓜瓜往前走。
<br/>　　叶少阳跟在后面，想起之前那个汉子的话，心中也是感到十分诡异。
<br/>　　前行不远，果然有个岔口，右边是上山，走不多远，就能看到一些依山而建的房屋，没有想象中那么破旧。上上的路也是水泥的，因为下雨，路边没什么人，三人经过了一间小商店，叶少阳跟陈悦商量了一下，一起进去买东西，如果她真是这地方的人，店主肯定能认出她来。
<br/>　　陈悦咬着嘴唇，不太情愿地跟了进去。
<br/>　　看店的是个中年男子，正捧着一碗饭在吃，一边在柜台前面津津有味地看着电视。
<br/>　　叶少阳随便买了点东西，过去结账，老板看电视入迷，瞟了他们一眼，随口问道：“你们是去农家乐的吧，走错路了，前面那个路口往左边才是，这条路就上山了。”
<br/>　　叶少阳道：“不是，我这同学家就是这个村的，我送她回家来的。”
<br/>　　老板听他这么说，这才抬头看了陈悦一眼，愣了一下，笑道：“仙姑啊，好些日子没……”话没说完，似乎猛然想到什么，一下子站起来，死死盯着陈悦，“你、你……”
