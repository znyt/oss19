叶翎侧眸，挑着一双勾人的水眸看着顾夜瑾的俊脸，“有你这样做哥哥的么，我什么时候有男人了，你怎么可以这样造谣？”
<br/>
<br/>        顾夜瑾勾了一下薄唇，目光锁定着她这张明媚动人的小脸，“我有没有造谣，你有没有男人，你比任何人都清楚。”
<br/>
<br/>        叶翎颤了一下蒲扇般的羽捷，“我已经忘了。”
<br/>
<br/>        顾夜瑾的黑眸往下落在了她娇艳欲滴的红唇上，薄唇勾出一道邪佞的弧线，“那要不要我帮你回忆一下？”
<br/>
<br/>        叶翎羽捷微颤，“你跟过来干什么，就跟我聊几句骚？”
<br/>
<br/>        顾夜瑾滚动了一下喉头，“我的生日礼物呢？”
<br/>
<br/>        “不好意思，我最近太忙了，没有时间给你准备生日礼物。”
<br/>
<br/>        她没有生日礼物。
<br/>
<br/>        顾夜瑾勾唇，低低笑了一声。
<br/>
<br/>        呵。
<br/>
<br/>        “你不开心了？刚才我看见包厢里有堆积如山的礼物，都是送给你的，你还差我一份礼物？不然你喜欢什么告诉我，我让我助理现在就去给你买。”
<br/>
<br/>        顾夜瑾俊美的眸色染上暗黑系的惺忪慵懒，他看着叶翎，“我喜欢什么你会不知道，如果你要我说出来那我现在就告诉你，晚上你到我房间里来，我等你。”
<br/>
<br/>        叶翎看了他几秒，“顾夜瑾，你缺女人么，为什么缠着我不放？”
<br/>
<br/>        “过去的事情我们都忘了，以后不要再提了好不好？我想好好的生活，我现在还很小，才20岁，人生刚开始，我不想毁在你手里，你从那些名媛里娶一房娇妻吧，我也会谈恋爱，遇到自己喜欢且喜欢我的男人结婚生子，我们都会过得很好，不是么？”
<br/>
<br/>        顾夜瑾看着她，缓缓将薄唇勾出了一道弧线，“死了这条心，只要我还在这个位置上，就没有男人敢对你动念头，也没有男人敢跟你结婚生子。”
<br/>
<br/>        “那就没得谈了，我走了。”
<br/>
<br/>        叶翎转身，直接去洗手间。
<br/>
<br/>        但是走到男士洗手间门口时，突然有一只大手探了出来，扣住她的纤臂直接将她给扯了进去。
<br/>
<br/>        叶翎撞进了一副精健而温暖的胸膛里，抬头一看，顾夜瑾那张俊美如玉的容颜在她的视线里放大了。
<br/>
<br/>        “你将我拉进男士洗手间干什么，顾夜瑾，你不会变态到让我看你尿尿吧？”叶翎迅速抵着他的胸膛想将他给推开。
<br/>
<br/>        顾夜瑾大手往下移，落在她水蛇一样的小蛮腰上，薄唇勾起一道不悦的弧线，“尿尿？这就是这两年你在娱乐圈学到的东西？”
<br/>
<br/>        “要不然尿尿该怎么说，我都忘了你喜欢夏妍妍，真不知道这位海城第一名媛会将尿尿说成多么高雅的词。”
<br/>
<br/>        顾夜瑾将她往怀里抱，薄唇擦在她茶色的卷发上，两年前她离开时还是直发，小姑娘一个，现在她已经是卷发，成为小女人了，她长大了。
<br/>
<br/>        这种意识让顾夜瑾收紧了健臂，“你怎么总是提妍妍，妍妍自然不像你这样…妖精会勾引男人。”
<br/>
<br/>        叶翎抬脚，用力的踹了一下他结实的小腿，“那你就去找夏妍妍啊，她冰清玉洁，你不要泡不到她就拿我泄欲，我看不起你！”
