　　第134章
<br/>　　三枚铜板落在地上，变成六瓣，断口整齐。
<br/>　　罗铁插剑还鞘，高昂着头颅，肆意打量着众人。
<br/>　　看着地上的铜板，本来闹哄哄的人群，在此刻安静了几分。
<br/>　　一剑三斩，能同时将铜板斩开，这样快的剑法，在场诸人自认都无法做到。
<br/>　　李善言乐意看到这一幕发生，罗铁的到来出乎他的预料，却也变相在幽州群匪面前露了露肌肉。
<br/>　　李善言可以陪着笑脸和大家客套，但他也能随时抬起头来，露出一口狰狞牙齿。
<br/>　　随意低头的人，往往也意味着可以随时把头昂起。
<br/>　　至于山贼，终究是不入流的。
<br/>　　程大雷四人此时已经准备开溜。
<br/>　　“程寨主，怎么要走了么？”李善言笑道。
<br/>　　“啊，山寨还有些事要处理，李军师放心，我这一颗心永远是幽州王的，但有一声令，水里水去，火里火来，一切我都听吩咐。”
<br/>　　罗铁打量程大雷一眼，趾高气昂道：“算你懂事。”
<br/>　　四人牵马准备离开，程大雷发现了人群中的齐德强，他兴奋的跳过去，大声道：“齐将，最近还好么，我可是蛮想念你的。”
<br/>　　齐德强对于程大雷这种莫名其妙的热情有些不太适应，但这次准备招揽幽州群匪当炮灰，他自然也需和程大雷虚伪客套一番。
<br/>　　“多谢程寨主挂心，我心里也一直挂念着程寨主。”
<br/>　　“嘿，我就知道是这样。”程大雷用力捶了捶齐德强的肩膀。
<br/>　　齐德强身边几名幽州兵将都看着这一幕，心道：“老齐和这山贼都关系这么好么！”
<br/>　　“哎，老齐这几位是谁啊，你怎么不给我介绍介绍。”程大雷道。
<br/>　　齐德强却不过，只好一一将几人介绍给程大雷。程大雷表现的很是亢奋。
<br/>　　“喔，久仰久仰，老齐的兄弟就是我的兄弟，我和老齐没得说。”
<br/>　　“原来是您啊，经常听老齐说起你们。”
<br/>　　“既然大家一见如故，相谈甚欢，干脆结拜吧。”程大雷。
<br/>　　“……”众人。
<br/>　　大家都是幽州上将，有大好前程，怎么愿意去和一个山贼结拜。
<br/>　　“程寨主，你不是要走么？”齐德强只好这么说。
<br/>　　“喔，就走，就走。”程大雷临走前用力拍拍齐德强的肩膀：“老齐，不用送我。”
<br/>　　齐德强暗付：自己好像没表达要送的意思。
<br/>　　程大雷走后，齐德强身边几人都看着他。齐德强被盯得有些毛骨悚然：“你们看我做什么？”
<br/>　　“齐将，我们是军人，与山贼水火不容，你切切惦记自己前程，不要和山贼走得这么近啊。”
<br/>　　“……”齐德强。
<br/>　　马奔车驰，渐渐远离风雨亭，程大雷裹着大袍，恨不得把头埋近袍子里挡风。
<br/>　　“那罗铁可是够狂的，看他的意思是根本没将幽州绿林道看在眼里。”高飞虎想起罗铁仍有些气闷。
<br/>　　“大当家没让我动手，不然我非给他个厉害不可。”秦蛮冷哼一声道。
<br/>　　“好啦，他狂由他狂，和我们有什么关系。”程大雷早已查看过罗铁的信息，此人也是个顶级人物，真如果动起手来，秦蛮也够呛。
<br/>　　而除了罗铁之外，程大雷还至少看到三名顶级人物。幽州王帐下，可以说是人才济济。
<br/>　　“大当家，我们这样走了没有关系么，幽州王要封将，我们不在，位置说不定就会被熊老大抢去。”
<br/>　　“今天到了在大家面前露个面，表达一下我们的诚意就可以了。”说到『诚意』两个字，程大雷自己忍不住笑出声来。他在空中摆摆手：“至于熊老大和我抢么，这次是幽州王用得着我，不是我求着幽州王，看吧，这官位他会自己给我送来的。”
<br/>　　又行几十里地，一路上风越来越大，徐神机道：“大当家，前面有棵大树，咱们在树下避避风，吃点东西再走吧。”
<br/>　　“好，那就歇歇再走。”
<br/>　　撑到现在，程大雷自己也有些饿了，谁能想到李善言竟然不管饭呢。
<br/>　　几人在树下挺住，把随身携带的干粮拿出来吃。一边吃一边说着话，正在此时，忽听得远方马蹄疾，一队人向这边奔来。
<br/>　　高飞虎拧起眉峰：“大当家，好像是冲我们来的。”
<br/>　　程大雷也打点起精神，道：“准备一下，如果真的冲我们来的，就好好招待一下他们。”
<br/>　　八匹快马急奔，正是冲这边来的，还未靠近，高飞虎已经认出对方是谁。
<br/>　　“祁刀，伏虎山的二当家。”
<br/>　　“原来是熊老大的人。”
<br/>　　呼律律马啸，八匹快马同时停住。祁刀的目光落在程大雷身上：“程大雷，我们寨主吩咐，今天送你归西。”
<br/>　　高飞虎腾地一声站起来：“熊老大什么意思，我们和伏虎山向来井水不犯河水，你们……”
<br/>　　“好啦。”程大雷已经握紧斧头：“还废什么话，动手啦。”
<br/>　　人家已经明显是拿着刀来找后事的，你还啰哩啰嗦干嘛，直接动手啦。
<br/>　　程大雷已经拎着斧头冲了过去。
<br/>　　祁刀没想到对方竟然直接，感觉被杀的比杀人的还要凶猛。
<br/>　　电光火石间，双方已经战成一团。秦蛮大枪横扫，一一将他们挑落马下，在秦蛮面前，他们根本如纸糊的一般。
<br/>　　祁刀根本不是秦蛮的对手，被一枪串了心脏。几个人死的死，逃的逃，还有一人没来得及跑，被程大雷当场抓住。
<br/>　　“饶命，饶命，程寨主不要杀我。”
<br/>　　“好啦，好啦。”程大雷道：“没人要杀你，你把祁刀的人头带回去，帮我给熊老大传句话。”
<br/>　　“什么话？”这人下意识的道。
<br/>　　“告诉他……”程大雷略顿了顿，道：“算了，他见到祁刀的人头，就明白我要说什么。”
<br/>　　祁刀带人来追杀程大雷，反而被程大雷反杀，几人的兵器马匹自然都被程大雷留下。
<br/>　　几人打扫完战场，重新出发，回去路上要经过落叶城。
<br/>　　程大雷忽然想起，自己已经好久没见过苏樱，趁此机会正好在落叶城转转。
<br/>　　“你们先回山寨，把兵器和马匹带上，到时候我自己回去。”
<br/>　　“那大当家你当心，保重安全，我们先回去了。”徐神机。
<br/>　　“别走，你和我一起。”程大雷抓住了徐神机的肩膀。
<br/>　　
<br/>(本章完)
