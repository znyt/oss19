　　“不错，有着如此资质，也许你真的足以重现当初天兵苏家的辉煌了。”
<br/>　　楚风眠看着苏荷笑着开口道。
<br/>　　他跟当初的天兵苏家，关系还算不错，如今天兵苏家的衰败，也是让楚风眠有些唏嘘。
<br/>　　不过苏荷的资质，在天兵苏家历代家主之中，都算是极为优秀。
<br/>　　“走吧，我带你去一个地方。”
<br/>　　楚风眠看着苏荷，目光闪烁，突然想到了什么开口道。
<br/>　　“是。”
<br/>　　苏荷起身，急忙收拾好东西，准备离开。
<br/>　　对楚风眠的话，苏荷几乎都不带任何疑问，便是去执行。
<br/>　　楚风眠若是想要害她，早在之前就已经是足以让她去死了无数次了，更何况以现在楚风眠的实力，想要捏死她，不比捏死一只蚂蚁一样简单。
<br/>　　可是楚风眠却是传授给了他锻造之术，成为一名炼器大师，绝对是比起她苦修武道，要强大的多。
<br/>　　对于楚风眠的话，她都不需要任何质疑。
<br/>　　两人离开洞府，便是一路向着一个方向飞去。
<br/>　　这一路上，楚风眠走走停停，似乎是在寻觅这什么。
<br/>　　在周围，逐渐的便是出现了不少人迹罕见的荒山，群山之间，就连一点点人眼都看不见，也极为贫瘠。
<br/>　　没有任何武者，愿意踏足与这个地方。
<br/>　　逐渐的，在这群山尽头，是一处烟雾缭绕的山峰。
<br/>　　当楚风眠跟苏荷，踏入其中的一刻，突然之间周围的空间便是变化的天旋地转起来，在面前，显现出了一处白色的迷雾。
<br/>　　这迷雾，深不见底，一眼望过去都无法看清这迷雾的背后，到底是什么。
<br/>　　“前辈，这里是？”
<br/>　　苏荷打量了周围几眼，不由的看向了楚风眠。
<br/>　　这处山峰显然是极为诡异，他们二人，在这白雾之中行走，却是似乎一直以来，都是在原地踏步，简直是来到了一处迷宫一样的地方。
<br/>　　这让苏荷也有些好奇，楚风眠到底是为何要带她，来到这样一处地方。
<br/>　　“这里是当初天兵苏家的一处圣地，苏家虽然被毁灭，不过这个地方，应该还留了下来吧。”
<br/>　　楚风眠头也不抬的开口道，他的双眼，还在周围扫视这，似乎是在探察这什么。
<br/>　　“我们苏家的一处圣地？”
<br/>　　苏荷一脸茫然的看着眼前的无数白雾，还未再度开口。
<br/>　　突然之间，楚风眠便是凝聚灵力，猛然一掌向着前方批去，周围的白雾，在这一掌下，突然全部散开。
<br/>　　在二人的面前，一道青铜大门，突然出现，这青铜大门，显现出极为古老的气息，至少是有着足足万年，应该是没有被人开启过了。
<br/>　　“将你的血，滴在门上。”
<br/>　　楚风眠看着苏荷开口道。
<br/>　　“是。”
<br/>　　苏荷点了点头，便是走上前去，灵力一动，在她的手指间便是划出了一道裂口。
<br/>　　一滴鲜血，随着苏荷指尖一弹，落入到了这青铜大门之上。
<br/>　　“轰隆隆！”
<br/>　　在这一滴鲜血之下，这青铜大门上，突然凝聚出了无数古老的纹路，其中蕴含的力量，就连楚风眠都无法打破。
<br/>　　这青铜大门，便是当时天兵苏家的杰作，比起天级灵器都要兼顾的多，超越生死的人物都无法打破。
<br/>　　除非是他们苏家弟子，不然没有任何人，可以打开这青铜大门，进入其中去。
<br/>　　“咔嚓咔嚓！”
<br/>　　在这无数纹路的凝聚之下，青铜大门，缓缓打开。
<br/>　　楚风眠跟苏荷二人，随即步入其中，进入其中，楚风眠便是一眼看见，这青铜大门的背后，乃是巨大无比的庭院。
<br/>　　其中宫殿，足足是有着上百座，而在其中最大的那一座宫殿之中，一阵阵的地火，突然间喷涌而出，而在这地火的上面，是一尊足足有着十人高的巨大火炉。
<br/>　　从这地底之中喷出的地火，都被这巨大火炉，收入其中去，一共是有着九尊巨大无比的龙头，雕刻在其上，每一尊龙头，都是栩栩如生，看起来是跟真正的龙头，没有任何一丝一毫的区别。
<br/>　　“九龙鼎，这件宝物果然在这里，还没有被七大宗门的人所得到！”
<br/>　　楚风眠看到这九龙鼎的一刻，都不由的感叹了一声，这九龙鼎，可是当初天兵苏家的至宝，虽然不是极道仙兵，但是他的价值，却是丝毫不在极道仙兵之下。
<br/>　　因为几乎天兵苏家所炼制的绝大部分有名的灵器，都是出自这九龙鼎，炼制灵器最为重要的三点，一是用的材料，二是炼制的炼器大师，而第三，便是炼制灵器的火炉。
<br/>　　这九龙鼎，绝对是所有炼制灵器的火炉之中，最为完美的一种，缺少了这九龙鼎，天兵苏家也就不是天兵苏家了。
<br/>　　像是现在，七大宗门虽然是将天兵苏家毁灭，得到了其中的炼器之术，但是在整个九域，却是并没有缔造出第二个天兵苏家来。
<br/>　　所缺少的，正是这九龙鼎。
<br/>　　“这里，就是我们苏家当初的圣地？”
<br/>　　苏荷张望这周围的一切，目光之中也有着无比震惊的神情。
<br/>　　这数百座宫殿，简直是比起那当初的王家还要繁华的多，尤其是其中的一座座仓库，简直是装满了炼器材料，这种财富，简直是恐怖到了极点。
<br/>　　“不错，这里就是你们苏家的圣地之一。”
<br/>　　楚风眠看着苏荷震惊的表情，也是不由的感叹了一声。
<br/>　　现在苏荷虽然是身为苏家之后，但是却并不知晓当初的苏家，到底是一副什么样的繁华样子了，绝对是比起任何人的想象中的，都要繁华的多。

<br/>　　不过那些圣地，只怕是早就已经是被七大宗门的人所得到了，不会在留下了。
<br/>　　只有这一处，这一处圣地，在当时就连大部分的苏家弟子，都不知道。
<br/>　　当时楚风眠也是悄然之间，才发现的，不然的话就连他都不知道，苏家还有这样一处圣地。
<br/>　　“在这里，你就可以好好的联系你的炼器之道了，这里面几乎是有着取之不尽用之不竭的材料，可以随你动用了。”
<br/>　　楚风眠看着苏荷开口道。
<br/>　　“如此一来，我也能安心离开了。”
