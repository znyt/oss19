　　第123章
<br/>　　这是落叶城外一座野码头，冬来百草枯槁，雪封千里。瑟瑟寒风吹来，风在脸上如刀子一般。
<br/>　　程大雷裹着大氅靠在一棵大树上，鬼面斧就在他身边，随手可以握到的位置。此时斧柄握上去是比冰还要冷的凉。程大雷忍不住去想，在这个时代，人在这样的天气中究竟是怎样打仗的。
<br/>　　船刚靠岸，程大雷便把船上的人撒出去，以粮船为中心布成一道警戒圈。这样一旦有人靠近，程大雷便可第一时间警觉。
<br/>　　身后响起枯草被鞋底踩碎的声音，程大雷下意识握住斧头，回过头看见靠近的人是苏樱。
<br/>　　“天这么冷，你怎么出来了？”程大雷问。
<br/>　　“我过来陪陪你。”苏樱小声道，在程大雷身边蹲下，低声感慨一句：“天真冷啊！”
<br/>　　“嗯。”
<br/>　　“你不冷么？”
<br/>　　程大雷心道：你这不是废话么。
<br/>　　忽然他感觉自己的左手被苏樱小手攥住，低头哈了口气：“还冷么？”
<br/>　　“其实……你的手更凉。”
<br/>　　“……”苏樱。
<br/>　　“你还是回去吧，待在这里，乱我军心。”
<br/>　　已经有人悄悄往这边看，想自己一寨之主，不陪兄弟们一起受冻，反而和一个女人卿卿我我，像什么样子。
<br/>　　此时忽听得马蹄急响，程大雷顿时来了精神，握紧了斧头。
<br/>　　来的是敌是友，那句话怎么说来着，朋友来了有好酒，敌人来了有斧头。
<br/>　　“大当家，军师回来了！”有人把消息传来。
<br/>　　只听得马蹄响，见得雪粉乱，呼啦啦一大群人闯入丛林。粗略不知来了多少人，看上去有数百人，但真实数目可能更多。
<br/>　　高飞虎，熊大熊二……青牛山各寨头领几乎同时都到了。一月不见，他们每个都瘦了一圈，看到程大雷时，眼里闪烁着绿光。
<br/>　　“大当家！”
<br/>　　所有人几乎同一时间跪倒，口中高呼，声音连成一片，树枝上的积雪似乎都被震乱。
<br/>　　“别喊了，别喊了，再给我把人招来。”程大雷跺脚大跳。
<br/>　　高飞虎行到程大雷身边，再次跪倒：“程当家，大恩不言谢，以后我这条命，便是程当家的。”
<br/>　　“先不提这个。”程大雷把高飞虎搀起来，忽然注意到他胳膊上缠着绷带：“怎么，你受伤了。”
<br/>　　“唉，别提了。于虬然带着一众人要走，我和他们争执的时候，胳膊擦破点皮。”
<br/>　　程大雷走了一个月，一路虽然也谈得上顺利，但也很是遇到几次危险。而至于青牛山的情形，程大雷却是不知道的。
<br/>　　山里的粮食由刘悲掌握，一开始他就对这件事不乐观，当然他对任何事都不乐观。山寨里剩余的粮食，由他左省右省，总算是撑了二十几天。
<br/>　　可到后来，大家每天能吃到的粮食越来越少，于是人心也就开始乱了。
<br/>　　对于程大雷能带粮食回来的事情，很多人都不太有希望。有些人拖家带口，想走也走不了，而有些山贼则是孤家寡人，觉得没必要陪着大家一起死，便悄悄溜走了。
<br/>　　事情是在三天前发生的，于虬然纠集了一二百人想撤，临走前想将剩余的粮食抢走。双方发生了战斗，互相都死了一些人，最后于虬然自知不敌，趁乱把剩下的粮食烧了，自己带着一帮人逃走了。
<br/>　　高飞虎在战斗中受了些伤，却也不是很眼中。关键是于虬然的叛变深深伤害了这个糙汉子的心灵。画虎画皮难画骨，知人知面不知心，他从没想过于虬然会背叛他。
<br/>　　粮仓被烧，山里也就彻底断粮，大家把能吃的都吃了，如果程大雷再不回来，人们易子而食的事情并不是不会发生。
<br/>　　“你也别太难过，不是不报时候未到，会有机会的。”程大雷安慰高飞虎。
<br/>　　“肚子饿得也没有时间难过了。程当家，你带回多少粮食？”高飞虎问。
<br/>　　程大雷比出四根手指：“四十万斤。”
<br/>　　“什么！”高飞虎错愕的看着程大雷：“程当家你不会是耍我么？”
<br/>　　“我有那么闲么，两艘船拉回来的。”
<br/>　　有旁边的人听到后，瞬间大呼小叫：“程当家带回了四十万斤粮食。”
<br/>　　具体数目徐神机还没告诉他们，在知道这个消息后，一群人发出阵阵怪叫，有人眼眶通红，激动得落下泪来。
<br/>　　“都别叫，都别叫！”
<br/>　　程大雷连忙呼喝，却止不住这些人的疯狂，最后过了好久，人群才算是安静下来。
<br/>　　“先别叫，当务之急是怎样把粮食运回去。”
<br/>　　“程当家放心，我就是用肩膀扛，我也能把粮食运回山，不让一粒粮食掉在地上。”高飞虎拍着胸脯。
<br/>　　话是这么说，但这里距离青牛山有七八十里，真让高飞虎背着粮食回去，除非把他类似。
<br/>　　“快，必须快，在天亮之前要把全部粮食运回山寨，不然若是走漏消息，我们谁都扛不住。”程大雷对高飞虎说。
<br/>　　“明白。”高飞虎点头道。
<br/>　　这次高飞虎带了五百人来，马约有五六十匹，立时派人返回山寨，派更多的人来，马车就地改成雪橇，上面堆满上好稻米。
<br/>　　他们这些日子都快饿疯了，平时这帮山贼莫说吃，便是见也没见过这么好的稻米，莫看饿了好几天，每个人却好似都有使不完的力气，上百斤的麻袋扛起来就走。在忙碌干活的过程中，时不时发出阵阵大笑。
<br/>　　可程大雷这个时候却笑不出来，整个青牛山满打满算，不过一百多匹马，其中大部分还是程大雷卖给他们的。就算每匹马能拉一吨，也就是两千斤，至少也需要来回两趟。到时候先不说要消耗多少时间，马能不能受得了都不好说。
<br/>　　现在，距离天亮没多长时间了。
<br/>　　程大雷一发狠心，马拉不走就用人拉，必须一次把粮食都运走。
<br/>　　青牛山只要能喘气的都叫上，就地伐木做成雪橇，马拉不完用人拉，老弱妇孺，哪怕是八岁孩子，肩上也得扛一袋粮食。
<br/>　　程大雷，高飞豹，小白狼，张肥等随船的几十人，充当保卫工作。
<br/>　　在天忙碌完这一切，天已蒙蒙见亮，一队人浩浩荡荡离开海边，向青牛山进发。
