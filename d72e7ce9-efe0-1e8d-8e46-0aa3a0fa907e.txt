<br/>　　一念禅师禅师听完之后，当场怔住。
<br/>　　“什么，只有一个人？”
<br/>　　弟子点点头，还想详细解释，一念禅师推开他，飞身而起，穿过一众僧兵，很快就找到了混乱的根源：的确是只有一个人，从远处缓缓走过来，无数僧兵试图阻挡，但一旦靠近他，立刻被冰冻起来。
<br/>　　一念禅师亲眼看见这一幕，也是震惊的说不出话，不过他很快回过神来，传令下去，让所有人退到自己身后集结，避免无谓的牺牲。
<br/>　　在阵型变换之后，对方那个鬼卒也来到了一念禅师面前不远处，站住了。
<br/>　　自从他出现之后，第一次站住了，静静地看着一念禅师。
<br/>　　“尊驾……”一念禅师迷茫地看着他，双手合十，行了一礼。
<br/>　　“一念。”鬼卒缓缓地吐出两个字。
<br/>　　一念禅师心中一动，这个看上去陌生的人，怎么会知道自己的名字？
<br/>　　“跪下。”鬼卒又轻轻吐出两个字。
<br/>　　什么！！
<br/>　　这两个字，圆通寺的弟子们也听得清清楚楚，顿时哗然起来，一个个义愤填膺地望着鬼卒。虽然这个神秘的家伙很强，但是一念禅师是他们的方丈，圆通寺也是空界四山十二门之一，虽然实力上比不过梨山和青丘山，但毕竟是一大宗门，一念禅师也是近乎尊者的境界。
<br/>　　一个陌生人，一见面就让他们的尊师跪下，这简直也太不给脸了，就算这些人都是号称修身养性的和尚，也根本忍不住了。
<br/>　　一念禅师怔了一怔之后，一张荣辱不惊的脸上也是现出了些许的怒意，冷冷说道：“尊驾何人，让贫僧下跪，也得有个理由吧。”
<br/>　　鬼卒静静地看着他，张口说道：“你三世为僧，五世为道，却不得造化之门，曾拜在我门下，恳请点化，我以永字八法的笔意点化于你，堪破六道轮回，穿梭空界，开宗立派，自求悟道。今日见我，敢不下跪？”
<br/>　　扑通！
<br/>　　一念禅师双膝一软，跪在地上，脑袋紧贴着地面，不敢仰视。因为激动，浑身剧烈地颤抖起来，凭这一番话，他已经知道来人是谁了。
<br/>　　这一幕，却是给他身边那些弟子带来了强烈的震撼，所有人怔在当场，傻傻地看着他们那平时高高在上的师尊，用如此谦卑的姿态跪倒在这个陌生人的面前。
<br/>　　他……究竟是何方神圣？
<br/>　　“抬头。”鬼卒又下了命令。
<br/>　　一念禅师抬起头，一脸的惶恐，嘴唇哆嗦着，哪里还有一点佛门高僧的气度。他早就是堪破生死关，灵身也没有虚无和实在的区别，不会流泪，不然这个时候一定是老泪纵横了。
<br/>　　“师尊……”一念禅师不敢仰视，又低下头，不停地磕头。
<br/>　　鬼卒心平气和地享受着这一切，没有局促，也没有一丝傲慢，只是心安理得，静静地说道：“而今你在青冥界开枝散叶，也是一派之主了。”
<br/>　　一念禅师一听，浑身一震，极为惶恐地说道：“弟子不敢，弟子几次到阴司拜谒，都不得机会觐见……师尊之恩，弟子一刻不敢忘怀。”猛然想到什么，急忙转头环顾左右，面对众多弟子，没有丝毫害羞的感觉，叫道：“还不跪下参府君陛下！”
<br/>　　眼前这个鬼卒的真身，正是崔府君！
<br/>　　通判天下生灵、执掌两界生死的阴间天子！
<br/>　　圆通寺所有和尚，在听到一念禅师的话之后，一个个都石化了。阴间天子……居然来到了青冥界！
<br/>　　青冥界是一个近乎孤立、不受阴司管辖的空间，但不代表阴司的大佬在这里没有威信，平时他们远离天子通判和六道轮回，对阴司没什么向心力，甚至心里有点藐视，但是当崔府君这样一个阴司真正的大佬站在面前，没有人不感到自己的渺小。
<br/>　　一念禅师身后几个长老，率先跪了下去，身后那些弟子也陆续下跪，不一会工夫，圆通寺所有僧兵都跪在了崔府君的面前。
<br/>　　“师尊，您怎么……来到这里了？”在震撼和激动过去之后，一念禅师猛然间想到了这个问题，心中顿时紧张起来。
<br/>　　崔府君一向坐镇在天子殿，极少离开天子殿，更不要说离开阴司，穿梭到别的空间来……就算是借用别人的躯体，只是神识穿越，这种事在一念禅师的记忆中也是头一遭，忍不住展开了浮想，不过不管哪一种想象，都让他感到冷汗直流。
<br/>　　“我很生气。”崔府君说了这四个字，也只说了这四个字，微微转头，望着左前方。那是星宿海的方向。
<br/>　　生气……一念禅师浑身哆嗦，崔府君不是钟馗那种火爆脾气，千百年来一向温文尔雅——每天批阅生死簿，见识各种各样的因果恩怨，还有数不清的鬼魂的哭诉纠缠，如果稍微敷衍一下，生死簿上随便勾一笔，都会影响至少一个人的生死机运。
<br/>　　都说地藏王无人相无我相，万念皆空，其实阴司之内，真正对任何事做到心如止水的，只有崔府君。
<br/>　　一念禅师很难想像，“生气”这两个字会从崔府君口中吐出。
<br/>　　天子之怒，谁敢承担？
<br/>　　一念禅师虽然极度好奇令崔府君生气的原因，但也不敢多问，嗫嚅道：“弟子惶恐……”
