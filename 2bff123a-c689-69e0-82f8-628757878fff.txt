
<br/>　　<br/>　　    老马看向王惊蛰，问道：“这位小哥，懂一些门道？”<br/>　　<br/>　　    王惊蛰淡淡的说道：“家里长辈以前做过阴阳先生，我就知道一点皮毛而已，恰好用上了”<br/>　　<br/>　　    王惊蛰低调的一笔带过了，他和菜刀文不说对方自然不会知晓他到底用了什么手段，陈臣自然也看不懂了。<br/>　　<br/>　　    老马“哦”了一嘴后也没再多问了，他料想王惊蛰也就是知道一点皮毛罢了，就跟陈重说道：“估计是因为有人盯上了你川南拿的那块地然后才下手的，对方可能知道明着来不行，就在背地里耍了手段，小臣和她妈也许就是他用来要挟你的筹码了，你要不答应他们两个往后可能就不太好说了”<br/>　　<br/>　　    陈臣一听顿时急了，连忙问道：“爸，我妈怎么了？”<br/>　　<br/>　　    陈重皱眉说道：“你妈在申城出了点事，现在还人事不省，送到医院去检查了也看不出什么问题，本来我今天下午打算飞过去的，但是没想到你这也出了问题”<br/>　　<br/>　　    老马从地上捡起两个草人，端详着上面的生辰八字后指着枯瘦的男子，说道：“抓到他了，解决就不麻烦了，剩下的就得看怎么把背后那人给揪出来了”<br/>　　<br/>　　    陈重叹了口气，脸色难看的说道：“手段用到我头上来了，这是怪我最近几年太收敛了？老马，我老婆那里······”<br/>　　<br/>　　    “我去一趟申城看看，今天把人接回来，这人你先看住了！”老马说道。<br/>　　<br/>　　    “好，那麻烦你了！”<br/>　　<br/>　　    到这时候，王惊蛰忽然插了一嘴：“陈先生，那看来没我们什么事了？”<br/>　　<br/>　　    陈重拱了拱手，挺真诚的说道：“惊蛰，这次真得要谢谢你了，要不是你正好帮小臣解决了麻烦，那我可就太被动了”<br/>　　<br/>　　    王惊蛰淡淡的说道：“正巧碰上罢了，陈先生如果没什么事，那我们就告辞了”<br/>　　<br/>　　    陈重点了点头，跟陈臣说道：“替我送送他们，等事情完事之后，我再单独请你们吃个饭”<br/>　　<br/>　　    片刻后，王惊蛰他们三个从陈家里出来，陈臣开着车送他们回去，路上王惊蛰跟她说道：“你这几天不要太累了，好好休息下，最好找个中医给你开个养神补气血的房子，不然容易落下什么病根”<br/>　　<br/>　　    被鬼上过身的人，普遍都会发虚，气血不错，不好好调理一番恐怕接下来就得要大病一场了。<br/>　　<br/>　　    陈臣扭头问道：“剩下的你就打算不管了？”<br/>　　<br/>　　    王惊蛰笑了：“你家里的事，我不好随意插手，再说你爸不是找了那个老马么？这种事外人肯定越少知道越好，我就不掺和了”<br/>　　<br/>　　    陈臣皱眉说道：“那我怎么就对你感觉比较放心呢？”<br/>　　<br/>　　    “你这是先入为主的观念，我就知道那么一点皮毛也帮不了什么大忙，这世上高人太多了，你得找个对症下药的人才行，至于我你就别产生什么错觉了”王惊蛰摊着手说道。<br/>　　<br/>　　    “好吧，那谢谢你了，等过几天没什么事了，我请你吃饭吧？”<br/>　　<br/>　　    “吃饭就算了，我的胃口简单的很，没什么追求，不过你要是打算谢我的话，帮我个忙行不行？”王惊蛰说道。<br/>　　<br/>　　    陈臣说道：“你说，两个忙都行，只要我能办得到”<br/>　　<br/>　　    “刚才你爸说，有人看上了他在川南拿的一块地皮，你打听一下这块地在哪，我想过去看看”<br/>　　<br/>　　    “就这个？”陈臣不可置信的问道。<br/>　　<br/>　　    “嗯，就这么简单······”<br/>　　<br/>　　    回到家里之后，丁武收拾了下就去酒吧了，王惊蛰和菜刀文随便弄了点午饭。<br/>　　<br/>　　    “你这是深藏功与名啊，就这么功成身退了？多好的机会，不巴结下那个陈重呢”菜刀文冲着他眨了眨眼睛，说道：“还有救美呢，你应该顺着杆子往上再爬一下的，跟小臣臣套套近乎什么的”<br/>　　<br/>　　    王惊蛰端着饭碗，无语的说道：“我巴结他干嘛？要钱要是要名啊，还有·······我现在也没心思近女色”<br/>　　<br/>　　    “那倒也是，不过你问那块地干什么？”菜刀文不解的问道。<br/>　　<br/>　　    “能让人请巫门子弟来给陈重的老婆和女儿下巫术，这人肯定是不怕他，又有这些手段，你说他瞧上的地皮能没有点什么意思么？真就是一块普普通通的地皮，他也不至于吧？”<br/>　　<br/>　　    菜刀文皱眉说道：“你觉得这地皮有什么门道？”<br/>　　<br/>　　    “不知道是啥门道，但我觉得肯定会有点什么让人感兴趣的地方·····”王惊蛰用筷子搅拌着米饭，搅了半天后，突然抬头问道：“你就不问问我为啥会有兴趣么？”<br/>　　<br/>　　    菜刀文被他问的有点发愣，半天后才憋出一句话：“因为闲的？”<br/>　　<br/>　　    “啪嗒”王惊蛰放下碗筷，看着窗外说道：“对，就是闲的，这一天天的除了吃喝拉撒睡，你没发现我每时每刻不都是闲的蛋疼么？人生好像都没啥追求了”<br/>　　<br/>　　    菜刀文笑道：“你这不是刚出过事么”<br/>　　<br/>　　    王惊蛰摇了摇头，说道：“不是，其实冥冥中我总是有一种感觉，我好像应该有什么重要的事去做，但可惜就是想不起来了，这操蛋的失忆生活，要给我折磨懵了······”<br/>　　<br/>　　    于此同时，和川中隔了几百公里的酆都城，多天以前茅小草来到了这里。<br/>　　<br/>　　    酆都俗称鬼城，古时也被叫做巴子别都，到现在已经有两千多年的历史了，据说，相传阴曹地府最正统的入口就在酆都山的下面，这里是所有亡魂进入阴间的通道，于是每年都有大把的游客来到酆都，想要找找传说中的阴曹地府，可惜的是古往今来有两千多年了，也从来没有哪个人找到过。<br/>　　<br/>　　    但阴曹地府的入口到底在不在酆都呢，其实酆都只不过是恰好有一处鬼门关罢了，进了这道鬼门就是黄泉路，顺着黄泉路往前是忘川河奈何桥。<br/>　　<br/>　　    这样的鬼门在各地都有不少，比如黄河鬼门也是，只不过是当年恰好有人看见群鬼酆都出没也看见过阴兵过路，所以口口相传下，再加上刻意的塑造，久而久之酆都鬼城的名声就被定型了。<br/>　　<br/>　　    茅小草此时就在酆都山上，她也是为了鬼门而来的，为了还在阴曹地府的王惊蛰。
