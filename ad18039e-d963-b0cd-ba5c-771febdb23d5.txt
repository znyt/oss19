　　“大妹！你怎么来了？”李承乾眼中闪过一抹惊讶，接着就是找到救星一般的狂喜。
<br/>　　总算是来一个真正的自己人，可以不用独自应付那两个小丫头了。
<br/>　　“还说呢，出来玩儿都不叫我，如果不是小四妹妹眼尖看到你们，刚刚差一点就错过去了。”长乐抱怨着，爬上李承乾的马车，抓起一颗果脯塞进嘴里，边嚼边咕哝着。
<br/>　　“小四也过来了？”李承乾面色一变。
<br/>　　“对啊，在外面和小雯姐和苏玫照相玩儿呢。”长乐点着小脑袋，继续往嘴里塞着果脯，丝毫没有注意到李承乾脸色的变化。
<br/>　　“那个……长乐，哥求你点事儿成不？”犹豫了一下，李承乾脸上露出透出一丝焦急的表情。
<br/>　　“什么事啊？这么神秘？”长乐自然而然的问了出来。
<br/>　　“哥有些内急，你先帮哥在这里支应一下，没问题吧？”
<br/>　　“哦，那你去吧，快点回来！”长乐并没有怀疑什么，很痛快的答应下来。
<br/>　　接着李承乾就让夜魅等人把挂在马车车厢后面的自行车取了下来，骑上之后飞一般的消失在了街角。
<br/>　　大唐必竟不像现代，公共场所都休了不少的公厕，所以想要找地方解决内急，的确是个大问题，长乐自然不会对哥哥的离开有什么怀疑。
<br/>　　不过当时间一点点过去，日头已经渐渐开始偏西，李承乾却依旧没有回来的时候，长乐等人终于觉查到了事情的不对劲。
<br/>　　把留下来的杨雨馨找过来一问，才从这个太子秘书的口中得知，无良的太子殿下早就已经尿遁，不会再回来了，而这些马车则全都是留下送女孩子们回家用的。
<br/>　　知道真相的女孩们面面相觑无言对视，一时间竟不知应该说些什么，长乐和程小四不知道前面发生的事情，对李承乾尿遁始终有些不解。
<br/>　　但是孔雯和苏玫两个却是整件事情的亲历者，自然知道李承乾为什么会半路抽身，是以都觉得有些难堪，对视一眼之后隐隐在心底泛起一丝悔意。
<br/>　　经过了一次失败的出行，李承乾消停了一段时间，照相车也在出行的第二天借给了李泰那小子。
<br/>　　这小胖子想要办个文会，觉得名士齐聚的场合如果不拍个照留个念什么的有些丢份儿，所以才会来找李承乾借车。
<br/>　　李承乾正巧得了美女恐惧证，照相车留在手中也是无用，索性问都没问就借给了李泰，同时借给他的还有‘高手’和另外一个方士。
<br/>　　必竟照相车里面的东西对于唐人来说还是很专业的，必须有专业人士来操作。
<br/>　　长乐对李承乾尿遁一直心怀不满，这段时间也躲回了自己的地盘，不再露头，让她的大哥也得以过上几天消停的日子。
<br/>　　只是无所事事的日子李承乾并没有过上几天，老头子身边的方老太监就再一次找上门来：“太子殿下，陛下召见！”
<br/>　　“父皇？”李承乾有些诧异，老头子这段时间不是跑去折腾老四去了么？怎么会突然想起自己了？
<br/>　　“殿下，快点走吧，一会儿陛下要等急了。”方老太监看李承乾站在原地瞎琢磨，忍不住出言催促。
<br/>　　“哦哦，好，这就走！”连声答应着，李承乾跟着老方出了‘兰若寺’。
<br/>　　而在去往太极宫的路上，李承乾好奇的问道：“老方，知道父皇找我有什么事儿么？”
<br/>　　方老太监四下看了眼，见没人注意自己这边，便压低了声音说道：“陛下这段时间一直想要东征，老臣估计应该是和高句丽有关系。”
<br/>　　李承乾点点头，没再追问下去，知道老头子不是找自己麻烦就行了，多的没必要问的那么清楚，反正一会儿老头子一定会说，现在知不知道都是一样的。
<br/>　　一路无话，两人步履匆匆的赶到甘露殿老头子的书房，在侍卫的通传之下之，很快就听到老李在里面唤他二人进去。
<br/>　　“父皇，您找儿臣？”老头子的房间中还有长孙无忌、魏征等人在坐，一个个面色凝重，似乎正在考虑着什么重大的事情。
<br/>　　“看看这个，再来说说有什么想法。”李二将前段时间收到的那份高句丽国书在桌上敲了敲，示意李承乾看看。
<br/>　　“高句丽的国书？”李承乾接过那份国书，先是打量了一眼，然后翻开封页，接着有些疑惑的问道：“怎么就两个字？父皇，您拿错了吧？”
<br/>　　“朕很想拿错，不过可惜的是，没有！”李二阴着脸，沉声说道。
<br/>　　“可是父皇，小小高句丽，怎么敢用此等语气说话？”
<br/>　　装傻充愣是李承乾特有的本事，老头子既然前段时间把他雪藏起来，那他就没有必要表示自己什么都知道，那样除了让老李觉得没面子之外，没有一点好处。
<br/>　　“不管他用什么语气说话，你只说说你是怎么想的。”李二瞪了儿子一眼，意思是：少给老子装傻！
<br/>　　老头子这段时间一直在纠结高句丽的问题李承乾知道，而且他还知道似乎整个朝堂上似乎有一半是反对东征的声音。
<br/>　　这一回老头子这次估计是顶不住了，想要寻找一个坚定的盟友，打算共同‘迎敌’。
<br/>　　如果此时不站在老头子的立场上说话，后果难料。
<br/>　　于是乎，李承乾几乎想都没想就双手抱拳，凛然说道：“父皇，儿臣认为此等宵小行径必须严惩，否则我大唐煌煌天朝上国颜面何存！”
<br/>　　“好，果真是我李家儿郎！”对于李承乾能这么快领会自己的意思，李二感到十分满意，觉得还是这个大儿子比较贴心，知道照顾自己这个老爹的脸面。
<br/>　　不像那个臭老四，天天‘之乎者也’，面对如此国书，竟然还想着什么教化、谴责，真是把书都读到狗肚子里去了。
<br/>　　不过就在李二和李承乾父子二人唱双簧正开心的时候，魏征在一边突然开口说道：“太子殿下，我大唐终究是宗主之国，岂可因一言而动兵罚之？”
<br/>　　“而且现下时局不稳，吐蕃、吐谷浑、西突厥、薛延陀此等蛮夷尽在观望，如果东征高句丽再次失败，只怕立刻就是举世攻唐的局面，殿下，难道就没有想过么？”
