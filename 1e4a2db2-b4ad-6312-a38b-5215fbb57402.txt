　　第550章
<br/>　　来若疾疾之风，有摧枯拉朽之势。
<br/>　　钱观鱼刚从二楼跃下，就感觉一阵劲风扑面，他下意识去拔身上剑，但就在手指刚接触到剑柄的刹那，一柄飞斧凌空而至，切断了他的喉咙。
<br/>　　他的剑只来得及拔出三寸，双眼便变得空洞起来，嘴巴开合，没来得及说出一句完整的话，身子便咚地一声倒地。
<br/>　　若离得近，便可以分辨出，他临死前拼命想说出口的两个字是——好快。
<br/>　　好快，的确好快。
<br/>　　钱观鱼自诩书院第一快剑，书院上下对他着力培养，至少证明他在这方面的潜力很强。然而，他还是没搞清楚一个事实，书院第一快剑，并不等于天下第一快剑。
<br/>　　真正的战斗也绝非双方相对而立，互报家门后你一招我一招切磋较技。说白了，钱观鱼是个学院派高手，而真正的战斗如何，他一窍不通。
<br/>　　所以在今日的战斗，他连出手的机会都没有，便被一柄飞斧斩断了喉咙。
<br/>　　一道黑影紧随而至，捡起地上的斧头。黑影左手还拎着一柄，相映成趣，根据对方纤细的身形判断，应该是女人。皮肤略黑，眼睛很亮……
<br/>　　咚咚咚。
<br/>　　几道人影从天而降，落在藏书楼前，彼此对视一眼，眼神干净直接。
<br/>　　“快快动手，找到东西就走，莫要打草惊蛇。”
<br/>　　有一人发号施令，其他人迅速开始行动。很显然事先已经过安排，有人直奔二楼，有人来到一楼书架前，迅速将书架上的书籍浏览而过，什么医卜星相，经史子集都一扫而过，他们很明显是为某一件东西而来。
<br/>　　他们行动迅速，配合默契，在书海中翻检。
<br/>　　咚咚咚。
<br/>　　急促的脚步声从外面响起，几人眼中一惊：有人来了。
<br/>　　书院以『万卷』为名，藏书何止万卷。整座山最要紧的地方，就是这座藏书楼。如此重要的所在，自然不会只有一个愣头青钱观鱼守卫。书院还安排了其他高手，他们才是保护这座藏书楼的真正力量。
<br/>　　“你们继续走，我拦住他们。”
<br/>　　一个黑袍大汉喝了一声，直接从窗子跃了下去，几乎刚刚落地，他就陷在了包围圈中。
<br/>　　七道人影立在他身前，人人身穿白袍，手持长剑。这幅打扮院的人。
<br/>　　“什么人，竟敢杀人盗书？”
<br/>　　黑袍大汉卸下背上重剑，单手提在手中，口中惜字如金：
<br/>　　“来。”
<br/>　　七柄长剑同时向他攻来，都是书院的制式长剑，长约三尺一寸，精铁打制。用的也是书院的书生意气剑，组成书院扬名天下的剑阵。
<br/>　　书院的剑阵可大可小，七人成阵，十三人成阵，二十八人成阵，四十九人成阵，七十二人成阵。当然，七十二人时是威力最强的状态，人数再多，就难以形成配合。
<br/>　　眼前虽然只有七人，但威力已不容小觑，七柄剑形成密集的剑网。黑袍大汉闪避的路线被封住，剑刃攻向他周身大穴。
<br/>　　黑袍大汉很明显不懂剑阵，也没有破阵的想法。他只是在攻击，干净直接的攻击。
<br/>　　剑阵在黑袍大汉心中是一件极其难以理解的事情，若是两军对垒，士兵经过艰苦训练，默契配合，的确可以达到以十当百的效果，但对于真正的高手来说，阵法有什么用。
<br/>　　十把柴刀绑在一起，也成不了神兵利器啊。
<br/>　　任你千变万化，我自一剑破之。
<br/>　　黑袍大汉用的就是剑，一柄重剑。大开大磕，一味强攻。任你千般巧，我一力降十会。
<br/>　　书院的剑阵暂时竟奈何不了此贼，但书院方面的人也并不着急。这里是书院，高手如云的书院，只要缠住他，时间拖得越久，对己方便越有利。
<br/>　　黑袍大汉明显也察觉到了这一点，他想挣脱剑阵。此刻才明白书院剑阵的了得之处，七柄剑形成一面剑网，你用力挣扎，剑网就会暂时松开。但只要你稍微一松懈，剑网就会缠上来。黑袍大汉像是陷入网中的野兽，猎人在等着他力气耗尽，便刺出致命的猎刀。
<br/>　　“大师兄，找到了。”
<br/>　　黑衣女子从楼中跃出来，手中拎着一本泛黄古卷，其他几道人影也出现在其身后。
<br/>　　一见目标已到手，黑袍大汉更是没有恋战的心思，他一挥手中重剑，将面前敌人逼退。
<br/>　　“走。”
<br/>　　双方人数上已经拉平，书院方面已没有人数上的便宜。
<br/>　　书院是什么地方，是读圣贤书的地方，讲究个体面，讲究风度翩翩。而这些人的打法却直接粗暴，如山林中的野兽，也不与你废话，上山路下三路，那里阴险那里招呼。
<br/>　　千金小姐遇见老混蛋，结果可想而知。
<br/>　　七人突破剑阵，向外逃去，七道人影健步如飞，走过山上崎岖道路如履平地。
<br/>　　而此时，整座书院已经苏醒过来，师长中的前辈高手，听见警报后，各持兵器奔出房间。
<br/>　　书院仁字牌坊下立着一人，五尺身躯，手持一杆大棍，颇有一夫当关万夫莫开的架势。
<br/>　　“什么人，与俺留下。”
<br/>　　书院张六，本是海上兴风作浪的巨贼。然打打杀杀一世，让海上的客商闻风丧胆。可最后却被手下背叛，丢入了大海中。后被书院夫子所救，心灰意冷之际成了书院的守门人。
<br/>　　所以他用棍，不用剑，虽然身在书院，但他不是读书人，而是个老混蛋。
<br/>　　一夫当关，长棍锁门，若问书院弟子最怕的是谁，未必是夫子，而是看门的老张。
<br/>　　毕竟，夫子还是讲道理的，老张却只给你讲大棍。
<br/>　　巨剑对大棍，老混蛋遇上小混蛋，一时竟谁也讨不到便宜。
<br/>　　远处人影晃动，书院的高手正在赶来，黑袍大汉知道事情不妙，今日冒昧闯山，的确小窥了书院。
<br/>　　“走！”
<br/>　　强攻三招，将张六逼退，带着手下向山下逃去。张六最近几年已经开始修身养性了，但今天心中的豪横劲儿的确被激发出来，他们如果能在我面前逃脱，以后的面子往哪里搁。
<br/>　　“给俺留下吧。”
<br/>　　手中的长棍当石头，狠狠投了出去。
<br/>　　耳边听得哎呦一声，几人消失在夜幕中不见。
<br/>(本章完)
