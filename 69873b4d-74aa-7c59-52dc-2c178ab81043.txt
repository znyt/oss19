　　第526章
<br/>　　程大雷看着刘发财，刘发财也同样看着程大雷。刘发财当然没见过程大雷，可冥冥之中，他已察觉到有什么奇怪的事情发生了。
<br/>　　“啊呀！”
<br/>　　程大雷抱着脑袋在地上一滚，口中大叫：“山大王杀人了，山大王杀人了！”
<br/>　　刘发财狠狠的吐出口吐沫，手脚并用从河里爬出来。刚才究竟发生了什么事，那种奇怪的感觉究竟是从何而来？
<br/>　　桥头乱作一团，黑河九鬼一帮吵着要杀人。过桥的平民逃的逃，奔的奔，有人想浑水摸鱼，趁机过桥，运气不好被一刀砍了。有人慌不择路，跑进河里，很显然水性不如何，连个浪花都没打起来，就被急流冲远了。
<br/>　　独角鬼带着手下已经将刘发财围在当中，他手中举一杆镔铁棍，脸上挂出狰狞的冷笑。
<br/>　　“好贼！敢在爷爷面前装神弄鬼，弟兄们给我宰了他。”
<br/>　　刘发财却也算个人物，手里擎着刀，道：“不怕死的尽管上来，与本当家的刀说话。”
<br/>　　独角鬼是哈哈大笑：“事到如今，你还嘴硬。传说中那位有时候用斧头，有时候用剑，可从听人说他老人家动过刀。”
<br/>　　“小子，本当家有很多本事，难道要一一说给你听。”
<br/>　　什么叫高级骗子，这就是高级骗子，要想骗过别人，就要先骗过自己。到现在，刘发财已经完全暴露，但他强撑着嘴硬，独角鬼手下难免有脑袋不好使的，忍不住会想：他会不会就是哪位！
<br/>　　因有这想法，心中难免就会有些怯，出手的时候难免就不够狠。
<br/>　　好在独角鬼可以横行跋扈这么多年，毕竟也不是泛泛之辈。如果真是那位到了，他自然不敢动手。可一个冒牌货，他又何惧之有。
<br/>　　刀拖斌铁棍他向刘发财冲了过去，二人在小喽啰的包围圈中缠斗起来。
<br/>　　所谓锤棍之将不可力敌，敢拎条棍子上战场的，必定有些力气。而这刘发财是用刀的，刀讲快、讲险、讲奇，可没人说过，用刀对力气有要求。
<br/>　　还有一句话说得好：一力降十会。
<br/>　　独角鬼的铁棍抡开，刘发财便进不了身。而对方一棍砸过来，他手中的刀就架不住。于是只能一味躲避，在间隙砍出一刀，但却威胁不到独角鬼。
<br/>　　二人之间的胜负已经很明显。
<br/>　　刘发财越来越慌张，他靠招摇撞骗过日子，却也不是没有被人识破过，但他本身功夫也不弱，所以往往能化险为夷。然而这一次，却比任何一次都要危险。
<br/>　　逆流从水中钻出来，那还剩什么力气，现在越打越疲，他想要冲出重围，可周围密密麻麻都是独角鬼的喽啰。让他再次跳河，他可没自信能活着爬上河。
<br/>　　“大风大浪都过去了，难道本当家今日真要死在几个蟊贼手里么！”筋疲力尽的他发出一声长叹。
<br/>　　程大雷也真算服了这货，怕他临死前，都不会说一句真话。
<br/>　　“哎呀呀，牛惊了，大家小心啊。”
<br/>　　独角鬼正准备上前，一棍撩翻刘发财。将他擒到山寨，剥皮放血，挖肝取心烹一桌好菜，身后忽然响起一道惊慌失措的声音。
<br/>　　他回过头来，见一头黑牛硬生生冲过来。牛背上一条大汉扶着牛角，口中胡乱喊着。
<br/>　　“散开，散开啊！”
<br/>　　来势汹汹，荡起一片烟尘，直挺挺撞进包围圈，当场就撞翻了几人。一群喽啰大惊失色，乱成一片。
<br/>　　独角鬼一头雾水，不明白这是怎么回事。当然，他也不需要明白怎么回事，在此地碰到碍眼的，他一般杀了了事。
<br/>　　“来，与我杀了他！”
<br/>　　喽啰们一拥而上，拿着各种兵器，长矛、鱼叉，朴刀、哨棍往大汉背上扎。牛背上的程大雷口中大叫，不停说着：“大王饶命，不怪我，是牛惊了。”
<br/>　　他双手扶着牛角，身形狼狈不堪，显然已经被周围的阵仗吓破了胆。
<br/>　　但刘发财看出了些不对，这黑牛在众喽啰的围攻之下，前冲后撞，左突右进，看上去毫无章法。但扎向他的兵器，往往能在最关键时刻闪避。而黑牛随意在包围圈中冲撞，就能顶翻两个小喽啰，将包围圈撞出一道缝隙。
<br/>　　擅于欺骗的人，往往也能最快识破别人的伪装。
<br/>　　等待，我刚才想到了什么……包围圈被撞破了。那我还等什么，管这货是那条路上来的鬼，老子我先逃一条活命再说。
<br/>　　他抖擞精神，抓起腰刀厮杀着往外闯，此刻是性命之争，自然用尽全力。一般的小喽啰挡在他面前，自然被其一刀斩杀。
<br/>　　就这样边杀边走，硬生生撞开一条路，向远处逃去。
<br/>　　独角鬼怎能容他逃脱，拎着铁棍冲过去，兜头一棍裹夹着莫大的杀威，棍风都在耳边呼呼作响。刚才两人已经交过手，刘发财不是这独角鬼的对手。
<br/>　　他被逼得不停后退，只有招架之力，而无还手之功。眼看已经退无可退，心中难免有些绝望。
<br/>　　正这时候，一头黑牛撞了过来。独角鬼反应不慢，急忙躲避，虽然没有被黑牛撞个正着，但仍旧被带翻在地。
<br/>　　刘发财一看，此时不走更待何时，他转身就往外逃，吃奶的劲儿都用了出来。
<br/>　　独角鬼站起身，想要追杀刘发财，面前却总是被黑牛拦着。黑牛来来去去，像一堵黑墙隔在他面前。周围乱哄哄一片，黄尘荡起，迷迷糊糊中，不知谁在他脑袋上拍了一下。
<br/>　　啪！
<br/>　　然后，独角鬼晕了过去。
<br/>　　等独角鬼醒来的时候，发现眼前既看不见刘发财，也看不见那骑黑牛的壮汉，他们都不见了踪影。
<br/>　　刘发财跑得气喘吁吁，只恨爹妈没有多给两条腿。一口气不知跑了多久，来到一片桃树林中，眼见没人追来，他才松了口气。
<br/>　　“好汉留步，好汉留步！”
<br/>　　刘发财一激灵，回过头，见那骑黑牛的汉子追了过来。他下意识握住刀，装出若无其事的模样。
<br/>　　“敢问好汉是那程当家么？”
<br/>　　“呵，我的名字是与你说的么。你叫什么，打听这些做什么？”
<br/>　　“呵呵，鄙姓刘。”牛背上的粗糙汉子道：“名发财。”
<br/>(本章完)
