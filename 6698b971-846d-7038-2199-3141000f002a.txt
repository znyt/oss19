夏夕绾动作滞了一下。
<br/>
<br/>        陆寒霆的大手在她平坦的小腹上来回的摩挲了一圈，俊脸埋在她的秀发里，“我们没有做避孕措施…”
<br/>
<br/>        夏夕绾心里知道根本就不用做啊，因为她的时间不多了。
<br/>
<br/>        她转过小脸，水汪汪的看着他，“不要！我要给你生宝宝！”
<br/>
<br/>        话音刚落下，陆寒霆眸色一沉，搂着她的腰肢就将她抵到了墙壁上，声音发哑道，“绾绾，不许胡闹！”
<br/>
<br/>        “我就要给你生宝宝，你不想当爹地啊，我就不信如果我怀上了，你还让我把宝宝打掉。”
<br/>
<br/>        陆寒霆盯着她看，一双狭眸晦涩不明，又异常的炙烈，好像熔浆要将她融化了一样，许久之后他道，“绾绾，我们…一起去帝都吧，我们不要分开了，好不好？我会努力的治病的，我会努力的让自己变好的，或许我给你的未来不是最好的，但一定是我的全部，绾绾，我爱你，我不能没有你。”
<br/>
<br/>        陆寒霆是经过深思熟虑说出这番话的，他不想跟她分开了，也许是在他昏迷睁开眼看到她的时候，也许是她成为他的女人的那一刻，更也许就在刚才，她说要给他生宝宝的时候，总之，他又变得自私和贪婪了，有生之年，他都想霸占着怀里这个女孩。
<br/>
<br/>        他不能失去她。
<br/>
<br/>        夏夕绾白皙的眼眶迅速一红，她知道他能说出这番话意味着什么，而且他是一个深沉内敛的男人，感情从不外泄，可是刚才他告诉她，他爱她。
<br/>
<br/>        夏夕绾觉得够了，足够了。
<br/>
<br/>        她抽了一下通红的小鼻翼，没心没肺的笑道，“陆先生，跟你开了一个玩笑而已，你怎么当真了，我不会怀孕的，这几天都是我的安全期，放心吧，你不会喜当爹的。”
<br/>
<br/>        陆寒霆高大英挺的身躯直接一僵，刚才他卯足了全身的力气跟她表白，对她诉说着他们的未来，可是现在她一盆冷水扑了下来，让他僵在了当场。
<br/>
<br/>        大手缓缓松开了，他俊脸上其实并没有多少表情变化，只是淡淡道，“哦。”
<br/>
<br/>        说着他转身，来到床边，将床单给扯了下来，“你休息一下，我去洗。”
<br/>
<br/>        他抱着床单就走了。
<br/>
<br/>        ……
<br/>
<br/>        夏夕绾觉得他生气了，因为他全程洗床单，都不肯理她。
<br/>
<br/>        夏夕绾就屁颠屁颠的跟在他身后，他去哪儿她就跟到哪儿，这男人什么时候洗过床单了，今天完全是第一次，动作有些笨拙又有些可爱，当看到她那朵血梅被他揉在手心里洗搓时，她雪白的小耳垂还是红了。
<br/>
<br/>        将床单晾晒起来，外面已经准备好了午餐，这些婶儿们很有眼力劲，一天都没有打扰他们，还掐着点给做好了午餐，夏夕绾耳垂更红。
<br/>
<br/>        填饱了肚子，夏夕绾亲昵的挽住了男人的健臂，仰着澄亮的水眸望着他，“陆先生，我们出去约会吧，到附近的镇上看电影逛街，像普通的情侣那样认认真真谈个恋爱。”
<br/>
<br/>        陆寒霆还别扭着，没有看她，不过却将她柔软的小手裹在自己的掌心里，“恩，走吧。”
<br/>
<br/>        才走了一步，夏夕绾“啊”一声，停了下来。
