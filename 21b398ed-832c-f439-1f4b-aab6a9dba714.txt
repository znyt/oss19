青睐，到时候自然就不必再忌惮何家……”<br/>　　<br/>　　林世骁说到这里，一双老眼中，流露出浓浓的炙热之色，正要继续往下说，这时房门忽然被人推开，一名带着眼睛的中年男子，气喘吁吁的冲了进来，<br/>　　<br/>　　“老爷子，出大事了！”<br/>　　<br/>　　林世骁不由得眉头一皱，有些不悦道：<br/>　　<br/>　　“小刘，什么事这么慌张？没看到我在开会吗？”<br/>　　<br/>　　小刘一手扶着腰，狠狠咽了一口口水，方才道：<br/>　　<br/>　　“老爷子，从望江楼那边传来的消息，何家背后的那位于宗师，死了！”<br/>　　<br/>　　“什么！”<br/>　　<br/>　　听到这个消息，林家众人顿时全都傻眼了，林世骁更是“霍！”的一下，就站了起来，<br/>　　<br/>　　“这怎么可能？那位于宗师可是化劲大成的修为，据我之前得到的消息，即便是在化劲宗师里面，也是极强的存在，怎么会突然死了？”<br/>　　<br/>　　中年男子小刘，一脸苦笑道：<br/>　　<br/>　　“老爷子，这消息千真万确呀！现在整个云州上层都在传，说那位于宗师，被曹四爷背后的那位叶宗师一拳打爆，叶宗师不但废除了之前云州众大佬与何家的合同，还夺了林州何家一半家产……”<br/>　　<br/>　　还没等小刘的话说完，林世骁两眼一瞪，<br/>　　<br/>　　“你刚刚说什么？那位于宗师，是被曹坤背后的那位叶宗师所杀？这怎么可能！”<br/>　　<br/>　　自从那天，曹坤的地下擂台，出现化劲宗师的身影后，林家一直都在调查曹坤背后的那位宗师的消息，虽然始终未能查到此人的准确身份，但是有一点林世骁几乎可以确定，这位传说中的叶宗师，应该年纪不大，虽然未来的潜力无穷，但眼下的实力在化劲宗师中，应该处于垫底的位置。<br/>　　<br/>　　所以此刻听到小刘这个消息，林世骁几乎下意识的觉得不可能。<br/>　　<br/>　　小刘再次苦笑，<br/>　　<br/>　　“此事千真万确，属下就算有十个胆子，也不敢欺骗您呀！”<br/>　　<br/>　　林世骁摇了摇头，还是不相信，慌忙拿起一旁的手机，一连拨通了好几位云州大佬的电话，最终得到的回馈，与小刘说的基本一致。<br/>　　<br/>　　“噗通！”<br/>　　<br/>　　消息确认之后，林世骁整个人顿时如同虚脱了一般，一屁股坐倒在太师椅里，脸色煞白，喃喃道：<br/>　　<br/>　　“曹家！唐家！居然全都早早的结识了这位叶宗师，为何唯独我林家……”<br/>　　<br/>　　第三更！<br/>　　<br/>　　ps：有票的童鞋投点票票吧~~<br/>　　<br/>　　(本章完)<br/>　　<br/>　　//<br/>　　<br/>　　天才一秒记住本站地址：。手机版阅读网址：<br/>
