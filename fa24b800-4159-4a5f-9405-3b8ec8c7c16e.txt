　　元始想了想，首先将自己本门传承心法《玉清元始经》赐予每人，这是阐教的传承仙法，乃一教之根本，不能动摇。
<br/>　　其次，元始在玉虚宫中建立起两座楼阁，一座名为仙神楼。一座名为三界楼。
<br/>　　其中，仙神楼共分九层，每一层都包罗万象，阵法、傀儡、剑道、遁术、雷法、磁光、箭术……种种法门神通秘术尽皆都有，从低到高，越往上越加玄妙。这第九层自然是最玄妙的。
<br/>　　仙神楼，来着不拒，只要是昆仑山的生灵，尽皆可以进来学习。不过，正所谓道不可轻传，想要学习一门秘术，可以，但是需要通过相应的考验。每一门秘术神通都有对应的考验，度过考验者，才能学习到这些法门。
<br/>　　能在昆仑山待的生灵，元始自然是算在了昆仑三清一脉，或者是他徒弟的坐骑、或者徒子徒孙，或者哪个人的仆从、灵兽等等，总之，一切属于昆仑一脉的，他都允许进去仙神楼学习秘藏。
<br/>　　他这也是为了增加昆仑山的底蕴，正所谓肥水不流外田嘛！当然，在这些神通秘籍之中，他也设下了相应的禁制，防止有向外人泄露的可能。
<br/>　　而三界楼，顾名思义，就是他将三界有名大神通者们的各种仙法、神通、道法、秘籍、秘技、道藏收集在其中。炼体、练气、炼神、炼器、炼丹、布阵、杀伐、战斗、控兽、等各类顶尖神通典籍，就连元始所创的各种元始九印、《玄黄不灭》、《元始无名经》、《神元九转》等都被他放在其中，还有老子与通天的一些秘术神通，以及其他洪荒之中有名的神通。
<br/>　　以元始掌控天道因果天罚的能力，得到这些神通秘术并不困难。更何况，除此之外，他的元心印也能够轻易入侵一个人的内心，在神不知鬼不觉的情况下，轻易得到他人的神通。
<br/>　　这里收集的是三界真正的顶尖大神通与一些可怕的秘术，不可轻易学得即便是亲传弟子，学得数门后，元始也会禁止再学习。
<br/>　　仙神楼是昆仑山所有生灵都可以进去，但三界楼却必须是元始的亲传弟子或者得到元始允许的人，方能进去。
<br/>　　三界楼中的秘术神通，能够学到一门或者数门便已经是天大的机遇了，毕竟平常仙神能够学得一门手段，便足以仗之纵横三界了。
<br/>　　进入三界楼，学习秘术，同样要完成相应的考验，只不过这里的考验要比仙神楼中要高级艰难许多。
<br/>　　而进入三界楼的另一个限制就是，要能够在仙神楼第九层学到一种秘术。能满足此条件者，才有资格进入三界楼，因为三界楼中最低级的神通，都要比仙神楼第九层要高级。若是你连仙神楼第九层的法门都弄不到手，你自然没有资格去学习三界楼中的手段。
<br/>　　至于如何培养门下弟子们对阐教的凝聚力或者依赖性、以及弟子之间的友情，师兄弟之间的相互帮助，弟子对老师的感情等，就要从这些考验之中下手了。
<br/>　　元始不会去管自己的弟子今后是善是恶，他只管弟子对门派的归属性。
<br/>　　最后，元始立下五百年一小比，三千年一大比的规定，规定自己门下弟子要进行比试争斗，比试胜利者，赐下奖励。
<br/>　　这样，可以使门派中形成一种良性竞争，可以不断督促门下弟子们努力修炼。因为同为老师的弟子，他们谁也不想在比试中输掉，被别人嘲笑。
<br/>　　作为拜师的礼物，元始每人送他们一件灵宝，至于以后的灵宝，可就要看他们以后对表现在再决定，分别给谁。
<br/>　　最后，元始立下出师条件，只有他看弟子表现，最终经过他允许了，弟子们才可以出师。
<br/>　　弟子们初次拜师，元始自然是要为他们讲道一番。故而，百年之后，元始在玉虚宫中摆下道台，为弟子们讲道三百年。
<br/>　　这些人来拜师，不就是要听圣人讲道吗，如今得以实现愿望，他们纷纷激动不已。能够被元始收下，他们都是天资福缘深厚者，元始所讲之道，都是从最简单的讲起，他们听得如痴如醉，片刻时间便沉浸在无边道韵之中，不可自拔。
<br/>　　……
<br/>　　靠近首阳山的一处山脉间，一个绝色女神身穿一袭银装，头发略显散乱，脸色苍白无比，她化为一道银光，飞驰而过。在她的身后，有五道黑色乌云，妖气滚滚，紧追着她而来。
<br/>　　显然，这个绝色女子正在被人追杀。
<br/>　　“嗖——”
<br/>　　两道流光飞过，两者紧追不舍。不知过了多久，女子终究是因为受伤不敌，速度渐渐慢了下来，而后面的五道乌光，却是追了上来。
<br/>　　轰！
<br/>　　女子面前一阵晃动，一道神通阻击在女子面前，女子被迫停了下来。瞬息之间，另外五道乌光便追了上来。
<br/>　　乌光落地，化为五个壮汉，他们身材魁梧，妖气沸腾，手中各自持有兵器，缓缓将女子包围起来。以他们身上的气息来看，他们都是大罗金仙。
<br/>　　“嫦羲，东皇陛下派遣我等前来捉你，你还是不要做无谓的挣扎了！”为首一人上前一步说道。
<br/>　　女子眼神坚定，不为所动，仍然没有放弃逃跑的机会。
<br/>　　她正是太阴星的嫦羲，因为拒绝帝俊的求婚，东皇太一怀恨在心，本来他是带兵前去太阴星捉拿嫦羲的，但奈何去了太阴星后，找不到丝毫的人影，甚至就连太阴之主的月桂秘境都隐匿虚空了，他搜寻一番后，并没有发现什么踪迹。
<br/>　　正在气头之上的太一听到有手下前来报告，发现嫦羲的踪迹出现在洪荒之中，顿时，他派遣一队妖神前来围捕。
<br/>　　经过一番厮杀，再嫦羲斩杀两个大罗金仙的妖神后，她因不敌逃跑，之后才有眼前这一幕。
<br/>　　“看来你是敬酒不吃吃罚酒，东皇陛下的命令不可违背，抓不到活的，死的也行，杀！”壮汉见嫦羲不为所动，便一声大喝，五个妖神手持兵器，杀向嫦羲。
<br/>　　“哧！”
<br/>　　嫦羲挥手连斩五下银色剑气，剑光阴冷如月，寒气逼人，带着凌厉之气，斩向五妖。五妖也不示弱，他们可不会怜香惜玉，挥动兵器，神通法术一道道的向着嫦羲轰去。
<br/>　　轰隆隆！
<br/>　　神通爆炸，威力无边，六人大战起来，相互僵持。
<br/>　　片刻后，嫦羲终究是不敌五妖，她虽然是大罗金仙后期，但敌人也是大罗金仙啊，而且还是五位，经过一番搏杀，她已经能够察觉到自己越来越虚弱了。
<br/>　　怎么办？难道要死在这里？此刻嫦羲心中焦急万分。
<br/>　　轰！
<br/>　　“灭神！”
<br/>　　一道冷酷的声音在她耳边响起。
