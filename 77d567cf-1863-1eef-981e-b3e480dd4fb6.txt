　　远远的跟在两人后面，刘哔很快找到了四目道长的住所，那是几间有些年头的房子，用一圈篱笆围着，坐落于山间，位置到也算是隐蔽。
<br/>　　目送着了两人进去，刘哔在屋外的树梢上站了一会，便就近找了一处地方，用四周的植被搭起了一座简易帐篷。
<br/>　　在末世生存了这么长时间，搭个帐篷并不是什么难事，只是夏末的蚊虫比较多，让他有些厌烦。
<br/>　　他讨厌虫子，尤其是讨厌蚊子。
<br/>　　山间的蚊子大都有指尖大小，长着狰狞的口器，这种东西一有机会就会吸食你身体内的血液，可不会管你是什么物种。
<br/>　　在他原本的世界，变异蚊群也很少有人愿意招惹，那种东西，一小会就会把你吸成干尸。
<br/>　　时间还尚早，搭完帐篷的刘哔拿出尸丹，按照紫女教授的方法将其研磨成粉末状，混杂着四两糯米粉放在一个罐子当中。
<br/>　　糯米和罐子是他来时从过路小镇上买的，除此之外，他还准备了一些黑狗血。
<br/>　　将罐子里的粉末搅拌均匀之后，刘哔从空间戒指中拿出黑狗血，犹豫了一下，全部倒入了进去。
<br/>　　只要浸泡一段时间，就可以拿出来来晒干了，二十九天之后，再将其洒在画卷上就可以解除紫女的封印。
<br/>　　这个月有三十一天，时间刚好足够，只要不出意外，这项任务也就算完成了。
<br/>　　把罐口封好，刘哔将其收入空间戒指中，谨慎的在四周设置了几个简易陷阱，确定没有问题之后，回到帐篷开始了修炼。
<br/>　　这些陷阱都是用就地取得的材料制作成的，并没有什么攻击性，唯一能起到的作用，也只是警戒而已。
<br/>　　虽然这个地方很偏僻，但难保不会有人过来。
<br/>　　未来二十九天的时间里，他基本上不会离开这里，所以这些基础的设施，都要提前进行准备。
<br/>　　时间缓缓流逝，大致晚上七八点的时候，刘哔修炼的动作微微一滞，他听到了远处传来的一阵嘶吼声。
<br/>　　这个声音，他再也熟悉不过了，是丧尸的声音，大概是自己白天留下的丧尸病毒起到作用了。
<br/>　　“叮~因为宿主强行改变为降临位面的剧情，导致《僵尸叔叔》位面发生崩溃，变为不可进入位面。”
<br/>　　也是在此时，系统提示音从刘哔心中响起，让他嘴角轻轻抽动了一下。
<br/>　　他的猜想果然没错，同世界不同电影位面的剧情人物，果然都是可以连接到一起的。
<br/>　　只是如果强行改变剧情的话，会导致这个位面无法进入，不过这并不能给他造成多大的影响。
<br/>　　利用这点规则，他足够去做很多事情，比如，提前获得另一个位面的资源。
<br/>　　按照现在的时间线来说，现在正是《僵尸叔叔》剧情刚开始的地方，在自己离开之前，还能顺手杀死那头被雷劈过的僵尸。
<br/>　　虽然是民国初期，战乱横行，到处都在死人，但高等级的僵尸和鬼怪也不是那么容易寻找的，刘哔不想放过任何一个提升实力的机会。
<br/>　　……
<br/>　　之后二十九天的时间里，刘哔都是在修炼中度过的，期间有几次感应到低级鬼物的气息，便停止修炼寻去，出手解决了几头鬼怪。
<br/>　　在这期间里，四目道长回来了，看到变成丧尸，被老和尚绑起来的徒弟，不免极为悲恸，和老和尚打了一架之后，下山去找寻击杀他徒弟的凶手。
<br/>　　趁着他们战斗正酣的时候，刘哔带着面具，出其不意的击杀了四目道长带回来的那些小僵尸，收获了一笔积分之后，转身离去。
<br/>　　虽然四目道长和老和尚正在战斗，但只要他相信，只要自己出手偷袭，两人绝对会毫不犹豫的联手对抗自己，两个筑基期的修士，他没有必胜的把握。
<br/>　　由于出手时机和过程都把握的十分完美，四目道长也根本来不及阻止，只能眼睁睁的看着刘哔将他刚带回来的那些僵尸杀掉之后，飘然离去。
<br/>　　但他也没有将杀掉自己徒弟的凶手和这个人联系到一起，这个世界上，仇恨僵尸的人多了去了，这种事情也不只是发生了一次。
<br/>　　不知道有多少家庭，都是因为僵尸变得支离破碎。
<br/>　　当然，这些收获对于那头剧情僵尸来说，都还算不上什么，那是一个筑基期的僵尸，被雷劈过之后更是实力大涨，就算刘哔也花费了不小的力气。
<br/>　　甚至，还受了点伤，但击杀完那头僵尸之后，他本来已经见底的积分，又积攒到了两千多点。
<br/>　　这些积分，他都留着没有动用，刘哔想要等到进入下一个世界之后，在好好计划一下，需要兑换什么东西。
<br/>　　最后一天的夜里，刘哔没有修炼，而是将之前晒在外面的尸丹粉给收了起来，摆在了画像面前。
<br/>　　明天他就要离开了，但这解除封印的材料，还没有制作完成。
<br/>　　按照紫女所说，这份材料完成之后会变成鲜红色的血粒，如同刚刚流出的新鲜血液一般，但现在的颜色，还是有些发暗。
<br/>　　展开画像，紫女纤细的身影显露了出来，知道自己即将要破除封印了，她脸上的表情也是有些激动。
<br/>　　“这样就算可以了吧？”看了一眼紫女，刘哔蹲在红色的粉末堆旁，有些疑惑的开口问道。
<br/>　　他也是第一次做这样的事情，有些地方还是不太熟悉。
<br/>　　压抑住内心的兴奋，紫女目不转睛的盯着红色粉末，道：“再等等，等颜色彻底变成鲜红色的时候，就可以洒在上面了。”
<br/>　　应了一声，刘哔也没有心急，蹲在一旁静静的等待着，此时两人都没有攀谈的欲望，只等待解封材料彻底转换完成。
<br/>　　同时，他也做好了和紫女开战的准备，虽然他和紫女并没有什么冲突，但也难保会有意外发生。
<br/>　　不管在任何情况下，刘哔随时都会保持一种戒备的态度，这是在十年的时间里，用无数鲜血和伤痕换来的习惯。
<br/>　　在那样的环境，如果不小心谨慎一些，他也不可能活的这么长久。
