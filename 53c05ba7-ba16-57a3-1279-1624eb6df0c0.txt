夏夕绾这话一出，所有人倒吸一口冷气，什么，她竟然公开挑战T大的校花学霸厉嫣然？
<br/>
<br/>        厉嫣然不但人气高，名声好，她私下也很努力的，每一次她提交的答案都很漂亮，她是周副校的心头爱，抱以了厚望，还没有人敢挑战她。
<br/>
<br/>        现在夏夕绾初来乍到，被T大退货不说，还进了考试垫底的A大，现在她竟然公然叫嚣厉嫣然。
<br/>
<br/>        这一幕真的让人大跌眼镜，大家纷纷震惊的看着她，她哪来的自信？
<br/>
<br/>        厉嫣然勾唇笑了，“夏夕绾，你说真的，你真的想挑战我？不如你再回去好好想一想吧，我感觉我跟你擂台对打，真的是欺负了你。”
<br/>
<br/>        厉嫣然是自信满满的，在联考上她真的瞧不上夏夕绾，这可不是比脸蛋比漂亮，而是比知识的积累比智慧。
<br/>
<br/>        现在璀璨的晨曦向大地镀下一层瑰丽的金黄，夏夕绾沐浴在阳光下，澄亮的水眸从容的看着厉嫣然，“厉嫣然，如果你没有听清，那我再说一遍，我要在即将到来的联考上挑战你，你敢不敢应战？”
<br/>
<br/>        厉嫣然被夏夕绾现在的气魄给震住了，她哼了一声，“好，夏夕绾，我应战，让我们在联考上一较高低！”
<br/>
<br/>        “好！”夏夕绾点头。
<br/>
<br/>        范恬受了夏夕绾的鼓舞，她觉得夏夕绾都敢挑战厉嫣然了，那她还有什么不敢的，她迅速站了出来，“绾绾，我怎么可以让你孤军奋战，昨天我们说了，荣辱与共，我虽然不能挑战厉嫣然，但是这一次联考，我要拼尽全力为我们A大冲进前五！”
<br/>
<br/>        很快就有其他人站了出来，男生或者女生，他们一个接一个的站在了夏夕绾的身边，“我也加入，想当年我也是以全省十二名的成绩进入了A大，这两年我都把书本丢了，不知道现在将书捡起来会不会觉得陌生！”
<br/>
<br/>        “我也加入，我成绩可不好，勉强进了A大，以前我没有好好学习，不知道好好学习的我会不会吓我自己一大跳！”
<br/>
<br/>        “我也加入，为了A大的荣誉而努力，多考一分是一分！”
<br/>
<br/>        大家纷纷站出来，声音铿锵有力，他们要跟夏夕绾一起奋斗，不知道日后他们知道夏夕绾就是刷新了整个医学界历史成绩的15岁双博士后，他们会不会笑话自己太天真，竟敢跟T大拼了。
<br/>
<br/>        对面T大的学生们面色变了又变，这么一幕让他们想起了昨天篮球赛场上团结在一起的A大，当时的他们爆发出了惊人的潜力，现在他们竟然又来这一招了。
<br/>
<br/>        T大的学生们突然有点害怕，他们怕历史会重演。
<br/>
<br/>        厉嫣然用力的拽了一下拳，她不得不承认夏夕绾很容易感染到别人，她进了A大，并没有被A大改变，而是改变了整个A大。
<br/>
<br/>        她身边的那些学生们，以前都在混日子，浑浑噩噩的，但是现在站在她身边的那些学生们神采奕奕，脸上都是风采。
<br/>
<br/>        厉嫣然目光冷了冷，“那好，我们在联考上见！”
<br/>
<br/>        厉嫣然带着T大的人离开。
