        明明很痛，但这样的痛，于她而言，已经开始麻木了。
<br/>
<br/>        鲜血顺着掌纹滴落在了地面上，童筱颖无动于衷地看着傅夜沉愤然离去的背影，心里一片惘然，眼泪止不住地从眼角溢出，“啪嗒啪嗒”地往下掉。
<br/>
<br/>        后背沿着墙壁，身子缓缓往下滑落，无力地跪坐在地上，她双手掩面，失声痛哭了起来。
<br/>
<br/>        什么时候，她可以不用为他而心痛？
<br/>
<br/>        不知道是不是因为吃了那事后药产生的副作用，还是因为过度悲伤而引起的胃部不适，童筱颖疲惫地回到家后，吐了一晚，只差没把胆汁给吐出来。
<br/>
<br/>        白天的时候她就觉得很恶心，一直没胃口吃东西，后面好不容易吃的东西，现在全给吐了个一干二净。
<br/>
<br/>        而这一晚，傅夜沉一夜未归。
<br/>
<br/>        翌日，童筱颖脸色惨白地继续去上班，刚走进办公室里，就听到女同事们拿着手机围在一起在议论傅夜沉和苏漫雪深夜共宿酒店的八卦新闻。
<br/>
<br/>        童筱颖原本惨白的脸色瞬间铁青，她踱步上前，夺走了其中一个女同事手里的手机看了看，网页上的照片是偷拍的，虽然不清晰，但是能辨认出被偷拍的女主角是苏漫雪，而男主角……
<br/>
<br/>        因为是侧面，光线不大好，相片上的男人看起来确实有点像傅夜沉，但是又有些像傅昼景。
<br/>
<br/>        童筱颖将手机还给了那个女同事，还被她劈头盖脸地谩骂了一句“有病啊！”
<br/>
<br/>        她的这声咒骂，让她被贴上了“傅夜沉前女友”的标签。
<br/>
<br/>        这一天才开始，员工私聊群里就还有人疯传，她童筱颖是照着苏漫雪的样子整了容。
<br/>
<br/>        面对这些流言蜚语，童筱颖倒是心如止水的继续工作。
<br/>
<br/>        只是她不知道是得罪了谁，被身边的同事给孤立了。
<br/>
<br/>        中午在员工餐厅里吃饭的时候，没有人愿意跟她一桌。
<br/>
<br/>        童筱颖打好了饭菜，一个人坐一张桌子，埋头细嚼慢咽，其实她是很没胃口才一口一口吃得很慢。
<br/>
<br/>        余光里，忽然有一抹阴影闪过。
<br/>
<br/>        童筱颖下意识地抬眸，只见傅夜沉身着笔挺的高定黑色西装，双手端着和她一样的员工套餐，淡定地在她对面坐下。
<br/>
<br/>        “我昨天晚上在自己的办公室里睡的，直到现在都没有离开过大厦一步。”他突然开口说道，然后夹起盘子里的一个鸡腿，放在了她的米饭上，接着说道，“我让郑特助特意吩咐了员工餐厅厨房的掌厨，每道菜里禁止放味精。”
<br/>
<br/>        童筱颖垂下眼帘，只是听着，然后自顾自地默默吃饭，一句话也不想跟他说。
<br/>
<br/>        其实他不用跟她解释那么多，苏漫雪那八卦新闻的男主角是谁，她心里有数。
<br/>
<br/>        “监视你，是为了保护你。”顿了一下，他又接着说道，语气平淡得就像是在跟她拉家常一样。
<br/>
<br/>        他现在正处于非常时期，而且他频繁跟她接触，她一定早就被人给盯上了。
<br/>
<br/>        她是他唯一的软肋，任何人都可以拿她来威胁他。
<br/>
<br/>        所以，他才派自己的人暗中保护她。
<br/>
<br/>        只是，真正监视她的人，不是他傅夜沉，而是他的顶级上司及干爹夏阳枭。
<br/>
<br/>        她童筱颖作为他傅夜沉的合法妻子，反正是瞒不住的，所以他已经打了报告向上级汇报了，她才会被监视。
<br/>
<br/>        至于监视她的理由，夏阳枭说是为了他傅夜沉好，但他心里，总感觉夏阳枭其实另有目的。
<br/>
<br/>        所以，他命令监视筱颖的那个手下，不管筱颖有什么动静，都先向他汇报，然后选择性地再向夏阳枭那边汇报。
<br/>
<br/>        昨天她去药店，暗中监视她的人，只是怀疑她买的药会对他傅夜沉不利，哪知道她买的只是事后药。
<br/>
<br/>        他不知道，他的解释，她有没有听进去，但终归气是气，事实是事实。
<br/>
<br/>        她还小，她可以任性，但他傅夜沉不小了，不可以妄为。
<br/>
<br/>        而他的这句解释，在童筱颖听来，让她觉得他只是在为他自己那不信任她的行为强行辩解罢了。
<br/>
<br/>        “保护我？保护到要明确知道我每一分每一秒都在做什么吗？是不是我跟你上床的时候，也有人暗中监视着啊？是不是怕我对你暗中捅刀子？给你下毒药？我差点忘了！大哥，你的身份可矜贵着呐！一个人掌控这么大的集团，手下那么多人靠着你吃饭，傅氏集团若是垮了，那得多少人失业啊！噢，对了，大哥还像还有另一个连我都不知道的身份吧？你和连医生是不是一直在追查着与‘药’有关的案子啊？你们追查的那个‘药’与我外婆有关系对吗？我是我外婆抚养长大的，你们自然也要把我列入怀疑的对象喽！说白了，你监视我，不就是怀疑我会对你不利吗？”童筱颖振振有辞地说道。
