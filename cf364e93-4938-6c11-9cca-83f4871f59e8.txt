　　“算账？”慕容克冷笑一声，“紫炎，你袭击我盛宝商行时…”
<br/>　　“慕容克，还是先算清楚你我的账吧。”慕容克话未说完，已被叶铭冷声打断了。
<br/>　　“叶铭，我与你可没什么恩怨。”慕容克沉声道。
<br/>　　“没有？刚才为何阻拦我救我的师弟们。”叶铭冷声质问着。
<br/>　　“事关大局，岂可轻易让你出手，我不也没有救我盛宝商行的武者吗？”慕容克反驳道。
<br/>　　“放屁。”叶铭喝道，“你盛宝商行的护卫队，人数过百，有数个洞玄九重武者，自保力过人”
<br/>　　“而我裂天剑派弟子，不过二十人，最强者，不过林劲四人，皆是洞玄四重。”
<br/>　　“刚才的兽潮，大多是三级妖兽，四级妖兽虽然也多，但普遍在洞玄三重以下实力。”
<br/>　　“洞玄九重实力的妖兽，不过10头。”
<br/>　　“我裂天剑派弟子，独自承担了5头。”
<br/>　　叶铭冷眼看着慕容克。
<br/>　　“你分明是想让我们牵制住那些妖兽，好让你们的人压力大减，伤亡也更少。”
<br/>　　刚才城墙下，人类猎妖师和武者数以万计。
<br/>　　有十八城的士兵，各猎妖队，本地家族的武者，盛宝商行的护卫队等人。
<br/>　　他们一个个都是本土的势力，自然武者力量更多，动辄过百人联手。
<br/>　　而裂天剑派，不过是叶铭带着一群新弟子过来历练，只有二十人。
<br/>　　就是这二十人，竟在抵抗了最强的5头妖兽，而且还面临其它数十头妖兽的围攻。
<br/>　　他们虽是天才，但终究还未成长起来。
<br/>　　花剑、何苍等人，早已重伤。
<br/>　　若非林劲、铁牛四人身怀灵器，本身战力过人，硬生生挡住围攻的妖兽，他们早死了。
<br/>　　而且，林劲四人也并未能在5头洞玄九重实力的妖兽，外加数十头普通洞玄境妖兽的围攻下支撑太久。
<br/>　　后来要不是萧逸及时赶到，他们必定葬身妖兽腹中。
<br/>　　“给我听好了。”叶铭的目光，陡然间变得如雪般冰冷。
<br/>　　“你们，有的是十八城的城主，职责所在，必须守城；有的，是本土势力，必须保住绿光城，免得影响家族利益。”
<br/>　　“你们，一个个都有责任。”
<br/>　　“而我裂天剑派弟子，不过是来历练的。只是身为武者，我们才没有退缩离开，留在此地一同与你们抵御妖兽。”
<br/>　　“但，再有下次，罔顾我裂天剑派弟子的性命，休怪我叶铭翻脸。”
<br/>　　叶铭一字一顿，态度强硬。
<br/>　　萧逸在一旁看着，暗暗点头。
<br/>　　他曾听易老说过，叶铭此人，生性温和，平易近人，且处事能力极强，乃是药堂长老高徒，也是爱徒。
<br/>　　今日听其铭一番言辞，确实如此，能力过人。
<br/>　　而且，他的话语中，无不是对师弟们的关心和爱护。
<br/>　　倒是颇有师兄的风范。
<br/>　　“翻脸？好大的口气。”慕容克手段狠辣，摆明了是想让裂天剑派的弟子多分担一些危险。
<br/>　　叶铭怡然不惧，“慕容克，收起你那些威胁。”
<br/>　　“破玄境之下第一武者？哼，不过笑话罢了。”
<br/>　　“我要杀你，并非难事，你最好不要触我底线。”
<br/>　　叶铭的语气中，透漏着无比自信。
<br/>　　“哟，裂天剑派内门首席，好大的威风阿。杀我？你也太不把我们东荒十八城的武者放在眼里了。”
<br/>　　“我说得对吧，城主们。”
<br/>　　慕容克，竟是话语中和十八城城主站到了同一阵地。
<br/>　　十八城城主是本土势力，自然帮慕容克，也不想自己的势力遭受太多损失。
<br/>　　“叶铭，听闻你这个内堂首席，实力过人。今日如此狂妄，是否要和我们交手一番？”
<br/>　　几个城主冷声说着。
<br/>　　“你们…”叶铭脸色一变，没想到慕容克如此阴险，竟让得十八城城主联手挤兑他。
<br/>　　现在是妖兽攻城的危急时刻，若待会儿的战斗，他们的手下处处针对，那么师弟们就麻烦了。
<br/>　　恰在此时，一道人影淡淡地站了出来。
<br/>　　“诸位，是否也要和易某交手一番？”萧逸戏谑地说道。
<br/>　　“紫炎？”刚才那几个说话的城主皱起了眉头。
<br/>　　“易霄执事，误会罢了。”猎妖殿的人员立马出来和稀泥。
<br/>　　“没有误会。”
<br/>　　嘭的一声，萧逸手中紫炎涌动。
<br/>　　那炽热的高温，顿时让得巨大城墙上坚硬无比的铁岩瞬间融化。
<br/>　　铁岩城墙，无数年来，无数次妖兽攻城，尚且只能让铁岩留下痕迹，却无法完全破坏。
<br/>　　此刻，竟是瞬间融化。
<br/>　　“这…”那几个城主，吓得后退了几步。
<br/>　　“紫炎，误会罢了，何必这么认真。”那些个城主们，连忙讪笑几声，再不敢冷言冷语。
<br/>　　同时，他们也在疑惑着，从未听闻紫炎与裂天剑派有关系，怎么此刻竟出来帮叶铭了？
<br/>　　“哼，最好是误会。”萧逸收回了紫炎。
<br/>　　而后看向叶铭，拱了拱手，道，“叶兄，一年前在百武城阔别，可是许久未见了。”
<br/>　　“易兄，客气客气。”叶铭连忙道，“当日在白家你我炼药切磋，可是让在下好生惊叹阿。”
<br/>　　十八城城主们，心中顿时恍然，原来紫炎和叶铭私下交情不错，难怪会帮叶铭。
<br/>　　这时，妖兽退去后，城墙下的武者，也陆续退回到城内。
<br/>　　只是，城墙下那一具具已然冰冷的尸体，深深刺激着在场所有人的双眸。
<br/>　　妖兽攻城，守护绿光城，竟是惨烈到这个地步。
<br/>　　刚才那一战，数万武者猎妖师，足有一成以上永远留在了战场中。
<br/>　　“叶兄。”萧逸忽然道，“大伙儿都有伤势，你我都是炼药师，先给他们稳下伤势吧。”
<br/>　　“好。”叶铭点点头。
<br/>　　那些个城主们闻言，顿时变得和颜悦色起来。
<br/>　　刚才的大战，他们的手下受伤不少，而且个个都伤势颇重。
<br/>　　要是有两位高品炼药师帮他们治疗，自然是极好的。
<br/>　　叶铭，药堂首席的名号，谁都知道；紫炎易霄，炼药的手段，更是如雷贯耳。
<br/>　　他们很清楚，自己等人刚才如此挤兑叶铭，要不是易霄出言，叶铭绝不会帮忙治疗。
<br/>　　此刻，他们看向易霄的眼光中，充满了友好。
<br/>　　萧逸也注意到他们的眼光，但并不为意。
<br/>　　他只是找个借口，好将完美丹药给林劲等人罢了。
<br/>　　嗖，萧逸挥手一弹，十数个玉瓶，弹到林劲四人手上。
<br/>　　再随后一弹，一个个玉瓶弹到其他武者手上。
<br/>　　当然，给其他武者的，只是寻常丹药。
<br/>　　一个时辰后，两人配合其他势力的三品炼药师，大致稳住了各个武者的伤势。
<br/>　　城墙上，势力最强的那批武者，再次聚合在了一起。
<br/>　　“易某初来咋到，还不清楚现在的局势，能否说明一下。”萧逸说道。
<br/>　　“当然。”一个城主笑着走了出来，“有你紫炎的加入，我绿光城此次抵御妖兽，必然把握更大。”
<br/>　　“哦，对了，在下是这绿光城的城主，柳苍崖。”
<br/>　　柳苍崖，洞玄九重武者，绿光城城主，也是最先发现兽潮，率先抵抗，并通知其余势力的人前来协助。
<br/>　　所以他对情况最了解。
<br/>　　“说正事。”萧逸沉声道，“那些妖兽，为何忽然退去了？”
<br/>　　“额，这个不知。”柳苍崖回答道，“三天前，兽潮毫无征兆地来袭，我马上组织手下兵力以及武者阻挡。”
<br/>　　“不过，无论那些妖兽是占上风还是下风；每隔五个时辰，就会出现一声震天兽吼。”
<br/>　　“兽吼一出，它们就会马上撤退。”
<br/>　　“哦？”萧逸眉头紧皱，并眺目远视。
<br/>　　远处的妖兽森林内，兽影重重，黑压压一片，显然不下十万之数。
<br/>　　而且，更远处，森林内，还源源不断地有妖兽涌出。
<br/>　　恐怕，要不了半天，妖兽的数量，将会翻个一番。
