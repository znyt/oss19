夏妍妍真的来了，她今天穿了一件粉色连衣裙，娇美漂亮的小脸上挂着笑容，看到几个熟人她还声如黄莺的叫了一声“师哥”。
<br/>
<br/>        夏夕绾听得鸡皮疙瘩都起来了，但那些男人受用极了，纷纷夸赞道，
<br/>
<br/>        夏妍妍真不愧是海城第一名媛皆天才少女，让人一见倾心啊。

<br/>        你看那个夏夕绾，从乡下回来的土包子，果然不能跟夏妍妍相比。
<br/>
<br/>        夏妍妍这么漂亮，可是我们枢密的院花了吧。
<br/>
<br/>        夏妍妍拿着饭盒坐了下来，她佯装没听到这些话，动作优雅的吃饭，但是她的唇边早勾起了一抹笑意。
<br/>
<br/>        这时赵立英也过来了，双双迅速露出了害羞又兴奋的表情，“赵组长，坐在这里吃饭吧。”
<br/>
<br/>        夏夕绾看了双双一眼，双双喜欢赵立英？
<br/>
<br/>        这时有人起哄道，“赵组长，你的未婚妻在叫你呢，其实双双长得还不错，就是胖了一点哈哈。”
<br/>
<br/>        双双是赵立英的未婚妻？
<br/>
<br/>        夏夕绾刚得知。
<br/>
<br/>        赵立英十分嫌弃的看了双双和夏夕绾一眼，然后拿着饭盒坐在了夏妍妍的对面，他将新买的草莓奶昔放在了夏妍妍的手边，“妍妍，这个是我为你买的，你看好喝不好喝。”
<br/>
<br/>        “赵组长，谢谢你哦。”夏妍妍若有似无的向夏夕绾和双双那里看了一眼，炫耀的意思。
<br/>
<br/>        夏夕绾没有理夏妍妍，她看向双双，“双双，赵立英是你的未婚夫？”
<br/>
<br/>        双双有点羞答答的，“恩，我们两家有婚约，不过赵组长好像不喜欢我，这也很正常，我这么胖，大家都不会喜欢我这样的胖妞。”
<br/>
<br/>        因为胖的原因，双双十分有自卑感，她在枢密也没有交什么朋友，就喜欢摆弄自己的花花草草。
<br/>
<br/>        夏夕绾勾了一下红唇，“谁说的，我就觉得胖妞很可爱，我可喜欢双双了。”
<br/>
<br/>        双双双眼一亮，“我也喜欢夕绾，以后我们就是好朋友了！”
<br/>
<br/>        夏夕绾带着双双离开了，夏妍妍喝了一口草莓奶昔，然后柔声道，“赵组长，我看绾绾挺刻苦的，她一定是回药房去记那些药材了，说不定…三日后她还真能过关呢。”
<br/>
<br/>        赵立英一顿，“怎么可能？”
<br/>
<br/>        夏妍妍看着窗外，“我觉得我们研究院的卫生还不够好，如果有人能拖拖地擦擦窗户就好了。”
<br/>
<br/>        赵立英迅速明白了夏妍妍的意思，他起身就去办了。
<br/>
<br/>        夏妍妍惬意的喝着手里的草莓奶昔，她拿出手机打开了枢密的论坛，论坛的横幅就是夏妍妍晋升为枢密新院花！
<br/>
<br/>        有人放了她的图，她穿着粉色长裙沐浴在璀璨的晨曦里，娇柔甜美，楚楚动人。

<br/>        夏妍妍真的好美啊，我来猜猜谁能虏获这位新院花的芳心？

<br/>        大家赶紧去追啊。

<br/>        陆总为了夏妍妍壕掷十二个亿，夏妍妍早已经被陆总给预定了。

<br/>        我们只能仰望了。
