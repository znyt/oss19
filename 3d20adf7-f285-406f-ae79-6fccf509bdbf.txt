　　第627章
<br/>　　其实陆洛也并不在意程大雷会派谁到凉州城作为人质。但送给程大雷一座城池，如果不提出自己的要求，程大雷也不会相信。
<br/>　　另外，假若能有位人质留在凉州城，以此掣肘程大雷，也是一件好事。
<br/>　　程大雷目光在大厅内转了一圈，没想好让谁去做人质。按道理来说，该是将程大雷的子女送到凉州城。不过程大雷没有子女，倒也省了不少麻烦。
<br/>　　程大雷长叹口气，道：“目前我们这里人手紧缺，当然也希望能派个人调和两家合作事宜。我有个人选，陆先生考虑一下是否可行。”
<br/>　　“谁？”
<br/>　　“薛问婷。”
<br/>　　程大雷缓缓道出三个字，这可是相府的千金，崔相视若掌上明珠。她还一直留在蛤蟆城，倒是可以派上些用场。
<br/>　　陆洛微微迟疑，最后道：“这件事我无法做主，还需回去和我家王上商量。”
<br/>　　“不着急，什么时候等你们商量好了，再谈也不迟。”
<br/>　　陆洛快马加鞭回到凉州城，与宋伯康商议此事。兹事体大，也并非一天两天就可以定下来的。
<br/>　　几乎他刚走，程大雷就摊开凉州地图，刘悲几个人凑过来，大家一起寻找朔方城的位置。
<br/>　　程大雷早就派出人手，绘制整座凉州的地形图。目前他虽然还没有取宋伯康而代之的心思，但准备工作还是要做的。
<br/>　　相比蛤蟆城来讲，朔方城是一座大城，蛤蟆城的前身琴川关，只是一座边关，主要发挥军事作用。程大雷折腾这几年，总算是有了些起色。但在人口、资源、位置等方面，都不能和朔方城相提并论。
<br/>　　当然，蛤蟆城的防御肯定强于朔方城，但程大雷有能力将蛤蟆城打造得铁桶相仿，未必不能经营好一座朔方城。
<br/>　　“只是，宋伯康为什么愿意将朔方城送给我们，这说不通呐？”刘悲皱眉思索道。
<br/>　　“唉，他要吃掉百里胜，自然不愿意我们插手，送给我们一座城，只不过买我们一个按兵不动。”和珅兴奋道，可以扩充地盘，其中最高兴的便是他，他的野心终于迈出了第一步。
<br/>　　“话是如此说，不过这天上掉下来的馅饼，好看未必好吃呐？”程大雷道：“总是要弄明白，他们心里打的什么主意。”
<br/>　　阿喜得到命令后，凉州城安插的探子立刻展开行动，刺探这件事的内情。不过，这件事做得极其隐秘，凉州城内也仅限于宋伯康与陆洛二人知道，阿喜手下的探子打听，最后也没打探到什么有用的消息。
<br/>　　凉州城方面又派人来过几次，双方进行过密切接触，目前看来，这件事差不多已算是成了。
<br/>　　到这一日，陆洛又来到蛤蟆城，一直被引到城主府的花厅，程大雷坐在厅中饮茶。
<br/>　　“恭喜程当家，贺喜程当家……”陆洛刚来便急忙道喜。
<br/>　　“陆大人请坐，不知我喜从何来。”程大雷摆摆手。
<br/>　　“我家王上已经答应不需要蛤蟆城派出人质，同意由薛姑娘交换朔方城。”陆洛。
<br/>　　“喔。”这倒是出乎程大雷的预料：“凉州王当真答应了？”
<br/>　　“答应是答应了，不过我家王上还有一个要求？”
<br/>　　“请讲？”
<br/>　　“此战需要蛤蟆城的协助，程当家需要派出一批人马，帮助我们打赢这一战。”
<br/>　　这要求也并不算过分，程大雷总要交给宋伯康一些东西，令他安心。
<br/>　　“这件事我应了，不知凉州王看中了我蛤蟆城那支队伍？”
<br/>　　“这自然听程当家安排，不过，若是龙字军的话，自然是最好没有。”
<br/>　　赵子龙率领的龙字军是蛤蟆城唯一一支骑兵，虽然数量不多，但战斗力极强。这陆洛便在龙字军手中吃过亏。
<br/>　　程大雷略微想了想，便开口道：“好，等凉州王需要时，我三千龙字军听候差遣。”
<br/>　　一听此话，陆洛大喜过望，他将这件事谈定，于自己也是大功一件。失去的身份和地位，可以稍微捞回来些。
<br/>　　程大雷亲自修书一封，让陆洛带给宋伯康，至于详细的交接时间，双方还要确定。
<br/>　　带着程大雷的信，陆洛快马加鞭走了。程大雷独自坐在花厅中，忽然轻轻叹了口气：可惜了百里胜呐。
<br/>　　陆洛走后，程大雷又差人将赵子龙唤到城主府。
<br/>　　“大当家，你找我。”
<br/>　　“嗯，有件事要与你商量。”程大雷将事情详细与赵子龙说了。
<br/>　　“这……让龙字军帮着别人去打仗？”
<br/>　　“嗳，你先别激动。”程大雷道：“如果我们人质不派，兵也不派，宋伯康怎么会信任我们。让你去了，无非是表示我们的诚意，不会在背后给他们使绊子。你也未必需要上阵，袖手旁观你总是会的吧。”
<br/>　　“喔，我明白了。”赵子龙恍然大悟：“可是，我担心这样做，会打击龙字军的士气。大当家也知道，龙字军没吃过败仗。”
<br/>　　龙字军是程大雷手中的一支强兵，他们的确不是很习惯失败。如此消磨士兵战无不胜的锐气，也未必是一件好事。
<br/>　　程大雷沉思片刻道：“那就派别的部队去，但仍由你率领，在外面仍打龙字军的旗号。”
<br/>　　凉州方面与程大雷敲定了这件事，经过互相交易后，已经确定合作关系。
<br/>　　这对程大雷来说是好事，但对宋伯康来说也未必是坏事。他磨刀霍霍，终于可以腾出手来，消灭百里胜在凉州的势力。
<br/>　　出兵打仗总是要有个由头，最近宋伯康就在炮制这个由头。他无端端抓了几个戎族，又从这几个戎族口中，得到他们和百里胜暗中有联络。
<br/>　　于是便修书一封递到朝廷，说百里胜暗中勾结戎族，图谋我帝国江山。不等朝廷回信，便整顿军伍，派出人马准备攻打百里胜。
<br/>　　至于那几个戎族是怎么来的，而不识字的戎族又怎地拿出一份详细的口供，也就没有人关心了。
<br/>　　当然，在这之前，宋伯康还要和程大雷完成朔方城的交接适宜，之后他才能全力以赴对付百里胜。
<br/>　　双方交接完毕，程大雷亲自来到朔方城，看自己新得手的这座城池究竟是怎样情况。
<br/>(本章完)
