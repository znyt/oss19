　　第407章
<br/>　　“呃……这个……还真是……”
<br/>　　秦蛮搓着双手，面泛红光，整个人变得手足无措起来。
<br/>　　程大雷瞥了他一眼，刚才是谁说『鬼神之言当不得真』，那为何你现在又这个样子。
<br/>　　“大当家，你说我是不是该把仙人请到家中供奉，早晚烧香。”
<br/>　　“你现在该关心的，不应该是神仙为什么提起你么？”程大雷没好气道。
<br/>　　“对，对，仙人都说了我什么啊？”秦蛮认真问。
<br/>　　“哈！”程大雷拖了个长音：“仙人说啦，秦蛮这个人是很老实的，本事也不错。只不过以后想做大事，靠现在的本事还不够。”
<br/>　　“仙人真这么说了？”
<br/>　　“怎么，你信不过仙人还信不过我么？”程大雷。
<br/>　　秦蛮无语，心道我那里是信不过仙人，我是信不过你啊。
<br/>　　“仙人还说什么？”
<br/>　　“你也不用担心，仙人决定传给你本事。”
<br/>　　“真的！”秦蛮整个人激动起来：“大当家，我该怎么做？”
<br/>　　“什么都不需要做，盘腿静坐，放空心思，仙人传授给你的东西，将由我的手交给你。”
<br/>　　秦蛮在程大雷面前坐立难安，来回踱步，好不容易才安静下来，盘腿坐在地上。脸上浮现虔诚的神色，似乎正等待圣洗的信徒。
<br/>　　程大雷心中颇为无语，怪不得林少羽可以在短时间拉起一支队伍。有些人，还真是好糊弄呐。
<br/>　　他伸出一指，点在秦蛮额头，心里默默说了一句：选择提升秦蛮等级。
<br/>　　紧跟着，没有任何事情发生，但已有源源不断的信息流涌入秦蛮脑海。程大雷经过许多次这样的事情，丝毫不觉得奇怪。系统赋能，可以提升速度、力量、反应，但更重要的是根据每个人量身定制的战斗经验。
<br/>　　也就是说，哪怕秦蛮是倒立，劈叉，都不碍秦蛮的等级提升。
<br/>　　不过人么，总是需要些仪式感，把事情搞得像那么回事。总不能在秦蛮蹲茅厕的时候，用手一拍他肩膀：嘿，你升级了。
<br/>　　程大雷同样盘腿坐在秦蛮面前，等待他清醒过来。自己可以装装样子，问他悟到了没有。
<br/>　　秦蛮会回答：悟到了。
<br/>　　显得有那么一点禅意，自己也像个高人。
<br/>　　想到这里，程大雷隐隐有些期待了。就只见到秦蛮眉头皱紧，眼角轻轻抽动，程大雷明白，他正在消化脑海中的信息。
<br/>　　然后，随着咚地一声，秦蛮一头栽倒在地上，就此昏迷了过去。
<br/>　　咦！
<br/>　　程大雷吓了一跳，忙扶住秦蛮，紧张道：“醒醒，你没事吧。”
<br/>　　可秦蛮依旧昏迷不醒，没有给程大雷任何反应。程大雷忙将刘悲找来，将秦蛮抬到里面房间。
<br/>　　昏迷不醒的秦蛮令程大雷一颗心揪了起来，刘悲与秦蛮把过脉，大概是心火攻心，一时间没受住昏了过去。
<br/>　　程大雷心里琢磨着，难道是信息流太大，超载宕机了。
<br/>　　而程大雷清晰得看到，秦蛮的确已经升级成功。
<br/>　　秦蛮，技能梅花枪，隐藏属性……守护者。
<br/>　　秦蛮昏倒后，程大雷心情一直很着急，生怕自己好心做错事，反而害了秦蛮。
<br/>　　秦蛮足足昏迷了三天，直到三天后才醒过来。程大雷得到消息后，第一时间赶了过来。刚见到秦蛮，程大雷就吓了一跳。如今的秦蛮，和三天前的他似乎有些不太一样了。
<br/>　　他立在院子中，腹背挺直，岩石般一张脸，可眼中却又阳光般的东西。给人一种踏实可信任的感觉，程大雷心里琢磨着，这应该是他的隐藏属性『守卫者』发挥的作用。
<br/>　　“感觉怎么样？”程大雷问。
<br/>　　秦蛮摇摇脑袋，脸上有困惑表情：“我现在脑子很乱，似乎身体里多了很多不属于自己的东西。”
<br/>　　程大雷明白秦蛮现在的状况，因为他也经历过。迈入绝世等级后，总有一个类似证道的过程，将体内的意识、力量嚼烂，消化干净，化为自己的东西。不然，就只是空有绝世的力量和速度，但不知该如何驾驭。
<br/>　　“试一试？”程大雷忽然有些跃跃欲试。
<br/>　　体内多了新的力量，秦蛮也急于尝试。他顿了顿，目光在周围转了一圈：“在这里？”
<br/>　　以二人的本事，如果是在这里切磋，怕又有开始一场拆房大作战。
<br/>　　“走，去演武场。”
<br/>　　蛤蟆城的演武场在兵营中，一听说程大雷要和秦蛮切磋，山寨里的人都凑了过来。刘关张赵高一个没落，此外还有樊梨花、李婉儿，甚至包括从诺顿王朝过来的一伙人。
<br/>　　程大雷自从升入绝世后，俨然已经成为山寨第一高手，就连赵子龙和他也有很大差距。然而秦蛮也并非弱手，今天这两个人要打起来，谁也不想错过这一幕。
<br/>　　演武场上有擂台，本来山寨里互相切磋的事情就不算少见。程大雷只打算和秦蛮随意较量较量，帮助秦蛮尽快掌握绝世等级的力量。然而，没想到手下人这么激动，生生拖了半天，等准备好时，擂台下已经挤满了人。
<br/>　　二人用的都是木制兵器，程大雷用木剑，秦蛮用一柄木枪。其实以二人的本事，以达到飞花摘叶皆可杀人的地步，就算用木制兵器，也同样可以伤人。不过今天不是生死搏斗，互相也不会朝对方要害出招，当然，如果真的打急眼了，受些轻伤也是在所难免。
<br/>　　众人拭目以待，只见程大雷和秦蛮在擂台上相对而立，只是一个瞬间便撞在一起。长枪木剑，却不发出任何碰撞，在对方攻出一招后，另外一方便立刻回避。
<br/>　　台下诸人都睁大了眼睛，脸上露出难以置信的表情。本来还以为二人该打得很火热，却没想到，二人越打越慢，慢得不可思议。
<br/>　　这很显然没多大观赏性啊。
<br/>　　内行看门道，外行看热闹，真正的高手对决，争的都是一招一式，根本也谈不上什么精彩。可对于赵子龙、关鱼这些行内人而言，却是看得目不转睛。
<br/>　　二人虽然打得很缓，但内里却杀机丛生。赵子龙等人把自己替换到二人的位置上，该如何防范，看得脊背一阵阵发凉。
<br/>　　
<br/>(本章完)
