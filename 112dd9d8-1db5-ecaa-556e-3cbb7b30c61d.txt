　　“大规模？”林峰心中一跳。
<br/>　　“最起码十万！”朱民说到这声音都忍不住降低了。
<br/>　　“除了丧尸之外，还有许多的怪物，变异狗变异猫变异蚊子变异蛇等等，这群怪物也都在聚集。所以对于曙光基地来说，情况不容乐观。”
<br/>　　朱民面色沉重道。
<br/>　　林峰一听也是心中一沉，这的确不是好消息。

<br/>　　“当然了，我们因为提前得知了情况，所以有了时间可以去防备，比如说布置地雷区，这几天战斗部队也会去进行歼灭任务。能歼灭的就歼灭，不能歼灭的就引诱到其余地方去，所以我们现在的情况，比当初的沙长基地要好太多了，他们是被打了个措手不及。所以这一切都还要多谢林峰你了，确切说，你救了我们基地一次。”
<br/>　　朱民认真的说道。
<br/>　　这么大的帽子林峰可不敢接下来，，“别别别，我也只是做一些力所能及的事情罢了，毕竟我的朋友老师都在基地内。”
<br/>　　朱民拍了拍林峰肩膀，“对了，这是军部发给你的荣誉勋章金刀勋章，这可是很珍贵的，代表的一种荣誉，当然了，他也有一些特权，比如说，你购买一些东西是可以打折的哦，又或者你开店的话，不需要每个月交管理费给基地了。”
<br/>　　朱民忽然拿出了一个小盒子给了林峰里面装着一枚金灿灿的勋章。
<br/>　　“这……”
<br/>　　林峰万万没想到还得到了军部的表扬，让他这个社会主义接班人特别的自豪。
<br/>　　“这是你应得的。有了这勋章，实际上，就等于你在军部有了军衔，相当于少校了，恭喜你了林峰，我混了这么多年，也才是个上校罢了。”朱民笑道。
<br/>　　简单聊了几句，朱民就急匆匆的离去了，对方非常多的事情要处理，能够抽空和林峰说这么多已经实属不易了。
<br/>　　看着那勋章，林峰没想到自己居然成少校了。
<br/>　　当然这个少校更多的还是一种荣誉，没有什么调兵遣将的权利。
<br/>　　“购买东西可以打折？岂不是说明我买房子可以便宜很多？”
<br/>　　林峰眼睛一亮，这东西来的正是时候。
<br/>　　“既然如此，继续猎杀吧！努力提升实力！”
<br/>　　……
<br/>　　而与此同时。
<br/>　　一辆神秘的大巴车进入了特等区。
<br/>　　这大巴车内都是些妙龄女人。
<br/>　　抵达了王家别墅后。
<br/>　　一个个女人都下了车兴奋紧张的看着这别墅区。
<br/>　　“都进去。”
<br/>　　开车的两人示意他们都进入别墅，一群人来到别墅内都被富丽堂皇的景象所吸引。
<br/>　　“哇，这么漂亮。”
<br/>　　“没想到我也可以住别墅了。”
<br/>　　“太好看了。”
<br/>　　每个人都露出了兴奋的表情。
<br/>　　然而那两个开车的对视一眼，露出一抹讥讽的嘲笑，旋即就离开了别墅把门锁上。
<br/>　　“欢迎你们的到来，我的小宝贝们。我已经有些迫不及待要品尝你们美味的鲜血了。”
<br/>　　一道冷笑传来。
<br/>　　旋即几个黑袍人出现在别墅之中。
<br/>　　下一刻。
<br/>　　别墅内传来了女人的尖叫。
<br/>　　“感觉如何。”
<br/>　　王梅看着一地的尸体无动于衷，这情况他见怪不怪了。
<br/>　　一个黑袍老者深吸口气。
<br/>　　“还不错，放心，你男人的死，还有老六的死，我们肯定会调查的清清楚楚，这个战门不好惹，但是我们尸傀门也不是泥巴捏的。敢杀老六，他就必须付出血的代价。”
<br/>　　“那我就等你们的好消息了。”王梅说道。
<br/>　　……
<br/>　　离开基地前，林峰逛了一下悬赏任务栏，这里的任务似乎无穷无尽的，每天都有任务，而且还一天比一天多。
<br/>　　“猎杀XXX地区的七阶怪物，奖励两百颗三阶晶石！五箱香飘飘奶茶！”
<br/>　　“香飘飘奶茶？好像苏老师最喜欢喝这个了，之前看她办公室里总是要泡一杯，这玩意现在和泡面一样，也算是贵族才吃的起喝得起的了。”
<br/>　　林峰想都没想就直接接取了这个任务。七阶，也算是他能够应付的范围之内了。
<br/>　　“猎杀XXX地区的六阶丧尸，奖励四十颗三阶晶石！双汇火腿肠十二根装三包！”
<br/>　　“接了！”
<br/>　　林峰果断也接了。
<br/>　　“猎杀XXX地区的五阶丧尸，奖励三十颗三阶晶石！小浣熊干脆面一箱！”
<br/>　　“接了！”
<br/>　　但凡是有猎杀任务，只要是七阶以下的，林峰全部都接了。
<br/>　　“快看，那个疯子又来了。”
<br/>　　“妈的，这一次居然又接了这么多任务，斩杀七阶怪物那个任务这小子也接了？这怕不是去送死吧。”

<br/>　　“上次我就看到他接了一大堆任务，没想到人没死不说，又来接任务了。”

<br/>　　“接着玩的是要被军部拉入黑名单的，你要知道，这整个基地都是军部的，你要被军部拉入黑名单，那你干脆也别在基地内呆着。所以根本没有人敢这样做。”

<br/>　　此时此刻。
