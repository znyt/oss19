　　话说林峰帮陆勇他们干掉了一些战门成员后，就看到了一伙人。
<br/>　　正是被包围的陈勾和罗风他们几个。
<br/>　　罗风身边跟着的几个星陨阁成员一个都没了，肯定都已经牺牲了。
<br/>　　哪怕是罗风也是受伤了。
<br/>　　陈勾也好不到哪里去。几个人都岌岌可危。
<br/>　　“妈的！”
<br/>　　林峰二话不说顿时掏出火箭筒就是对那十几个战门成员轰去。

<br/>　　砰！
<br/>　　火箭弹爆炸，这群战门成员压根没有注意到林峰的到来，顿时直接被炸死了三个。
<br/>　　陈勾侧头瞧去就看到林峰。
<br/>　　“林峰，你总算来了！你再不来，胖爷我两百斤肉就交代在这里了！”
<br/>　　罗风看到林峰也是松了口气，“先遣部队如何。”这家伙就是个大好人，这种时候了，不想着自己，而是想着曙光基地这个大局的安危，若是条件允许的话，林峰都要送他一个大红花以资鼓励了。
<br/>　　“没事，多亏你们把那些异能者都击破了，先遣部队之前被袭击了，死了大概三四百人上下，不过那群战门成员我也都解决了。”
<br/>　　林峰拔出斩灵刀来到了陈勾这边。看着之前围殴他们的那伙人道。这伙人也都从爆炸之中回过神来，在那里吐口水，爆炸的泥土烟尘全部都钻到他们嘴巴里去了，搞得他们灰头土脸的。
<br/>　　“这些人也是战门的？”
<br/>　　罗风点了点头，“那个年龄最大的是战门外门长老，八阶的实力，其余的战门成员，足足三十人，另外还有几个血尸堂的成员和尸傀门的成员，我们刚刚灭掉那十二个异能者，他们就出现了，我的几个朋友都牺牲了。”
<br/>　　林峰拍了拍他肩膀，“幸苦了。”
<br/>　　“小子，就是你！”
<br/>　　也就在这个时候，一个血尸堂成员忽然瞪大了眼睛，他认出了林峰，然后就是咬牙启齿的嘶吼道：
<br/>　　“就是他摧毁的我们血尸堂地盘！”
<br/>　　“什么？”
<br/>　　“不是说是战门成员吗？”
<br/>　　血尸堂成员恍然大悟，纷纷暴怒道：“我们被耍了！这家伙根本不是战门成员，他故意让我们以为是战门成员做的！”
<br/>　　“妈的，岂有此理！我们居然被人戏耍的团团转，岂不是说刚才我们几个血尸堂成员白死了！”
<br/>　　几个血尸堂都面色难看，居然像个傻子一样被人家玩，刚才他们和战门血拼，死了好几个成员，虽然后来因为罗风的出现，他们不内斗了，转而一起对付罗风那伙人，但毕竟人是死了。
<br/>　　林峰戏谑的看着他们每个人，“我说是就是啊，那我说我还是你们的爸爸嘞，你要不现在就叫个爸爸给我听听！”
<br/>　　“找死！”
<br/>　　血尸堂成员勃然大怒，顿时一个个都直接出手。他们早就一肚子怒气了，自然是动手不动口。
<br/>　　“去，千年僵尸和他们玩玩！”
<br/>　　林峰立即派出自己的千年僵尸。
<br/>　　“什么？千年僵尸？”
<br/>　　血尸堂成员面色阴沉，他们是玩尸体的行家，自然知道千年僵尸的厉害。
<br/>　　“这小子居然有这么好的宝贝，杀了他，就是我们的了”
<br/>　　几个血尸堂成员眼中露出了一抹贪婪之色，千年僵尸，千年的道行，时间这种东西最宝贵，算是可遇而不可求的存在。
<br/>　　林峰扫了眼其余人，暗道麻烦，对方人数太多，正面硬钢，他们这边还真不是对手，而且这战门还有个外门长老在这里。
<br/>　　不好对付。
<br/>　　“对了，之前的三十个C级空投还没融合为B级空投！”
<br/>　　林峰眼睛一亮。
<br/>　　“罗风狗子，你们先掩护我，我修炼个神功！”
<br/>　　“小妲己你也出来，帮我的千年僵尸对付那群坏人。”
<br/>　　“知道了主人。”
<br/>　　妲己立即跑了出来去帮助千年僵尸了。
<br/>　　千年僵尸七阶，妲己六阶，血尸堂几个成员都是六阶，他们的尸体也都是六阶。
<br/>　　双方暂时还可以打个旗鼓相当。
<br/>　　“我们也上！这群人居然敢用我们的旗子在外面招摇撞骗给我们战门招引麻烦！罪不可赦！！”
<br/>　　战门外门长老大手一挥顿时三十多个成员全部来了。
<br/>　　林峰见此直接来了一招影分身之术。
<br/>　　瞬间几十个分身全部杀了过去，暂且掩护林峰一二。
<br/>　　趁这个功夫。
<br/>　　林峰立即合成B级空投。
<br/>　　B级空投，仅仅是外表看起来都无比的高大上。
<br/>　　合金做的箱子，本来林峰意外很大的体积，毕竟D级空投相当于课桌，C级空投就是旅行箱大小了，按照这逻辑，那B级空投肯定也是更大的。
<br/>　　但是，B级空投却只有巴掌大小。

<br/>　　系统：B级以上的空投采用的空间储存，外观大小只是个形式，别看他这么小，说不定可以开出你故乡的十万吨级核动力航母出来。

<br/>　　林峰震惊了。
<br/>　　咔擦。
<br/>　　林峰也不墨迹直接开启了，顿时这个B级空投就好像是魔方一样开始自我旋转了。

<br/>　　恭喜宿主领悟《降龙十八掌》！
<br/>　　叮！
<br/>　　恭喜宿主获得孙悟空！
<br/>　　叮！
<br/>　　恭喜宿主获得100发火箭弹！
<br/>　　叮！
<br/>　　恭喜宿主获得汽油弹100发！
<br/>　　叮！
<br/>　　恭喜宿主获得99式主战坦克七辆！
<br/>　　叮！
<br/>　　恭喜宿主获得98K一把！子弹100发！
<br/>　　叮！
<br/>　　恭喜宿主获得《狙击精英之无敌爆头狙杀传承》

<br/>　　林峰直接是把孙悟空给放出来，这是和妲己一样来自于王者荣耀的英雄，人称猴三棒，外表也是经典皮肤齐天大圣。

<br/>　　“俺老孙去也！”
<br/>　　孙悟空二话不说翻着跟头就是跳了过去。

<br/>　　那血尸堂成员压根都没有察觉到孙悟空的到来，根本就看不见忽然砰的一下。
