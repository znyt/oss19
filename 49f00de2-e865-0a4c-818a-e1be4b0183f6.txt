        殊不知，朱小溪正是因为暗恋严文斌，鼓起勇气向严文斌表白被拒，结果严文斌喜欢的人却是她连星儿。
<br/>
<br/>        因此，朱小溪才会对连星儿的友情阳奉阴违、表里不一。
<br/>
<br/>        连星儿并不知道朱小溪对自己假心假意的原因，同时也不知道严文斌喜欢自己。
<br/>
<br/>        所以，当连星儿在这北城里偶遇严文斌时，也纯粹的只是念在昔日同窗同学之情上，才会跟严文斌侃侃而谈。
<br/>
<br/>        “你怎么会在北城？你不是去国外念书了吗？”严文斌好奇地问道。
<br/>
<br/>        当初，她连星儿被保送国外名牌大学的事情，可谓是传的人尽皆知。
<br/>
<br/>        连星儿微笑着回答道：“我来看《世界黑客战队大赛》。而且，我也没出国去念书了。”
<br/>
<br/>        “那你现在在哪里念大学？”严文斌的眼中闪过一丝欣喜。
<br/>
<br/>        连星儿接着答：“在临海城的临海大学啊！”
<br/>
<br/>        “以你的成绩，应该念我们北城大学的。”严文斌不禁感慨道，同时又有些悻悻地叹气。
<br/>
<br/>        连星儿却一脸无所谓地回答道：“反正我这种未来的道路，都已经被父母安排好的女孩子，读哪所学校都一样。”
<br/>
<br/>        她未来的路是什么路？
<br/>
<br/>        家企是轮不到她去继承，但是，她的爹地妈咪却早已给她存了一大笔钱，足够她下半辈子不仅衣食无忧，还可以挥霍无度。
<br/>
<br/>        所以，她压根就可以不用去考虑工作，也不用去为了得到什么而奋斗，就连自己的婚姻，也被长辈们安排得妥妥帖帖。
<br/>
<br/>        像她这种女孩子，能自己做主的时候真的很少。
<br/>
<br/>        这算不算是逆来顺受？
<br/>
<br/>        从小到大，她想要做的事情，都被爹地妈咪扼杀在摇篮里了。
<br/>
<br/>        久而久之，连星儿养成了一种不好的习惯。
<br/>
<br/>        那就是，爹地妈咪说是什么便是什么。
<br/>
<br/>        哪怕爹地要她放弃去国外念书的机会，选一个离家很近的临海大学，她都毫无怨言地答应了。
<br/>
<br/>        还有傅家和明家为了缓解世仇关系，牺牲她去联姻的事，她同样丝毫不反对。
<br/>
<br/>        虽然，每次爹地妈咪明着是有征求过她的意见，但是她心里明白，爹地妈咪还是希望她能实现他们的意愿。
<br/>
<br/>        连星儿并不觉得自己很委屈，也不觉得老天爷对自己不公平，毕竟有得就有失。
<br/>
<br/>        上天给了她优越的物质生活，必然就会从她这儿夺走一些东西，来平衡这个世界。
<br/>
<br/>        “难得我们有缘偶遇，不如一起去吃个饭吧！我在这北城念书，也算是东道主。我请你。”严文斌微笑着转开了话题。
<br/>
<br/>        而碰巧的是，连星儿这个时候，肚子也饿了。
<br/>
<br/>        身无分文不说，连手机也没有。
<br/>
<br/>        不如，趁着这个机会，跟他严文斌借点钱回酒店。
<br/>
<br/>        连星儿心里这么划算着，于是二话不说就答应了严文斌的邀请。
<br/>
<br/>        明司寒追出来后，只见连星儿跟着一个陌生的男生上了一辆出租车。
<br/>
<br/>        担心这丫头会出事，他立马又拦了下一辆出租车，坐到副驾驶座里后，命令出租车司机跟在连星儿所坐的那辆出租车后面。
<br/>
<br/>        严文斌带着连星儿去了北城一家最有名的烤乳猪店，坐的是一楼大厅的隔断雅座。
<br/>
<br/>        每一桌与每一桌之间有桃木色吉祥图案的屏风隔开，虽然耳畔时不时地传来邻桌的客人们的谈话声，但彼此的空间隐私还是得到了保障。
<br/>
<br/>        但是，连星儿却不大喜欢这种集中式用餐的环境。
<br/>
<br/>        因为，在她看来，这跟去学校食堂里吃饭没什么区别。
<br/>
<br/>        严文斌没有连星儿那奢华的家世背景，只是生在一个普通的小康家庭里，所以，他对吃饭这方面没什么讲究，同时也没什么挑剔。
<br/>
<br/>        他觉得，能带连星儿来北城最好的餐馆，就算是自己对她特好的了。
<br/>
<br/>        连星儿和严文斌落座后，明司寒让服务员给他安排了邻座的位置。
<br/>
<br/>        三个人之间，仅仅只隔着一道屏风。
<br/>
<br/>        所以，连星儿和严文斌说什么话，明司寒这边听得一清二楚。
<br/>
<br/>        服务员拿着菜单过来后，严文斌问都不问连星儿的意见，便直接点了两菜一汤，还嘱咐服务员要快点上菜。
<br/>
<br/>        连星儿全程默不吭声，但也没有反感严文斌。
<br/>
<br/>        至少，他点的菜，她都能吃，虽然素了点。
<br/>
<br/>        不过，现在对于她来说，能填饱肚子就行。
<br/>
<br/>        明司寒听完严文斌报的菜式后，向服务员招了招手，点了餐馆里的几个招牌菜后，云淡风轻地说道：“送去旁边那桌，就说是你们免费送的体验菜。”
<br/>
<br/>        顿时，服务员一脸懵然地看着明司寒，愣了一秒后，又立即尽职尽责地点头哈腰道：“好的，请稍等。”
