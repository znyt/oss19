        大哥人这么好，想必，诺司峻嘴里说的童筱颖的那个闪婚老公傅昼景，人也一定不错。
<br/>
<br/>        唐倩兮顿时替童筱颖感到很欣慰，欣慰她童筱颖找了一个好老公。
<br/>
<br/>        从云城到西镇还要历经两个小时的路程，途中还要经过云城县区。
<br/>
<br/>        几经波折，童筱颖带着傅夜沉和唐倩兮回到了梁家。
<br/>
<br/>        但是，梁家的店面本该这个时候就已经开门营业的，却是大门紧闭。
<br/>
<br/>        童筱颖连忙掏出手机给诺爸打电话，但是听筒里传来的是空号的提示音。
<br/>
<br/>        无奈之下，童筱颖不得不询问邻居，她家里到底发生了什么事情。
<br/>
<br/>        这不问还好，一问，顿时令童筱颖和唐倩兮傻眼了。
<br/>
<br/>        “你哥哥不知道犯了什么事，被抓去坐牢了。你爸妈为了赎你哥哥出来，把家和门面全都卖了。两老搬回村上去住了！”邻居跟童筱颖叙说道。
<br/>
<br/>        童筱颖立即带着傅夜沉和唐倩兮，去了村上原来那栋土屋。
<br/>
<br/>        果然，当她回来的时候，诺爸正在家门口给药草分类，而诺母则在一旁晒药草，看到童筱颖的时候，两老都忍不住哭了起来。
<br/>
<br/>        这才一个月不到的时间里，童筱颖突然发现诺爸诺母好像老了好几岁，头上的银丝比以前多了许多。
<br/>
<br/>        “筱颖……你、你怎么回来了？”诺爸用颤抖暗哑的声音问道。
<br/>
<br/>        诺母只是哭着，不说话。
<br/>
<br/>        “爸爸、妈妈，我带哥哥的女朋友过来，找哥哥。”童筱颖还是忍不住唤他们为爸爸妈妈。
<br/>
<br/>        毕竟，这声“爸爸妈妈”，童筱颖叫了十八年，不可能轻易改口。
<br/>
<br/>        诺母连忙进屋泡茶，招待他们。
<br/>
<br/>        唐倩兮也非常懂礼貌地唤了一声：“叔叔阿姨好！”
<br/>
<br/>        傅夜沉一直陪在童筱颖身边，诺爸诺母还误以为他是诺司峻说的童筱颖的那个闪婚老公。
<br/>
<br/>        但见这男人也只唤他们为“叔叔阿姨”，诺爸诺母还误以为他是替筱颖介怀他们换女儿且毫不顾念这么多年的亲情的事情。
<br/>
<br/>        几个人一起坐下之后，诺母就一边抹眼泪，一边向童筱颖倾诉：“你哥哥那个败家的东西，怎么不知好歹，参与到种植罂粟花的份里去了！”
<br/>
<br/>        “哥哥种植罂粟花？！怎么可能？！”童筱颖惊愕地从椅子上站起来。
<br/>
<br/>        就连唐倩兮也不敢置信。
<br/>
<br/>        傅夜沉却皱起了眉头。
<br/>
<br/>        “一个月前，村里来了一批警察、还有什么特种兵，烧毁了南山后边的一大片罂粟花培养基地，而且还抓了村上好几个人。没想到，你哥哥居然也拿了钱去投资了那片罂粟。每年，从那投资人手里拿到了一些分红！你说你哥哥怎么就这么糊涂呢？家里是做药草生意的，他怎么可能不认识那是罂粟！”诺母一把鼻涕一把泪。
<br/>
<br/>        诺爸连连摇头，痛心疾首。
<br/>
<br/>        “那现在该怎么办？”唐倩兮焦急地问道。
<br/>
<br/>        诺爸唉声叹气地回答道：“我们听说，只要凑两百万出来，就可以将你哥哥从监狱里赎出来。但是……我们两口子，砸锅卖铁，就连给梦萌出国去念书的钱都用上了，还只凑了个五十万。哎——”
<br/>
<br/>        “我这里还有二十万了，是我和司峻在一起的这些年攒下的！”唐倩兮二话不说，便将自己包里的银行卡给掏了出来。
<br/>
<br/>        这是她所有的积蓄了。
<br/>
<br/>        看到唐倩兮二话不说就拿钱出来给诺爸诺母，童筱颖不得不低下头去。
<br/>
<br/>        她……
<br/>
<br/>        如果要拿钱出来的话，得去问昼景借。
<br/>
<br/>        如果，她告诉昼景，要拿钱去救她那个毫无血缘关系又犯了事的哥哥，昼景一定不会答应的。
<br/>
<br/>        傅夜沉看着这一家子被逼入了绝境，终于按捺不住，开口说道：“你们的钱，都各自留着。筱颖的哥哥的事情，我会替你们想办法，让他从监狱里出来。只不过……”
<br/>
<br/>        “只不过什么？你说什么，我都答应你！”诺母比任何人都积极。
<br/>
<br/>        就连诺爸和唐倩兮都用非常期待的目光看向傅夜沉。
<br/>
<br/>        童筱颖却在此刻忽然间想起了什么，心里猜想着，一定是她那次告知他罂粟花田的事情后，他回去后才做好了充足的准备派人下来抓人的吧！
<br/>
<br/>        她的好心，虽然害了她哥哥去坐牢，但是，童筱颖并不后悔，至少她减少了罂粟花变成那种毒药出去祸害更多无知的人。
<br/>
<br/>        “只不过，二老以后恐怕得带着你们的儿子，远走他乡了！”傅夜沉接着说道。
<br/>
<br/>        诺母立马就同意了。
<br/>
<br/>        童筱颖微微偏头看向唐倩兮，轻轻地问道：“嫂嫂，你呢？你会愿意跟我哥哥远走他乡吗？”
<br/>
<br/>        “我愿意！无论司峻去哪里，天涯海角，我都跟着他去！”唐倩兮深情款款地看着诺司峻，信誓坦坦地说。
<br/>
<br/>        傅夜沉点了点头，让童筱颖和唐倩兮就留在这老家里等消息，他先离开去办这件事情。
<br/>
<br/>        总而言之，童筱颖不知道傅夜沉是如何把这件事情给办成的。
<br/>
<br/>        她和唐倩兮在老家等了两天，哥哥诺司峻终于平安归来了。
<br/>
<br/>        诺爸诺母激动得抱着哥哥诺司峻哭了起来，大骂他的不孝与无知。
<br/>
<br/>        诺司峻也保证，以后再也不干非法的事了。
<br/>
<br/>        当诺司峻再次看到童筱颖的时候，还是忍不住将她紧紧地搂入了怀中：“我的好妹妹，以后要好好照顾自己。谢谢你这次救了哥哥的命！”
<br/>
<br/>        “哥哥，我……”
<br/>
<br/>        “什么都不要说了！你在临海城，要好好过日子。”诺司峻放开童筱颖后，唉声叹气地拍了拍童筱颖的肩头，然后伸手牵住了唐倩兮的手，这一次，他再也不会扔下他心爱的女人不管不顾了。
<br/>
<br/>        诺母这回自知之明，如果不是筱颖，她压根就看不到儿子从监狱里出来。
<br/>
<br/>        经过这件事情之后，她从亲生女儿那儿拿了钱之后，梦萌就跟他们断绝来往了，又重新回到了她妹妹那个家中。
<br/>
<br/>        俗话说，患难见真情，对他们不离不弃地还是她养了十八年的养女筱颖。
<br/>
<br/>        诺母进了一趟屋内，拿出一个小木箱，走到了童筱颖面前，接着又将自己手腕上戴了四十多年的金镯子给取了下来，拉起童筱颖的手，将金镯子戴到了童筱颖的手腕上。
<br/>
<br/>        就算是儿子出了事情，她卖房子卖店，也没想过要把这金镯子给卖了。只因为，这金镯子传了好几代了，而且只传女不传男，诺母感觉这金镯子有灵性，会护主。
<br/>
<br/>        守护了她这么多年，也该守护守护她这个苦命的女儿了。
<br/>
<br/>        “这是你外婆传给妈妈的，这金镯子只传自家女儿，而且，你外婆在世的时候，就希望我将来能把这金镯子传给你……”诺母抹着眼泪，抽泣道。
<br/>
<br/>        童筱颖只是静静地看着诺母，心里五味陈杂。
<br/>
<br/>        诺母说着说着，接着又将小木箱放到了童筱颖的手上，声音哽咽了：“还有这个小木箱里，是你外婆毕生经验和技术，你外婆说也要留给你，如今，妈妈如了你外婆最后的心愿。妈妈从今以后，就只有你这么一个亲女儿了……”
<br/>
<br/>        “妈——”童筱颖鼻子一酸，声音是从喉咙处挤出，忍不住抱了抱诺母。
<br/>
<br/>        以前她还埋怨妈妈的对自己的各种不是，此时此刻，她任何怨言都想不起来了，只知道，自己心里满是不舍。
<br/>
<br/>        这是童筱颖最后一次真正地与家人告别，从此以后只怕再也无法见到了吧！
<br/>
<br/>        为了不牵扯到童筱颖，傅夜沉命令诺司峻一家人从此跟童筱颖断绝来往。
<br/>
<br/>        毕竟，罂粟花田的事件，首脑人物在慌乱中逃走了，至今还未抓到。
<br/>
<br/>        诺司峻不知道是傅夜沉救了自己，只知道，那个救了自己的人，是看在筱颖的份上才出面让上头领导让他诈死秘密安排出狱的。
