　　第300章
<br/>　　徐神机第二天醒来，发现已经人去屋空，屋里摆着一个喝空的酒坛。至于昨夜做过什么，说过什么，徐神机都忘得一干二净，他只意识到一件事：牧长庚逃了。
<br/>　　他忙将这个消息告诉程大雷，程大雷倒是没多大反应，只是随口道：“走了就走了吧，对了，你没有和他说什么吧？”
<br/>　　“怎么会？”徐神机拍着胸脯道：“我一直防着他呢，什么都没告诉他。”
<br/>　　“那就好。”对于徐神机的口风，程大雷还是相信的，他道：“不是还有几个人么，派人看紧点，瞧瞧他们有什么动作。”
<br/>　　程大雷排查内奸，还真发现了几个人，都被丢在铁匠铺锻炼身体。过去这么久，有人耐不过折磨，就悄悄撤了，如今铁匠铺还有几个人忍熬着。
<br/>　　对于他们这些人，自然是不得不防。
<br/>　　至于牧长庚逃走的事情，程大雷确实也没有太放在心上，心里记下后，就开始忙碌别的事。
<br/>　　话说，牧长庚泅水离开蛤蟆寨，一路还要小心浅滩的毒蛇，最后废了九牛二虎之力，终于成功登岸。
<br/>　　在当天黄昏，他终于回到琴川关，此刻，他整个人已经累得虚脱，浑身上下，全靠一口气撑着。
<br/>　　“咦，这是牧校尉么……”
<br/>　　守门的小卒接住了他，一时竟有些不太敢认。只过去了大半个月时间，牧长庚活脱脱瘦了一圈，脸上又黑又糙，手上结着厚厚的老茧。
<br/>　　“快去找将军，找将军……”
<br/>　　牧长庚被抬进了琴川关，喝了些稀粥，又喂着服了些参汤，才悠悠醒转，恢复些体力。
<br/>　　“牧兄弟，你终于回来了，我们还以为你已经遭了贼人的毒手。”莫鸣米。
<br/>　　牧长庚未语先泪流，脑海里想起连日来的遭遇，委屈翻滚着涌上来。
<br/>　　“将军，将军，我还以为一辈子都见不着你了。”
<br/>　　“回来就好，回来就好。”莫鸣米轻拍着他的肩头。
<br/>　　“有没有打探到什么消息？”严笛问。
<br/>　　牧长庚点点头：“幸不辱命，将军交待的任务总算是完成了。”
<br/>　　屏退左右，房间内只留下严笛、莫鸣米与病榻上的牧长庚三人。
<br/>　　牧长庚咳嗽一声，道：“消息是从徐神机口中得来的。”
<br/>　　“就是那个号称神机妙算的徐神机么？”严笛眼睛一亮。
<br/>　　牧长庚重重的点点头：“正是他。”
<br/>　　“严军师也听说过他？”莫鸣米问。
<br/>　　“自然听过。”严笛露出神往之色：“将军有所不知，这徐神机是程大雷身边最得力之人，青牛山破戎族大军，长安刺杀杨龙停，都由他出谋划策，不然事情绝不会办得这么轻易。”
<br/>　　“世间竟有如此了得之人。”莫鸣米感慨一声，颇有此种人才竟不能为我所用的遗憾。
<br/>　　“我也知道徐神机了得，一直小心提防，不敢在他面前露出破绽。直到取得他的信任，在昨夜灌醉了他，从他口中套出破寨之法。”
<br/>　　“徐神机是蛤蟆寨的人，怎么会想破寨之法？”严笛追问一句。
<br/>　　“严军师这就有所不知。据徐神机所言，擅守者必先知道对方如何攻，擅攻者必先知道对方如何守。有了破寨之法，才能懂得如何对症下药。”
<br/>　　“原来如此。”严笛摇摇头，颇有种自叹不如，甘拜下风的感觉。
<br/>　　莫鸣米看了严笛一眼，眼神中的意思是：你的境界和徐神机还差得远呢。
<br/>　　“那徐神机如何说？”
<br/>　　“他只对我说了八个字。”牧长庚压低声音，一字一顿道：“要破山寨，当用火攻。”
<br/>　　莫鸣米和严笛对视一眼，都从对方眼神中看到诧异与惊愕。严笛愣了好大一会儿，才道：“牧校尉，难道你不知道，蛤蟆寨是在一座岛上……”
<br/>　　接下来的话，严笛没有说出口：四面都是水呐！
<br/>　　在一个四面都是水的岛上用火攻，这不是扯淡么。
<br/>　　莫鸣米道：“会不会是徐神机早已识破牧兄弟，所以故意用话诓牧兄弟，用的引君入瓮之计。”
<br/>　　“不会。”牧长庚摇摇头：“这个问题我回来的路上也想过，若真的是诱兵之计，应该说个更像真的计策，这火攻之计因为不像真的，偏偏就是真的。”
<br/>　　莫鸣米和严笛同时点点头，皆感牧长庚这话有理。
<br/>　　“可这火攻之计……究竟该如何做呢？”
<br/>　　牧长庚在病榻上坐直身子，道：“我想请问将军和军师，若论对蛤蟆寨的了解，我们比得上徐神机么，若论眼光犀利，计策高明，我们比得上徐神机么？”
<br/>　　“这个……”严笛摆摆头：“对于徐神机，我是甘拜下风的。”
<br/>　　“这便是了，火攻之计是徐神机的肺腑之言，那么必然行得通。只不过……”牧长庚叹口气：“以我们的智力，还无法跟得上徐神机的思路罢了。”
<br/>　　莫鸣米和严笛同时点头，内心对牧长庚的话深以为然。
<br/>　　“现在大家就想想，这火攻之计该如何做？”莫鸣米叹口气：“现在答案已经给我们了，如何破题我们却一无所知，这徐神机……还真是高呐。”
<br/>　　“将军，我一走，徐神机必然有警惕，这件事要快呐，若他有了提防，我们就错过了这万无一失的机会。”
<br/>　　“我明白。”
<br/>　　三人彻夜未眠，拿出蛤蟆岛的地图，研究如何破寨。这件事宜快不宜迟，在第二天早上，莫鸣米点齐大船十艘，小船五十艘，总兵力超过五千，船上多备硫磺火药等引火之物。
<br/>　　莫鸣米的计划是这样的，先用火箭焚毁蛤蟆岛的船只，使对方失去下水的能力。然后抢滩登陆，在岛上展开肉搏战，重点目标是烧毁蛤蟆寨的建筑，船坞，粮仓。只要将蛤蟆寨的粮仓毁了，蛤蟆寨的贼人就不破自破。
<br/>　　当想到这个法子后，莫鸣米长松了口气，徐神机给出答案，自己用一夜时间就有破题思路，看来自己和徐神机的差距没有想象中那么大么。
<br/>　　当然，这只是他自己的想法，事实上，徐神机和他的差距有多大，绝对是他想不到的。
<br/>　　这是必胜一仗，由莫鸣米亲自领军，发兵五千，攻打蛤蟆寨。
