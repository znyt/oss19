        明司寒看向连星儿，眼神里满是宠溺地笑了笑，索性将手中的刀叉相互换了个边，和连星儿同步。
<br/>
<br/>        裴萱妍看了明司寒一眼，顿时无话可说。
<br/>
<br/>        而郎安切着切着，似乎是觉得刀子很多余，于是直接拿叉子叉起牛排，一口全塞入了嘴中。
<br/>
<br/>        连星儿目瞪口呆地看着郎安，脑海里又莫名其妙地回想起了那天自己在外公的实验室门外，被一只巨大的灰狼袭击的画面。
<br/>
<br/>        郎安吃东西，几乎可以用“狼吞虎咽”来形容。
<br/>
<br/>        连星儿见郎安像是被食物噎到了，立马将自己的杯子递给郎安。
<br/>
<br/>        郎安刚伸手，杯子却在半空中被明司寒劫走。
<br/>
<br/>        明司寒随后招手，让服务员重新倒了两杯果汁过来。
<br/>
<br/>        他才不会给这个男人和自己的未婚妻间接接吻的机会。
<br/>
<br/>        郎安吃饱喝足后，起身去了一趟洗手间。
<br/>
<br/>        而明司寒随即也跟着起身，尾随在了郎安的身后。
<br/>
<br/>        两个男生先后上完卫生间，并排站在了洗手台前。
<br/>
<br/>        “如果你敢伤害星儿，我是不会放过你的！”明司寒语气凌冽地警告道。
<br/>
<br/>        郎安甩了甩手上的水珠，转眼便对上了明司寒的目光。
<br/>
<br/>        “……”
<br/>
<br/>        郎安什么话也没说，只是与明司寒对视了几秒后，便与他擦肩而过。
<br/>
<br/>        另一边，裴萱妍趁着明司寒和郎安离开时的这一小段时间里，跟连星儿聊起了郎安的事情。
<br/>
<br/>        “喂，小丫头，别怪姐我没提醒你哦！那个男生，很危险的！”裴萱妍一本正经地说道。
<br/>
<br/>        连星儿却只是充耳不闻。
<br/>
<br/>        她知道，外公派郎安来到她的身边，一定是有着某种目的。
<br/>
<br/>        但是，就目前而言，郎安并未伤害过她。
<br/>
<br/>        只不过，监视她的话，到底是监视她什么呢？
<br/>
<br/>        连星儿还未从郎安的身上摸清楚头绪，但至少，应该不是像裴萱妍所说的那样很危险。
<br/>
<br/>        “他是我的贴身保镖，你想多了。”想到这里，连星儿不冷不热地回了裴萱妍一句。
<br/>
<br/>        裴萱妍却意味深长地拖长了语调：“哦，原来是贴身保镖啊——”
<br/>
<br/>        “那你呢？刚看你那身手，倒是跟我家那些保镖叔叔们是一个级别的了。”连星儿一边舀着冰激凌，一边云淡风轻地说。
<br/>
<br/>        裴萱妍的眉梢挑了挑，会心一笑：“我也是明家小少爷的贴身保镖啊！”
<br/>
<br/>        “不是前女友就好。”连星儿淡定地笑了笑。
<br/>
<br/>        裴萱妍脸上的笑容渐敛，错愕地看着连星儿，质问道：“你这话是什么意思？”
<br/>
<br/>        “第一次见你的时候，你说你是明司寒的女人。现在看来，不是。”连星儿微微一笑。
<br/>
<br/>        其实她也不知道自己在庆幸什么。
<br/>
<br/>        “我告诉你一个秘密，怎么样？”裴萱妍故作妖娆地撩了撩自己的马尾辫，红唇微扬，“我不仅仅只是小少爷的贴身保镖，而且还是他的床伴噢！我们可是经常过‘性’生活的。哎呀呀！我怎么跟你这种黄毛丫头说这种事呢？男人嘛！没女朋友的时候，总会有那么一个固定的床伴。”
<br/>
<br/>        连星儿笑容渐渐僵硬，脸色也极其的不好看。
<br/>
<br/>        但随即，她又高傲地挺直了腰板，就像高高在上的公主殿下一样，蔑视着裴萱妍，说道：“明家是官宦世家，家教很严，招妓这种事情，明司寒是绝对不可能去做的。”
<br/>
<br/>        “你——”顿时，裴萱妍被连星儿的话怼得一句话也反驳不上来。
<br/>
<br/>        这丫头真的是太可恶了！
<br/>
<br/>        连星儿微微一笑，拿起冰激凌的小勺子，将刚刚舀好的那一勺冰激凌塞入了嘴中，自顾自地赞许道：“嗯，这冰激凌还是香草味的最好吃。”
<br/>
<br/>        裴萱妍抬眸，见明司寒和郎安正朝这边走过来，若无其事地笑了笑：“也对！毕竟香草味的冰激凌才是最经典的味道嘛！”
<br/>
<br/>        连星儿听着她这话，只笑不语。
<br/>
<br/>        明司寒和郎安坐下来后，连星儿和裴萱妍之间的气氛终于得到了缓和。
<br/>
<br/>        这茶余饭后，明司寒摊了摊双手，看了一眼郎安又看了一眼裴萱妍，意味深长地反问：“我和星儿接下来要去约会，两位是打算继续当多少瓦的电灯泡？”
<br/>
<br/>        “去看电影怎么样？”连星儿提议道。
<br/>
<br/>        明司寒微笑着点了点头。
<br/>
<br/>        结果，他一个人买了四张电影票。
<br/>
<br/>        郎安和裴萱妍似乎是跟他和连星儿杠上了，无论他俩去干什么，他俩都会跟着。
<br/>
<br/>        不过说来也巧，他们看的电影居然是吸血鬼与狼人。
<br/>
<br/>        看到荧幕里，帅气的狼人对女主的那份热情，连星儿鬼使神差地想到了郎安。
<br/>
<br/>        她的重点，不是狼人对女主的热情上，而是“狼人”这个身份上。
