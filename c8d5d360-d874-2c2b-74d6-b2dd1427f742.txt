　　有点灰就洗头，洗头也就算了，最可恶的是居然还洗澡！
<br/>　　这绝对是万恶的资本主义，绝对是无比的奢侈。
<br/>　　洗完舒舒服服的热水澡后，苏云居然已经做好了一个小菜。
<br/>　　菜，非常的简单。
<br/>　　一盘西红柿炒鸡蛋，和一盘凉拌皮蛋。
<br/>　　但这绝对是奢侈品。
<br/>　　西红柿鸡蛋，他得有西红柿还得有鸡蛋，外加菜刀这种东西。
<br/>　　菜刀也就罢了，但西红柿和鸡蛋，非常的珍贵。尤其是鸡蛋，一来这玩意他容易碎，不好保存，二来这玩意要鸡才可以下，所以非常珍贵。
<br/>　　但只要有晶石，一切不是问题。
<br/>　　“老师，你也吃啊。”林峰道。
<br/>　　苏云摇了摇头，“我做给你吃的。我不饿。”
<br/>　　林峰楞了一下，放下了碗筷。
<br/>　　“老师，你是不是有心事啊。”
<br/>　　苏云十分意外，“这你都看得出来？”
<br/>　　“老师，你内裤我都知道什么颜色的，何况是这。”林峰笑道。
<br/>　　“你……”
<br/>　　苏云都懒得和林峰都逗嘴了。
<br/>　　“唉，林峰，老师感觉现在的我虽然过的很安逸，但是这不是我想要的生活，我感觉现在的我就好像是被圈养起来的金丝雀一样。”
<br/>　　林峰愣了，没想到是这事情，林峰沉默了一下，的确，除了一开始林峰他还带着苏云外，其余时间都没有去带苏云出去了，因为林峰实力提升飞速，遇到的危险也越来越大，为了安全，没有带苏云出去。
<br/>　　而苏云老师又是个独立女性，长这么好看不去当小三，而是去当人名教师。
<br/>　　小三多好啊，开跑车买包包，这是社会上多少女生梦寐以求的，甚至一些女大学生，读书的时候就已以毕业之前找到一个大老板保养自己为主要学习任务，甚至一个班级的女生还相互攀比，比的不是谁学习好，而是包养他们的老板谁更有钱。
<br/>　　虽然听起来如此的歪风邪气，但是末世之前的三观已经开始扭曲了，笑贫不笑娼，这是不得不承认的事情。这也是为什么第一邪教他会如此壮大，甚至源源不断有人加入其中的原因之一了。
<br/>　　“老师我虽然也很乐意给你们做饭做菜，但是我也很向往在外面杀丧尸杀怪物。而不是这日复一日的在这大屋子里住着。”苏云道。
<br/>　　林峰闻言点了点头，“老师，这倒是我疏忽了，既然你想要去外面历练，那你就去吧，我不会阻拦你的。”
<br/>　　“你同意了？”苏云愣了一下。
<br/>　　林峰笑了，“老师，你干嘛要这样子问我啊，难道你已经帮我当成你男人吗？干什么还要经过我的同意？我们是平等关系，而且你还是我老师是我姐姐，你做什么我都很乐意，你若是做菜做累了，想要去外面，那我支持你，大不了大家吃泡面嘛，可你若是在外面厌倦了鲜血，想要回来做菜，我依然还是支持你。”
<br/>　　苏云闻言心中悸动了一下。
<br/>　　然后她深深的看了眼林峰，“谢谢弟弟，有你这个弟弟，我感觉是我这一辈子最大的幸运了。”
<br/>　　第二天。
<br/>　　林峰吃完苏云做的早餐后。
<br/>　　苏云就要出去做任务猎杀丧尸了，对方身上穿着火凤肚兜，自保没问题。
<br/>　　“那林峰我走了。”
<br/>　　苏云道。
<br/>　　“注意安全。不管是活人还是丧尸，都要注意。”林峰嘱托道。
<br/>　　“知道了，你老师我不是傻白甜。”苏云笑道然后就穿着个干练的服装离开了。
<br/>　　“唉，苏云老师真是的，我这么有钱，还不给我包养，非要证明自己。”林峰无奈的摇了摇头，苏云老师出去杀怪物，不是为了晶石，晶石林峰有不少，而纯粹的就是为了体现自己生命的价值。
<br/>　　这是一些混吃等死之辈，随波逐流之辈，一辈子都无法体会到的一种境界。
<br/>　　不过很快林峰也忙起来了。
<br/>　　因为有许多人来面试了。
<br/>　　“你好，这里是老龙口矿业公司吗？”
<br/>　　一个男人小心翼翼的问道。显得十分的拘谨。
<br/>　　“没错，你是来面试的吧。”林峰道。
<br/>　　“是是是。”这个男人还很激动。
<br/>　　“还有我们。”
<br/>　　这一看，林峰才发现原来屋子外面已经挤满了几十个人。
<br/>　　林峰让他们都进屋内来，这群人一个个都很紧张。
<br/>　　“唉，我之前住的是别墅，可惜啊可惜，现在只能住大街了。”其中一个看起来有点气势大腹便便的男人感慨道，他之前是情趣用品的生产商，情趣用品虽然现在许多人还很羞涩，但实际上利润很大，而且这几年需求也很高，他厂子每年都有近千万的利润，住的也是大别墅，但现在一切化为了虚无。
<br/>　　不少人看到林峰住的房子都感慨，林峰这个房子放在末世前，豪宅那根本算不上，面积虽然不小，但是装修一般，要知道那些豪宅装修都是几千万的。
<br/>　　顶多算是个小资人家。可是现在，那就是豪宅了。
<br/>　　这群人看的眼睛都冒光，羡慕嫉妒恨。
<br/>　　“快看，这果盘里不是空的，他居然还在果盘里放了瓜子！这味道！这香气！我敢肯定，这绝逼是洽洽香瓜子！！”
<br/>　　“我去，你们看？那是什么？黄黄的，长长的，好熟悉却又好陌生？”
<br/>　　“难道说……那是香蕉？”
<br/>　　咕噜。
<br/>　　许多人都看到了放在茶几上果盘里的水果，一个个都咽了口唾沫，他们万万没有想到，自己居然可以看到新鲜的！还没有被踩得稀巴烂的水果！！
<br/>　　林峰自然把他们的表情都看在眼中，只是心中唏嘘感慨。香蕉和瓜子居然都让人这么震惊了。
<br/>　　不过林峰也不是烂好人，这些东西他林峰也同样都来之不易，虽然是天上掉的，但毕竟还要他去捡不是。
<br/>　　“这些东西，你们若是能够成为我老龙口矿业公司的员工，那么逢年过节，这些东西都会作为福利发送的。”
<br/>　　林峰道。
<br/>　　“什么？还有福利？”
<br/>　　“太奢侈了吧。”
<br/>　　许多人都激动起来。没有想到干活居然还有福利。这在末世之前很正常，各大公司都有节假日福利什么下午茶团建之类的，现在统统没有，一个不少的把晶石结算了，那都已经是走大运了。
