　　东风吹，战鼓擂，轩辕蚩尤谁怕谁？
<br/>　　“咚——”
<br/>　　战场上，战鼓敲响，鼓声悠扬，声波中蕴含着一种神威，那是神兽的威压以及神灵的威严，好似万兽之王，无上神灵在看着他们，顿时九黎部落勇士们的坐骑猛兽纷纷瑟瑟发抖，匍匐在地，不敢起来。
<br/>　　不少九黎族的勇士猛然摔落在地，悲惨不已。而与之相反的轩辕一方，却是士气大涨！人族共主亲自给你敲战鼓，这还不鼓舞士气吗？
<br/>　　“看，那是共主在擂鼓，我们能够看见敌人了，杀啊！”
<br/>　　“兄弟们，有共主在给我们助威，给我杀！”
<br/>　　“杀！”
<br/>　　见到这一幕，轩辕一方士气大盛，人人都变得凶猛起来，宛若化身神灵与神兽，全身上下有使不完的劲，杀敌变得更加勇猛了。
<br/>　　胜利的天平，渐渐向着轩辕一方倾斜。
<br/>　　……
<br/>　　这场大战整整进行了三个月，三个月后，九黎一族大败，数十大巫被生生击毙，蚩尤的九黎魔兵，也被阐教众仙击溃灭杀，最终只剩下了蚩尤与刑天两大巫族强者。
<br/>　　看着战场上那大战三月只剩下孤家寡人却仍然屹立不倒的蚩尤，轩辕心中也有些钦佩，这种人，天生就是为战而生。不过，钦佩归钦佩，公事公办，该动手的时候，他一样会毫不留情，蚩尤他一定不能活下来。
<br/>　　公孙轩辕此刻看着那破板不堪的战场，心中却没有一点胜利的喜悦之情，他一闭上眼便会想到他们脚下这片土地，埋葬着千千万万的人族勇士的骸骨。这片土地，是他们用鲜血换来的。
<br/>　　人族内战消耗如此多的人类，却让外族看了笑话，想到这里，他的心中就有些隐隐作痛。他的眼神开始变得冰冷，华夏之地，岂容外族侵略，这一刻，他已经将自己今后的目标定在了外族身上了。
<br/>　　接着，公孙轩辕眼神有些复杂的看着战场上的魔神蚩尤与战神刑天，问道：
<br/>　　“尔等，还有何话可说？”
<br/>　　“呸！要杀要剐，我蚩尤又岂会皱一下眉头？我蚩尤一生征战无数，无愧于心，本欲借此机会重振巫族，但奈何人算不如天算，难道我巫族真的要如此落寞下去吗？天意真的如此难违？”
<br/>　　魔神蚩尤满身淋血地站在战场上，仿佛一块屹立不倒的丰碑，可声音中却透露着一种难言的孤寂与落寞。
<br/>　　“天意如此，造化弄人罢了！”公孙轩辕皱了皱眉，淡淡说道，心中却在暗暗思索，我人族将来的命运又会如何。
<br/>　　“天意？呵——”蚩尤有些讽刺的问了一句，之后接着说道：“我巫族早已不信这天了，这天还是父神的天吗？天不佑人，尊天何用？”
<br/>　　众仙都有些皱眉地看着蚩尤，这话说得有些大逆不道，修士修仙不就是修天吗？蚩尤仅仅这话就得罪了大多数修仙者了。
<br/>　　“来人，将蚩尤与刑天斩首示众！”公孙轩辕看着战场上的两人，果断地下了命令。
<br/>　　“是。”
<br/>　　就在此时，变故陡生，天空突然暗淡下来，给人一种难言的压抑感，每个人心头都仿佛被压着一座巨山。
<br/>　　“哈哈哈，好一个‘天不佑人，尊天何用？’，本尊劫魔殿中正好缺少一位魔君，你正好合适！”
<br/>　　一道桀骜中充满狂傲的声音响起，紧接着，一只灰色巨手遮天，直接将战场上的蚩尤与刑天抓了起来，然后，消失不见。
<br/>　　虚空中，一道人影出现，被扔进了盘古殿中，那不是刑天又是谁。
<br/>　　洪荒北部，北俱芦洲，盘古殿中。
<br/>　　玄冥祖巫看着出现在眼前的刑天，似乎是明白了什么，对着虚空抱拳谢道：“多谢魔尊出手相救我巫族之人。”
<br/>　　虚空中并没有什么反应，玄冥祖巫也不在意，他知道对方听得见这句话，之后，她又对着虚空拱了拱手，才带着刑天走了。
<br/>　　战神刑天已走，哪还有什么“刑天舞干戚，猛志固长存”的事情？
<br/>　　“哼！”
<br/>　　诸圣道场之内，圣人们冷哼一声，对于万劫魔尊这番行为表示不满，心中不知这万劫魔尊又要搞什么，但却将对方记载心中，绝对不能轻视。毕竟对方是能够让西方二圣吃瘪的人物，不是好惹之人。
<br/>　　对于一个蚩尤，他们并没有多么在意，毕竟逐鹿之战已经结束了。只是人族众人却有些恼怒，但也没有办法，不至于为了这么点事就与劫魔殿开战。
<br/>　　战场上，早已没了蚩尤与刑天的身影，只是公孙轩辕刚刚下令斩首示众，别人就来劫法场了，这打脸也打得太快了吧。
<br/>　　公孙轩辕脸色非常不好看，看了众仙一样，沉声问道：“如今该如何是好？”
<br/>　　这时，广成子突然一愣，他刚刚接受到元始天尊的信息，于是，他对着公孙轩辕说道：“逐鹿之战已经结束，九黎大败，蚩尤败亡，人族一统，尊公孙轩辕为人族共主！”
<br/>　　对啊，九黎族都大败了，自己胜利了，还管那些细枝末节干什么，弄不好平白无故得罪一个圣人。于是，众仙也只好面对这个事实，管那么多干什么，有功德拿就行了。
<br/>　　就这样，逐鹿之战结束后，人族就恢复了一统。人族公孙轩辕成为人族共主，开始了他的人族统治生涯。
<br/>　　轩辕奠定天下后，制定国家管制制度，监管天下，同时，他对外抗战。内修圣道，外修霸道。
<br/>　　公孙轩辕感念龙族相助之恩，以龙族形象为人族图腾国兽。
<br/>　　至于应龙，自从当初大战之后，他将旱魃葬于黄泉冥海，他有预感，日后定能再次见到旱魃。之后，他便一直守在人族部落之中，好似在等着某人的归来！
<br/>　　而被葬于幽冥的女魃，却又要开始一段四大僵尸始祖的故事。
<br/>　　人族轩辕在位期间，外族来犯，轩辕举兵反抗，一生征战，抗击外族，扩大疆土，妖魔鬼怪，不敢轻易来犯。
<br/>　　轩辕一生戎马，杀到外族不敢侵犯，是杀伐最重的一位人族共主。
<br/>　　轩辕在位期间，人族强盛，政治久安，文化进步，有许多发明，如文字，音乐，历数，宫室，舟车，衣裳和指南车等。
<br/>　　同时，轩辕的修为也没有落下，因为人族共主的原因，整个人族的气运加持下，他的修为用突飞猛进都不足以形容。逐鹿之战后，刚刚继位，轩辕便突破到转轮皇者境，在之后的第一个百年内，凝聚出第二道转轮，成为皇者中期，四百年后，凝聚第三道转轮，成为转轮后期。
<br/>　　最后一百年内，人皇公孙轩辕研究经典道藏，开创出自己的绝学，其中包括多方面，并取名为《黄帝经》。
<br/>　　《黄帝经》又分为《黄帝内经》与《黄帝外经》，其中《黄帝内经》主要是一些医术养生之法，而《黄帝外经》才是他一生所学武道、剑道、仙道等的总结。其内包含一些绝世剑诀和仙法，如葬剑诀，乱剑诀，以及自创的平乱诀等。又如御女心经，轩辕道经等。
<br/>　　相传轩辕御女三千，得道飞仙，还是有些道理的。
<br/>　　五百年人族共主之期已满，公孙轩辕传位其孙颛顼。传位之日，轩辕于泰山搭起祭坛，焚香祷告，祈天拜祖，最后才能功德圆满。
<br/>　　当天，无量功德降下，金光灿灿，染黄了整片天空。
<br/>　　公孙轩辕得无量功德之后，修为直接飙升至半步帝境，证得人皇果位，与天皇伏羲，地皇神农一般，是人族的业位之一。
<br/>　　当日，在万众瞩目下，轩辕乘龙飞天而去。
<br/>　　其余帮助过人皇的修士，也或多或少的得到了一些功德，当然，因为人数众多的原因，功德分散开来也没有多少，就如轩辕之师广成子，他是这些人中除了公孙轩辕外得到功德最多的人，但也只是让他修为略有精进而已，想要突破到准圣中期，还差一点。
<br/>　　至于其他人，得到的功德那就更少了。
<br/>　　元始对此没有丝毫的意外，他想要的好处已经得到了。人族与阐教之间已经存在气机的联系了，以后人族的气运中，必有阐教一份。只因为三皇之一的人皇是阐教弟子。
<br/>　　仅此，足矣。
