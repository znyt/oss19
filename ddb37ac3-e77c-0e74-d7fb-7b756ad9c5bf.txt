　　论起实力。
<br/>　　眼前这一位荒古门半步天人境强者，虽然不如少帝，可是却至少算得上是半步天人境之中的强者了。
<br/>　　尤其是这一尊荒古门半步天人境强者，他身上的气息，其实比起五行刀圣，还要弱上几分。
<br/>　　可是爆发出的力量，却是要超越五行刀圣。
<br/>　　这便是功法的差距。
<br/>　　七大宗门的功法，皆是传子无比古老的时代，在那个时代的九域，还是曾经的四圣域之一。
<br/>　　哪怕五行刀圣，出身自的势力，五行天，也是大罗天星域之中的巅峰势力。
<br/>　　可要是跟荒古门相比，底蕴却是差的远，这种底蕴，在实力弱小的武者之中，差距不大。
<br/>　　可是一旦到半步天人境，甚至真正天人境的时候，便是足以显现出差距。
<br/>　　九域中的半步天人境强者，要比起大罗天星域之中的，还要难缠的多！
<br/>　　不过楚风眠却是丝毫不惧，他现在已经是踏入到了九劫古帝了，实力暴涨。
<br/>　　“给我破！”
<br/>　　楚风眠看着天空之上，冷喝一声，他手中剑锋之中的力量突然间暴涨起来，化为了一道锋利剑气，冲天而起。
<br/>　　那凝聚而出的荒古大戟，在这一道剑气之下，猛然碰撞，在下一刻，当即是破碎开来，化为了一股股的灵力，消散在了天空之上。
<br/>　　“什么？”
<br/>　　从虚空深处，一声惊疑的声音，突然响彻而起，似乎是这出手的半步天人境强者，都没有想到。
<br/>　　楚风眠居然是足以化解了他的攻势。
<br/>　　他想要一招直接诛杀楚风眠，却是被楚风眠这样，轻易抵挡下。
<br/>　　九劫古帝，在半步天人境强者看来，简直是可以随意捏死的存在，哪里想到楚风眠居然如此轻易的，化解了他的攻击。
<br/>　　“看来，你果然是得到了剑道门的传承，甚至是将剑道之主那个老家伙是传承都给得到了，不然不可能有着如此逆天的实力。”
<br/>　　虚空之中，再度传递出来了那一尊半步天人境强者的声音。
<br/>　　以九劫古帝，爆发出力量，足以比拟半步天人境的强者。
<br/>　　这种实力，的确称得上是逆天二字。
<br/>　　毕竟哪怕是大罗天星域四大势力之一，圣龙秘境之中最为杰出的天才之一的无道，也只是能够堪称接近半步天人境的实力。
<br/>　　催动秘术之后，一身实力，足以碾压帝尊境，可是跟真正的半步天人境相比，还相差的多。
<br/>　　而要像是楚风眠这样，一出手爆发出来的力量，已经是足以跟半步天人境，平分秋色的，根本不可能。
<br/>　　天下之间，根本无人应该能够做到才对。
<br/>　　“看来剑道门，果然是死而不僵，还留下了你这样一个余孽，不过也正好，剑道门之中的许多秘密，宝藏，肯定在你的手里！我先擒下你，看看剑道门到底是搞得什么鬼！”
<br/>　　这一道声音，自说自话之间，突然之间又是出手了。
<br/>　　在楚风眠的头顶之上，空间突然被撕裂开来，一条长河显现而出，这一条长河之中，蕴含这古老无比的力量。
<br/>　　荒古长河！
<br/>　　比起荒古大道，更加强横的秘术。
<br/>　　这荒古大道，便是足以借助荒古之力，来进行对敌，自身踏入荒古之中，足以躲避无数攻击。
<br/>　　而这荒古长河，却是足以将武者硬生生拉扯在了其中，被其中无数的荒古之力所镇压。
<br/>　　在古老的时代之中，这荒古长河，可是号称足以镇压同境界的人物了。
<br/>　　像是眼前这一尊荒古门半步天人境强者出手，爆发出这一招，足以镇压同为半步天人境的存在。
<br/>　　荒古长河凝聚显现而出的一刻，楚风眠便是感觉到，一股巨力，在拉扯这楚风眠的身躯，要将楚风眠的身形，拉扯进入其中。
<br/>　　“想要镇压我？笑话，就凭你一个人？简直是在痴心妄想！”
<br/>　　楚风眠看到这荒古长河，大笑一声，从他的背后，突然之间一道巨大无比的漩涡，浮现而出。
<br/>　　正是吞天秘术！
<br/>　　随着楚风眠实力的强横，这先天神兽血脉之中蕴含的强大力量，楚风眠也足以越发的掌控。
<br/>　　这吞天秘术打出的一刻，便是在楚风眠的头顶之上，不断的变大，强横的吞噬之力，笼罩在了那荒古长河之上，疯狂的催动这。
<br/>　　在这一道荒古长河之中，其中源源不断的荒古之力，都被拉扯了出来，被吸入到了吞天秘术之中，化为了楚风眠的力量。
<br/>　　这荒古长河，虽说是借助荒古时代的力量，可根源还是来自于那荒古门半步天人境强者的力量。
<br/>　　楚风眠催动吞天秘术，疯狂的吞噬起那荒古之力来，几乎是在吞噬这那一尊半步天人境强者的力量。
<br/>　　“吞天兽血脉？吞天秘术？这种东西你怎么可能得到！”
<br/>　　从虚空之中，传递出了无比震惊的声音。
<br/>　　那一尊半步天人境强者，也是见多识广，一眼便是认出了楚风眠头顶之上，那深邃漩涡的真正来历，到底是什么。
<br/>　　吞天兽！
<br/>　　这可是只有在九域古籍之中才有所记载过的先天神兽。
<br/>　　在九域之中，根本是没有吞天兽的血脉，所以无人掌控过这等力量。
<br/>　　可楚风眠却是之前在妖古大陆的时候，机缘巧合之下，得到了这吞天兽血脉。
<br/>　　在对付那三大宗门武者的时候，楚风眠也没有动用过，以至于令这一尊半步天人境强者，顿时都有些措手不及。
<br/>　　不过他也是当机立断，突然之间从虚空之中，无数灵力爆发而出，似乎是要打破这吞天秘术，阻止吞天秘术在继续吞噬下去。
<br/>　　可就在这时，楚风眠却是突然动了，当这无数灵力打出的一刻，楚风眠同时出手了，他的手中剑锋灵力凝聚吗，猛然一剑向着虚空深处斩去。
<br/>　　“不用在装神弄鬼的了，给我滚出来！”
<br/>　　楚风眠打入这虚空之中的剑光，猛然爆炸开来，强横的力量，席卷着整个虚空之中。
<br/>　　在那虚空深处，一直躲藏的荒古门半步天人境强者，也是受到了剑光的横扫。
<br/>　　一股股灵力波动传递而出，旋即一道身影，猛然是从虚空之中飞了出来。
