　　望着眼前那能够毁灭一切的空间乱流，一阵风暴刮过，周边空间瞬间支离破碎，什么也不剩下。
<br/>　　“你说的地方便是此处吧。”元始看着眼前的风暴，面色淡然道。
<br/>　　“不错，便是此处，我等此行的目的便在其内。”道祖点头道。
<br/>　　“希望不会猜错，我倒是想要看看，这斩仙台是何人所为？又有何目的？”元始淡淡道。
<br/>　　“走吧！”
<br/>　　说着，元始率先先前走去，看着那狂暴无比的空间乱流，以元始强悍的肉体都有些支撑不住，他心神一动，头上立刻便多了一个混沌大钟，大钟护持这肉体，方才能够让他再这时空乱流中游走。
<br/>　　后方道祖见到元始头顶的混沌大钟，眼色一凝，他竟然完全看不透，这是什么级别的至宝？眼前这个自称元始的人，果然不简单。同时，他也取出自己的法宝，一块青色的玉牒，悬浮在他头上，不断散发出青色玄光，缓冲着周边的空间乱流。
<br/>　　“造化玉牒？”
<br/>　　元始看见这一幕，眼神一缩，问道。
<br/>　　“不错，这就是贫道法宝，造化玉牒。”道祖回答道。
<br/>　　“嗯。”元始点点头，没有多说。心中却在思索，相似的世界，相似的结局，相似的人物，相似的法宝，这个世界与洪荒，存在太多的相似了。诸天万界难道真有这样的世界，还会不会有更多相似的世界，那么哪个世界又是主界呢，唯一真界，永恒世界又是什么？
<br/>　　这一切都是迷，有太多太多的未知，等着元始去探索，可他又能在这探索未知的道路上走得多远？
<br/>　　“轰隆隆！”
<br/>　　越往前，风暴便越是剧烈，一般真仙到此，恐怕都活不过一时三刻，片刻时间，便会化成碎片，随着空间风暴吹走。
<br/>　　还好，两人都不是一般人，而且都有至宝镇压，只要小心点不被风暴卷走，落入未知的次元空间就好，虽然走得慢，但至少安全。
<br/>　　大约前进三个时辰后，两人终于有惊无险地度过了混乱的空间风暴，然而，九鼎危险过去了，新的危机再度降临。
<br/>　　两人踏入全新的区域之中，入眼所见，只是一片灰色，天边是灰色的，大地也是灰色的，一股难以言明地莫大凶险，呼啸而来。
<br/>　　“沙沙沙……”
<br/>　　像是风吹起沙粒的声音，自禁区深处不断传来，看似轻微，却又蕴含着一股无与伦比的毁灭之力，似乎是要毁灭一切。
<br/>　　“小心！”
<br/>　　元始放眼看去，只见一股灰色迷雾不知何时出现在空间中，紧接着，便可以看到一股黑色的气流如同潮水一般自地平线上汹涌而来，声音依旧很微弱，不仔细听，完全听不到，像是落叶纷飞的声音，沙沙作响，非常轻柔，完全看不出一点儿狂暴的样子。
<br/>　　但就是这样的温柔，却又蕴含着一种莫大的危机，好似被死神盯住一般，逃不脱，躲不掉。那是一种很压抑的气息，由远及近，像是死神的温柔。
<br/>　　黑色的雾气之中蕴含着强烈无比的黑暗能量，其中还包括诅咒、腐蚀、侵染、污浊等气息，更蕴含着浓浓的死亡之意，即便强如元始这样的准圣强者，也不得不小心应对。
<br/>　　“沙沙……”
<br/>　　轻微的声音响起，黑色雾气好似有眼睛似的，径直地朝着两人入侵过来。接下来让元始吃惊的是，混沌钟的抵御能力竟然不能抵御这种黑色雾气。
<br/>　　那雾气径直穿透过混沌钟的防御力，朝着元始身上侵袭过来。元始连忙运转护体罡气，护体罡气终究是起了点作用，抵挡住了黑雾，但很快元始的脸色就难看起来了，因为这罡气消耗的太快了，那些罡气很快便被黑色物质给吞噬了，黑色雾气一进入元始体内，便开始疯狂掠夺，疯狂吞噬，无尽黑色气息喷涌而出，想要将元始身上的每一个细胞都污染。
<br/>　　让元始感到荒谬的是，他仿佛在这种黑雾之中感受到了生命力，这些黑雾就好似一个个小了无数倍的生物组成的，这种生物身上带着黑暗的能量，更混合诅咒、腐朽、等能量，这一刻，元始的心里想到的是被病毒感染的丧尸，丧尸之所以成为丧尸，那就是因为他自己的身体掌控权被病毒控制了，意识泯灭，他就犹如行尸走肉，生死不能自已。
<br/>　　如今自己身上的情况与丧尸十分相似，那些黑色物质便如同病毒一样，它们在疯狂的破坏自己身上的组织与细胞，争夺身体的掌控权，只不过，这些东西可比病毒厉害千万倍不止。
<br/>　　元始并不着急，因为他从些黑色物质中找到了熟悉的力量——黑暗气息。他同样精通黑暗大道，当初他还灭杀过黑暗魔神呢。
<br/>　　想到此处，元始调起体内的黑暗道则，向着那些入侵身体的黑色物质发起攻击，结果很有效，那些入侵他体内的黑色物质，有的被逼出了体外，但效果也有限，不能完全逼出。
<br/>　　元始的黑暗法则与这些黑暗物质也有不同，如果说元始的黑暗是正统的，那么这些黑色物质中的黑暗气息便是邪恶的，令人作呕的，若说元始的黑暗是阳性，那么这些黑色物质的便是阴性，两者是现对的，但也不完全相对。
<br/>　　元始又依次实验了其他道法神通，发现光明属性的道则或者宝物基本可以压制，元始大喜，黑暗与光明两种道则在体内交织而成，驱散了那些诡异的黑色物质。
<br/>　　元始看了看旁边的道祖，见道祖好似早有准备，他并没有被黑色物质入侵，或者说他有对付黑色物质的办法，只是他之前没有说出来而已，元始深深的看了一眼道祖，并没有多说，继续向前走去。
<br/>　　不管是对他的试探，还是别有用心，元始都不想管。只是在这一刻对这个道祖他心里多留了个心眼。
<br/>　　“道友好本事，竟然凭借自己的力量将这黑色迷雾逼出体外，佩服！”道祖对着元始道。
<br/>　　“我要佩服道友才是，道友竟然能够抵挡这诡异的黑色物质，真是令人惊讶！我要不是有点法宝，恐怕早死了。”元始假装有些后怕道，他并没有说是靠自己的法则逼出黑色物质，而是将一切推到法宝上。
<br/>　　“道友好运道，竟然有光明至宝，佩服！”道祖也认为是元始用了法宝，他并没有过多怀疑。
<br/>　　“道友有点不够意思，竟然早知此事，为何之前没说？”元始淡淡道。
<br/>　　“贫道也是上次被真主算计进入此地，还好我运气好，身上刚好有件法宝克制这种诡异黑雾。至于为何没说，我以为道友早已知晓了。”道祖说道。
<br/>　　“呵呵。”
<br/>　　元始无话可说了，早已知晓？本座知晓了还需要你带路吗？
<br/>　　两人顿时没有说话，继续向前走去。
<br/>　　也不知走了多远，更不知走了多久，突然地，两人遭受到了黑色迷雾之外的第一次攻击，一道强大的力量，竟然躲在这黑雾之中，避过两人的感知，轰杀过来。
<br/>　　“嗡……”
<br/>　　混沌神钟轻轻一荡，绝世神威，光华闪烁，挡住了这绝强一击，同时，两人也看清了黑雾中的迷影。
<br/>　　那是一个形生物！
<br/>　　……
