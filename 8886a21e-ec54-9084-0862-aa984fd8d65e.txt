        “不是，路边捡的，不认识。”明司寒淡淡地说。
<br/>
<br/>        “不认识你就敢随便捡回家？”
<br/>
<br/>        “总不能见死不救吧？”
<br/>
<br/>        应光济不禁皱了下眉头：“如果是不认识的话，可能就麻烦了。这个女人中的是枪伤，说明她的来历不简单。”
<br/>
<br/>        “她醒了后，我会让她马上离开。”明司寒接着说道。
<br/>
<br/>        应光济微微点了点头，收拾完医药箱后，提着医药箱离开了别墅。
<br/>
<br/>        翌日。
<br/>
<br/>        连星儿起床后就去洗了个澡，下楼吃早餐的时候，童筱颖将一个六寸大的巧克力慕斯蛋糕推到了连星儿的跟前。
<br/>
<br/>        “待会你去一下小寒的家，把这个带过去，就说是谢谢他昨晚送你回来。”童筱颖微笑道。
<br/>
<br/>        连星儿顿时憋着嘴，一脸不乐意：“我还以妈咪是做给我吃的了，弄了半天，是给明司寒那家伙的啊！”
<br/>
<br/>        “你可以跟他一起吃啊！正好，你们俩可以培养一下感情。”童筱颖笑眯眯地说。
<br/>
<br/>        连星儿抿了抿唇，一手托腮，一手拿着茶匙搅动着杯中的牛奶，慵懒地回答道：“不用培养感情了，反正我跟他将来又不会真的结婚。”
<br/>
<br/>        “什么？”童筱颖诧异道。
<br/>
<br/>        连星儿立马将双臂交叠在餐桌上放平，咧嘴道：“妈咪，我跟你说噢！明司寒答应我，说他会去跟他爷爷说解除联姻的这件事情。”
<br/>
<br/>        “所以，这就是你选他的理由？”童筱颖有些恍然大悟，她早就料到这丫头的葫芦里卖得不是什么好药了。
<br/>
<br/>        “当然！”连星儿笑嘻嘻地回答。
<br/>
<br/>        童筱颖却不以为然，但也只笑不语。
<br/>
<br/>        这群小孩子之间的事情，她不好说，但也不是没看出来。
<br/>
<br/>        那个明司寒应该是对她的这个小丫头有好感。
<br/>
<br/>        至于她家这小丫头嘛！
<br/>
<br/>        可能在感情方面开窍得比较晚，但迟早有一天，也是会有她自己心仪的男生的。
<br/>
<br/>        “我不管你们打的是什么如意算盘，总而言之，你吃过早餐后，就把这个巧克力慕斯蛋糕给他送过去。他家的住址，我发你的手机上。”童筱颖淡定道。
<br/>
<br/>        连星儿忽然想到了什么，一脸狐疑地看着童筱颖：“我都不知道他住哪里，妈咪你却知道。莫非，你和爹地派人调查了他？”
<br/>
<br/>        “这难道不是必须的吗？妈咪和爹地的未来女婿，妈咪和爹地有权知道他所有的一切。”童筱颖笑道。
<br/>
<br/>        连星儿不禁好奇地问道：“那你们调查后，他有没有什么不良嗜好？”
<br/>
<br/>        “打游戏算不算不良嗜好？”童筱颖忍俊不禁地反问。
<br/>
<br/>        连星儿点了点头，又摇了摇头。
<br/>
<br/>        “他是在搞电竞那方面的工作。”童筱颖若有所思地说。
<br/>
<br/>        提起这个事情，连星儿突然莫名其妙地将明司寒和她在游戏里拜个大神师父“余寒犹厉”给联系在了一起。
<br/>
<br/>        吃过早餐后，连星儿带着蛋糕，让家里的司机送她去了明司寒的别墅。
<br/>
<br/>        在别墅区大门口做了登记后，连星儿下了车，让司机先回去，自己提着巧克力慕斯蛋糕独自按响了明司寒的别墅的门铃。
<br/>
<br/>        而开门的人，不是明司寒，而是一个身上只穿着一件白色长袖T恤，衣摆刚好裹臀，露出两条雪白的大长腿，长发披肩，脸色惨白的女人。
<br/>
<br/>        女人的右腿大腿上还绑着纱布，渗出了一丝血迹。
<br/>
<br/>        连星儿看得有些目瞪口呆，愣了半晌后，才问道：“明司寒呢？”
<br/>
<br/>        女人也犹豫了一下，似乎像是在思考着什么，抬起手来，将胸前的长发撩到耳后根，不冷不热地回答道：“他还在睡觉。”
<br/>
<br/>        “？！”听到这里，连星儿不免有些想歪了，“你是他女朋友？”
<br/>
<br/>        女人抿了抿唇，警惕性地点了点头。
<br/>
<br/>        她不清楚面前这女孩的来历，所以犯起了职业病，回话的一字一句都很小心翼翼，生怕暴露了自己的身份。
<br/>
<br/>        “你的腿怎么了？”连星儿关问道。
<br/>
<br/>        女人不经意间地皱了下眉头，不冷不热地回答道：“昨晚太激烈了，不小心被他弄伤的。”
<br/>
<br/>        “啊？”连星儿一开始没听懂，缓过神来后才明白这女人是什么意思，不禁喜上眉梢。
<br/>
<br/>        怪不得明司寒那家伙答应她，会跟她解除婚约关系，原来是金屋藏娇啊！
<br/>
<br/>        连星儿想到这里，立即将手中的巧克力慕斯蛋糕递给面前这个冷艳的女人：“这是我妈咪要我带给他的，你拿进去跟他一起吃吧！我就不打扰你们了！”
<br/>
<br/>        女人从连星儿的手中接过蛋糕后，二话不说就把门给关了。
<br/>
<br/>        一阵门风扑面，连星儿只觉一脸莫名其妙。
<br/>
<br/>        不过，她没多想，转身就离开了。
