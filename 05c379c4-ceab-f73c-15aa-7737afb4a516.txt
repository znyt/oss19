　　“额。”
<br/>　　萧逸要紧了牙，面具之下的脸色，痛苦到极点。
<br/>　　这种疼痛感，一如数月前他在山洞内强行用太阴太阳之眼窥探天地武道时的反噬。
<br/>　　只不过，那时的反噬，乃是无比的炙热中，夹着惊人的冰冷感。
<br/>　　那时的反噬，让他身体一边炙热如火，一边冰冷如霜。
<br/>　　冰火交加的折磨，连他都无法忍受。
<br/>　　现今这种炙热融化感，便一如当世的炙热如火之感。
<br/>　　可还有另外一点不同的是，当时窥探天地武道的反噬，仅仅是让他难受到极点，并未要他性命。
<br/>　　可是，现今龙炎的反噬，痛苦中，却又让萧逸产生了一股致命的威胁感。
<br/>　　冰鸾剑、太阴太阳之眼，终究是他的东西，与他一体，怎可能要他性命。
<br/>　　最大最大的反噬效果，便也只是修为全失。
<br/>　　如以前的冰鸾剑反噬，顶多就是让他痛苦万分，甚至好几次险些让他经脉、小世界、气泉尽皆受损，一身武道修为就此尽丧。
<br/>　　如果用‘身死道消’四个字形容的话，那么，顶多便是‘道消’。
<br/>　　可龙炎的反噬，却是等同‘身死’。
<br/>　　龙炎，并非他萧逸掌控的东西，只是他借用的外物。
<br/>　　现今他的身体，便是浑身通红，一股股热气，从身体中溢散而出。
<br/>　　甚至是他闷哼一声，口中吐出的，也是热腾腾的白气。
<br/>　　再这般下去，萧逸甚至相信，自己真的会彻底融化而死。
<br/>　　……
<br/>　　萧逸的速度，本是飙升到极致。
<br/>　　短短时间，他便出了六龙地域。
<br/>　　毕竟在六龙地域内，龙啸等同无敌罢了，可出了六龙地域，龙啸只是传奇之下最强。
<br/>　　出了六龙地域，萧逸未必就怕了龙啸。
<br/>　　可现在，他的速度却骤然而减。
<br/>　　浑身炙热下，不仅难受至极，连体内元力，竟也在快速变得紊乱不堪。
<br/>　　体内的那股炙热，仿佛在燃烧着他的整个身体。
<br/>　　他的小世界内，元力沸腾不堪，凌乱不堪。
<br/>　　甚至于，元力有了外泄，不断冲撞之状。
<br/>　　如此情况，让他还如此保持御空。
<br/>　　剧烈的疼痛下，甚至让得他眼神迷离，心神渐渐受损。
<br/>　　“好热…”
<br/>　　额头上的汗水，早便密布。
<br/>　　汗水，不是滴落眼眸，让得他本就模糊的视线，变得愈发难受。
<br/>　　热，这是萧逸现今唯一的感觉。
<br/>　　热到了极点，热到了身体即将融化。
<br/>　　“水…”萧逸的意识，愈加模糊。
<br/>　　龙炎的反噬，太过可怕，也太过迅速了。
<br/>　　快到了萧逸甚至没多少反应的时间。
<br/>　　但萧逸很清楚，若他现在失去意识，等待他的，将真的只有死亡一途。
<br/>　　他的身体，将在龙炎反噬下，被焚烧殆尽。
<br/>　　数分钟后。
<br/>　　萧逸模糊的视线中，蓦地看到前方有一辽阔大湖。
<br/>　　萧逸未多想，连忙朝那大湖飞去。
<br/>　　天知道他这数分钟的煎熬下，是如何撑住保持意识的。
<br/>　　轰…
<br/>　　火焰流光，轰然飞落湖内，激起大片水浪。
<br/>　　冰冷的湖水，霎时减缓了萧逸的一丝灼热感，意识也恢复了些许清明。
<br/>　　“还不够冷。”
<br/>　　萧逸咬了咬牙，身影疾速下降。
<br/>　　炙热的身躯，在水中的穿行，几乎每时每刻都会带起大片白气。
<br/>　　那是他周遭的湖水被瞬间蒸发。
<br/>　　一直降落，越是下降，湖水温度愈低。
<br/>　　待得下降数千米，冰冷的湖水，仿佛冰寒刺骨。
<br/>　　这个范围，已是湖底，且暗无天日。
<br/>　　这漆黑、冰冷的环境下，方让萧逸愈加‘舒服’了些许。
<br/>　　但，也仅此而已了。
<br/>　　萧逸手中光芒一闪，一株散发着冰冷寒意的天材地宝陡然而出。
<br/>　　他必须抵御下体内的龙炎反噬。
<br/>　　否则，他真的会被活活烧死的。
<br/>　　这株天材地宝，乃是一株皇品巅峰的寒冰果。
<br/>　　果子握于手中，缓解了一丝灼热。
<br/>　　但，充斥着惊人寒冰属性的朱果，却在手中以肉眼可见的速度，不断融化。
<br/>　　仅仅是转瞬间，整颗朱果，被融化殆尽。
<br/>　　嗖…嗖…嗖…
<br/>　　萧逸这些天剿杀邪修，还是能得到不少高品天材地宝的。
<br/>　　此刻，手中一株株寒冰属性，或者水属性的天材地宝，被快速拿出。
<br/>　　只是，其结果，都尽皆是被以极快的速度所融化。
<br/>　　除了给予萧逸减轻些许炙热感外，这一大堆珍贵的皇品天材地宝，并未能起太大作用，便已融化殆尽。
<br/>　　“该死。”萧逸咬了咬牙。
<br/>　　在他的感知中，他乾坤戒内的寒冰属性天材地宝，在这短短时间内，已被悉数耗尽。
<br/>　　而他体内的灼热，仍旧不断加剧。
<br/>　　他的身体，仍旧通红。
<br/>　　他双手，宛若要融化了一般。
<br/>　　龙炎的反噬，霸道而狂猛。
<br/>　　“真的要死在这里了吗…”萧逸的意识，再次愈加模糊。
<br/>　　剧烈的炙热感，不仅仿佛要将他的肉体融化，连他的意识，也在不断被焚烧…
<br/>　　……
<br/>　　六龙城，龙家。
<br/>　　龙家的府邸，无疑是六龙地域内最为广阔，也最为气派的府邸。
<br/>　　此时，家族深处。
<br/>　　龙天的身体，被安放在地面。
<br/>　　周遭，阵法之力环绕。
<br/>　　外头，是龙啸，以及三长老等一众家族长老。
<br/>　　“三长老，此法真能助天儿恢复武道根基？”龙啸皱眉问道。
<br/>　　武者的武道根基被毁，几乎再无可能恢复。
<br/>　　这一点，龙啸深知。
<br/>　　哪怕是天下炼药师云集的药尊殿，哪怕是其中的十大殿主，也决计没有办法。
<br/>　　可现在，他龙家的三长老，却有办法。
<br/>　　“嗯。”三长老点点头，取出一枚丹药，道，“家主，幸好你速度快，早早便回来布置好一切。”
<br/>　　“此丹，乃是绝世丹药，六衍六生丹，即便是绝世强者服之，都有脱胎换骨之效。”
<br/>　　“生死人，肉白骨，不再话下；恢五脏，复根基，亦同样能做到。”
<br/>　　“而且，再加上这六生大阵。”
<br/>　　“阵法一道，结合炼药一道，必能让少家主恢复。”
<br/>　　“只不过，少家主的身体，此时无比虚弱；故药力、阵法之力，不宜猛烈。”
<br/>　　“这点好说。”龙啸说道，“以我的实力，操控阵法之力与药力缓慢进入天儿体内便是。”
<br/>　　“嗯。”三长老点点头，“以域主的实力还有操控力，这一点，自是能做到的。”
<br/>　　“故这些天，便有劳家主操控此阵，还有将药力完美进入少家主体内了。”
<br/>　　“据我猜测，顶多一月左右，丹药之力与阵法之力便能发挥到极致，让少家主恢复。”
<br/>　　龙啸闻言，脸色一喜，“此番有劳三长老。”
<br/>　　三长老摆摆手，“说起来，老夫是家族内三长老；按辈分，天儿算老夫侄孙儿，老夫自该全力救助。”
<br/>　　“呼。”三长老说白，脸色一白，气息有些虚弱。
<br/>　　“三长老…”龙啸一惊。
<br/>　　“无碍，家主放心。”三长老摇了摇头，“只是炼制丹药还有布置大阵，耗费了不少心力，有些疲惫罢了。”
<br/>　　“来人。”龙啸霎时大喝，“开家族宝库，但三长老所需，尽皆应允。”
<br/>　　“是。”周遭长老，恭声应答。
<br/>　　“谢域主。”三长老‘恭敬’地行了一礼。
