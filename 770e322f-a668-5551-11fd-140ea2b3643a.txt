陆寒霆昨晚一夜未眠，到了凌晨的时候才睡着，所以早晨起晚了，等他下楼时，老夫人已经坐在餐厅里吃早餐了。
<br/>
<br/>        “阿霆，你醒了？我发现你最近经常起晚。”老夫人乐呵呵的笑道。
<br/>
<br/>        陆寒霆坐在餐椅上，无视奶奶的取笑，他扭头看向厨房，很快就在厨房里面看到了那道纤柔的身影。
<br/>
<br/>        今天的夏夕绾穿了一件红色毛衣，下面黑色铅笔裤，陆寒霆第一次看她穿红色，大红的颜色衬的她肌肤赛雪，就连纤柔绝丽的五官也更加俏媚，如迎风舒展的一朵蔷薇花，让人移不开目光。
<br/>
<br/>        老夫人见陆寒霆盯着夏夕绾看，她心如明镜的咳嗽了一声，“绾绾，早餐好了吗，我肚子都饿了。”
<br/>
<br/>        “来了！”夏夕绾从厨房里走出来，将一碗面条送到了老夫人的面前，“奶奶，今天吃面。”
<br/>
<br/>        老夫人现在的身体只能吃素面，不过夏夕绾加了鲜美的鸡汤，还有几颗小青菜，老夫人用筷子翻动面条时，那面条的香味都跑了出来。
<br/>
<br/>        陆寒霆坐在对面，当然闻到了面条的香味，他在默默的等待自己的面。
<br/>
<br/>        但是吴妈走出来，送上了牛奶鸡蛋还有精致的糕点，“少爷，这是你的早餐。”
<br/>
<br/>        正在等面的陆寒霆，“…”
<br/>
<br/>        “少爷，怎么了，是今天的早餐不合你的胃口吗？”这时吴妈关心的问。
<br/>
<br/>        陆寒霆看了看老夫人手上的面，又看了看自己的早餐，他蹙了一下英气的剑眉，嗓音低沉道，“没事，很好。”
<br/>
<br/>        老夫人吃了一口面，大赞夏夕绾的手艺，“对了绾绾，你怎么想起今早给我煮面条了？”
<br/>
<br/>        夏夕绾解下了身上的围裙，她看着老夫人眉眼弯弯的笑道，“奶奶，吃面吃面，我希望你长命百岁啊。”
<br/>
<br/>        “绾绾，你这小嘴巴真是越来越甜了。”
<br/>
<br/>        陆寒霆看着对面的夏夕绾，昨晚两个人在书房里不欢而散，现在她还没有看他一眼，一眼都没有。
<br/>
<br/>        他抿了一下薄唇，俊脸的轮廓显得十分立体冷硬，这句话奶奶是说对了，她的小嘴是很甜，他亲过~
<br/>
<br/>        “奶奶，有一件事我想要跟你说，我已经给你施针三天了，后面不需要再施针了，只要调理身体，我会将食谱留给吴妈，从今天开始我就不来这里了，奶奶你多多保重。”夏夕绾道。
<br/>
<br/>        这话刚落下，陆寒霆直接一僵。
<br/>
<br/>        她说什么？
<br/>
<br/>        她说，她不会再来这里了。
<br/>
<br/>        老夫人滞了一下，迅速看了对面的陆寒霆一眼，然后她拉住夏夕绾的小手，叹息一声点头，“好，绾绾，不用担心奶奶，奶奶不想成为你的负担，奶奶一定会乖乖吃饭睡觉的，你走吧，以后你不要为任何人而活，做自己想做的事情，开心就行。”
<br/>
<br/>        “恩！”夏夕绾用力的点头，“奶奶，我回家去了，如果你想我的话，还可以给我打电话，或者去夏家做客，我的爸妈很热情的。”
<br/>
<br/>        “好！”老夫人应下。
<br/>
<br/>        夏夕绾站起了身，她对着老夫人和吴妈挥了挥小手，“奶奶，吴妈，我走了，再见。”
<br/>
<br/>        夏夕绾离开了西苑。
