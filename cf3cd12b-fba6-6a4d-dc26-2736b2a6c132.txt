        &lt;h3&gt;第四卷第十二章  小样，跟我斗？&lt;/h3&gt;
<br/>
<br/>        “爸，杨总让我们去他家吃饭。”江俞轩怕父亲不去。
<br/>
<br/>        “你这级别高啊，蹭饭都蹭到人家家里去了，得了，俞琪，我们今天就给这轩轩一个面子，走一起去蹭饭。”江爸爸傲娇的语气能雷死人。
<br/>
<br/>        “轩轩，你这天天蹭饭就空着手？”江爸爸看着江俞轩安欲言又止的样子。
<br/>
<br/>        “他家什么都不缺，就不让我带东西过去。”
<br/>
<br/>        “带上两瓶酒吧，我们一家三口去人家家里吃吃喝喝，空着手我可不好意思。”
<br/>
<br/>        “听你的！”
<br/>
<br/>        江俞轩带着父亲和姐姐一起去了杨东家，杨东和候玉屏果然都在家里忙碌着准备晚饭。
<br/>
<br/>        “俞轩，你来啦，快进来，这是你爸爸和你姐姐？”侯玉屏笑着说。
<br/>
<br/>        “我们这小子不但经常来叨扰你，今天连我们一家人都来了。”江爸爸笑着说。
<br/>
<br/>        “这小子已经成了我们家一份子了，这要是几天不见他，我们还不习惯了呢。”杨东将江爸爸让到客厅。
<br/>
<br/>        “咳，合着这小子就差和你们姓了？”江爸爸哈哈大笑。
<br/>
<br/>        “我啊，女儿都出嫁了，要不然你的儿子就真和我姓了。”杨东也大笑，江爸爸是一个豪爽大气之人。
<br/>
<br/>        “女儿不女儿的无所谓，既然你喜欢领走便是。”江爸爸笑着说。
<br/>
<br/>        “嗨嗨嗨，老爸，你这刚刚一见杨总的面，就将我送人了啊，你还是不是亲爹？这好歹你也养了快三十年了，养只小狗小猫都有感情了。”
<br/>
<br/>        杨东和江爸爸因为江俞轩的原因，感觉一下子就熟悉了，俩个人抛开江俞轩聊开了，博古论今，就目前的时事新闻聊到环境生态再聊到经济，江俞轩听着俩个老人聊着，偶尔插上一两句话，看着俩人聊得投机，江俞轩偷偷的溜到厨房去看看候玉屏和江俞琪做饭。
<br/>
<br/>        其实江俞琪也就是打打下手，全程有保姆在做，江俞琪就和候玉屏在聊天。按说江俞琪是客人，但是杨东他们确实是把江俞轩当成自家人了，相对的就对江家人也感觉熟悉一些，没有那种疏离感。
<br/>
<br/>        “我知道俞轩还是子昂说起来的，那几年只要子昂回来嘴里就挂着俞轩俞轩，我就很好奇俞轩是一个什么样的人，谁知道后来阴差阳错，杨东也认识了俞轩，不但认识了，还和俞轩成了好朋友，我们也就经常来往。”候玉屏应该是在和江俞琪讲怎么知道江俞轩的。
<br/>
<br/>        “你也认识陈子昂啊？”
<br/>
<br/>        “认识啊，好多年了，记得那个时候她应该是初回西城吧，刚刚回来在西城拓展业务，我问及她在北京的生意，她就说是俞轩在照看着。有一次她告诉我说她好傻，都不知道江俞轩是江家的人，也不知道江俞轩有了女朋友，等知道情况了人家都快要结婚了。”
<br/>
<br/>        “可能她不怎么注意俞轩吧。”
<br/>
<br/>        “也不是，子昂那人吧，其实有时候是死心眼，她认定你了，你把她卖了她都会相信你是为她好，还有啊，她是又骄傲又自卑。明明是那么能干的一个人，可是因为遭遇，让她总是不敢去追求自己的幸福，不敢正视自己的内心，把自己紧紧的包裹在自己的壳里，一个人疼，一个人苦，还要装作坚强。”
<br/>
<br/>        “候姐，你对子昂好了解啊。”
<br/>
<br/>        “我们也是忘年交了，有什么话都会相互说出来，鼓励一下或者是安慰一下，彼此说说心里话也能舒坦一些。”
<br/>
<br/>        “子昂其实是很能干的，像她现在的成就一般人都是遥不可及，我其实挺佩服她的。”
<br/>
<br/>        “是很能干，就是没有一个好归宿，我本来想将她介绍给我家外甥，她却出国了。”
<br/>
<br/>        “你外甥是？”
<br/>
<br/>        “周泽瑞，一会就来了，人品不错，俩个人也很相配。”
<br/>
<br/>        “子昂好像有两个孩子？”
<br/>
<br/>        “有孩子又怎么样？只要俩个人真心相爱，那些都不是问题，况且周家很开明的。”候玉屏叹了口气：“只是可惜子昂出国了，也不知道什么时候回来。”
<br/>
<br/>        江俞轩悄悄的退回到了客厅，静静的坐在杨东身边。
<br/>
<br/>        而江爸爸和杨东聊到了未来信息化的发展，俩个人还潮的不行，从翼支付要云存储到大数据，谈到信息化及互联网对各行各业的发展将会产生什么样的影响。
<br/>
<br/>        “其实我以前是做医疗的，一个医生干了一个不是我专业的事情，让我这把老骨头很是学习了一段时间，现在看看医疗行业的发展，真的是日新月异。”
<br/>
<br/>        “你这跨度确实有些大，无论是那种行业对你来说都不是问题。”
<br/>
<br/>        “刚开始确实是有些困难，摸不着头脑，好的是那个时候年轻，能够安静下来思考和学习，现在的信息化发展很快，工作起来也比之前省心了很多，各个部门对接起来也很轻松。”
<br/>
<br/>        “是啊，我们集团也是跟上了社会发展的步伐，先发展了一步，不过这都是因为俞轩的原因。”江爸爸看了一眼坐在旁边一言不发的江俞轩。
<br/>
<br/>        “说来说去，还是你这个父亲开明，要不是你放手让他出去闯荡，他也只是局限于一个行业的发展。”
<br/>
<br/>        “年轻人嘛，让他们多锻炼锻炼是有好处的。”
<br/>
<br/>        “是啊，还是年轻人有魄力，几个人从一个行业小白鼠几年变成行业龙头，这份聪明才智不是一般人能比拟的，智商情商财商都必须要一流，要有意志力和勇气。”
<br/>
<br/>        “哈哈哈，说真的，我也很欣赏陈子昂那个姑娘，无论是做人还是做生意都无可挑剔。”
<br/>
<br/>        “看来欣赏陈子昂的不是我一个人啊，我们家的那口子这几天天天给她外甥安利陈子昂，呦，西城地方邪，说曹操曹操到，来了，泽瑞，来来来，给你介绍一下，这个是.......”
<br/>
<br/>        “江叔叔好！”周泽瑞笑着和江爸爸打招呼。
<br/>
<br/>        “哦豁，你们也认识？”杨东有些不相信。
<br/>
<br/>        “我和江叔叔有见过几次面，应该是很多年前了。俞轩。”周泽瑞便和杨东说着话，便对看见站在旁边不说话的江俞轩打着招呼。
<br/>
<br/>        “学长，你来了，这个世界真是小，转来转去都转到一起了，原来你和我爸爸也认识。”
<br/>
<br/>        “嗯，江叔叔应该是和我爸爸认识，我们就见过几次面，当时我是和我爸爸在一起的。”周泽瑞和江爸爸打过招呼后，自然的就坐在江俞轩旁边，有着长辈在身边，他们年轻人是不会多说话的，除非是和他们一起讨论问题。
<br/>
<br/>        “学长现在还的单身吗？”江俞轩问周泽瑞。
<br/>
<br/>        “孤家寡人一个。”
<br/>
<br/>        “向学长这么优秀的人为什么还单身？”
<br/>
<br/>        “因为没有找到合适的人啊，你呢？”
<br/>
<br/>        “我有女朋友了，就是那天视频的陈子昂。”江俞轩低声对周泽瑞说
<br/>
<br/>        “啊哈？”周泽瑞一脸愕然。
<br/>
<br/>        江俞轩看着周泽瑞愕然的表情，心里想着：“小样，跟我斗？我可是和陈子昂一起同甘共苦过来的。”
