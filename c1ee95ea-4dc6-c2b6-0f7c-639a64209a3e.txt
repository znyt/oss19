        “我的。”傅夜沉从容自若地回答。
<br/>
<br/>        “是真的吗？”冷太太也是一脸狐疑地看向童筱颖。
<br/>
<br/>        怎么就看不出来她怀孕了呢？
<br/>
<br/>        不过，刚刚看她是穿着平底鞋进来的，应该是真的吧！
<br/>
<br/>        如果是真的……
<br/>
<br/>        她这当妈的，心里还是挺高兴。
<br/>
<br/>        童筱颖却完全没有在意这些小细节。
<br/>
<br/>        傅夜沉微微一笑，温和地抢先回答：“当然是真的！妈妈，您要当奶奶了。昨天一早验出来的，已经去医院里检查确定了。本来想等满了三个月后再告知您和爷爷，谁知道今儿就被爷爷叫过来问候了。”
<br/>
<br/>        瞧他这说得跟真的一样……
<br/>
<br/>        童筱颖却心虚地烫红着脸，现在压根就没任何心思去吃这一桌子的美食。
<br/>
<br/>        傅夜沉见状，睐了童筱颖一眼，会心一笑：“筱颖，别害羞也别害怕。爷爷和妈妈，是开明的人，不会为难你。”
<br/>
<br/>        “是啊！筱颖！以后，我们就是一家人了。”冷太太笑不拢嘴地应道。
<br/>
<br/>        傅炎廷却一脸严肃，不以为然地说：“小沉，筱颖有没有怀孕，家里是有医生的，重新检查一下便知道。不要以为爷爷好骗！”
<br/>
<br/>        “爷爷想另请医生确诊，当然没问题。”傅夜沉淡定从容地回答，顿了顿后，接着说道，“爷爷若是故意命令医生说假话，非得拆散我和筱颖。爷爷，您是知道我的性子的。”
<br/>
<br/>        傅炎廷顿时勃然大怒地拍着餐桌，呵斥道：“你反了！就为了一个女人，而且还是你弟弟的前妻，你竟然威胁你爷爷，你简直大逆不道，你就是个不孝子孙！”
<br/>
<br/>        若是以前，他还可以接受她童筱颖。
<br/>
<br/>        但是，现在，她童筱颖不行！
<br/>
<br/>        因为，童筱颖跟了傅昼景三年。
<br/>
<br/>        三年意味着什么？
<br/>
<br/>        三年的时光可以完完全全地改变一个人！
<br/>
<br/>        傅昼景和童筱颖没有遵守承诺，安安分分地待在澳洲过日子。
<br/>
<br/>        反而一个跟季氏集团的千金季思妍苟合，一个又跟他的长孙傅夜沉纠缠不清。
<br/>
<br/>        他这当爷爷，又怎么能接受这样一个心思不纯的女人当自己的长孙媳？
<br/>
<br/>        “爷爷，您别生气！我是真心喜欢……喜欢大、不，我是真心爱夜沉的。爷爷可以不接受我，但是不要生夜沉的气。爷爷若是真的接受不了我，我可以签协议书，您让我做什么都行，我也愿意无名无分地给夜沉生孩子。爷爷若是不放心，我可以放弃孩子的抚养权。我只求爷爷您不要生夜沉的气，生气对您身体不好。”童筱颖信誓旦旦地允诺。
<br/>
<br/>        她的这番话，让傅夜沉目瞪口呆。
<br/>
<br/>        就连原本在气头上的傅炎廷，也忽然间气消了、释然了。
<br/>
<br/>        不知道为什么，童筱颖说完这番话后，有种想哭的冲动。
<br/>
<br/>        曾经，她也是这样信誓旦旦地在法庭上，向法官和陪审团允诺，她爱昼景，愿意无条件地给昼景生个孩子去救季思妍的那个孩子。
<br/>
<br/>        当时，她是出于爱。
<br/>
<br/>        现在，她同样是出于爱。
<br/>
<br/>        每一次的“爱”，她都爱得好卑微。
<br/>
<br/>        这是最后一次……
<br/>
<br/>        她希望自己再次付出的真心，不会再遭到抛弃。
<br/>
<br/>        这一刻，在场所有的人都沉默了。
<br/>
<br/>        不知过了多久，傅炎廷才开口说道：“既然如此，我让林贵给你们择个黄道吉日，把婚礼给办了。”
<br/>
<br/>        他看在那未出生的小曾孙份上，心软了，也妥协了。
<br/>
<br/>        “我打算就今天把婚礼的过场走一遍，反正只是遵循旧俗，拜堂成亲。”傅夜沉回过神来，淡定道，顿了顿后，又接着解释道，“爷爷，我现在有任务在身，结婚的事情不宜张扬。”
<br/>
<br/>        “筱颖若是没意见，我也就没意见。”傅炎廷还是照看起这孙媳妇的面子来。
<br/>
<br/>        童筱颖慌忙摇头摆手，尴尬地笑着应许道：“我、我没意见。”
<br/>
<br/>        她这是被赶鸭子上架了，还能有什么意见？
<br/>
<br/>        午饭过后，傅夜沉牵着童筱颖在大庄园里散步。
<br/>
<br/>        当童筱颖想问他话的时候，傅夜沉却只是伸出食指比在了她的嘴唇上，微微摇头，示意她什么话也不要多说。
<br/>
<br/>        婚礼的事情，他其实早有计划，只是计划赶不上变化。
<br/>
<br/>        可童筱颖的心里总隐隐地冒着一丝不安，这种不安的感觉，让她无法用语言去形容，而且思绪也有点儿乱。
<br/>
<br/>        “凤冠霞帔、红鸾帐暖，床上铺满红枣、花生、桂圆、莲子，整个地面铺满玫瑰花瓣的洞房花烛，愿得一心人，白头不相离。筱颖，你真的决定好了吗？”傅夜沉拉着童筱颖的手，一派自若地问。
<br/>
<br/>        “诶？”决定什么？
<br/>
<br/>        她刚刚走神了，完全没认真听他所说的话。
<br/>
<br/>        他噙着笑意，温柔地提醒：“愿得一心人，白头不相离。还有洞房花烛夜。”
