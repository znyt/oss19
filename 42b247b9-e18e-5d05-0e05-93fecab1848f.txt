
<br/>　　<br/>　　    易天逸一把火把陆程一的尸体给烧了，这就跟投名状是一个意思，他要表达的潜在内容就是，你看尸体被我烧了我也算你的帮凶之一，咱俩再往下聊的话你就别有压力了呗？<br/>　　<br/>　　    丁武果然把戒备放下了不少，问道：“你什么意思？”<br/>　　<br/>　　    “我要找王惊蛰······”<br/>　　<br/>　　    从易天逸的角度来讲，他对王惊蛰是有求的，两人本来没有什么过节，他为冯天良破易天逸的风水局，说白了就是拿人钱财与人消灾，但没想到在破局的过程中忽然发现王惊蛰居然是赊刀一脉的人，这就给易天逸打开了一扇渴望的大门，他们相字脉太渴望能把残缺的相术补全了，而这非连山和归藏两书不可，恰巧赊刀人就握有这两门奇书，能和王惊蛰坐在一起好好聊聊，比跟他成为敌人要好太多了。<br/>　　<br/>　　    “我要见王惊蛰没有恶意，你可以先给他打个电话，剩下的我来说，和我见或者不见自然是他说了算的，对吧？”易天逸指着被烧焦了的那块地方说道：“这还不足以说明一切么？”<br/>　　<br/>　　    易天逸的手法让陆程一的尸体都被烧没了，就剩了一点灰也随风而散了，这种毁尸灭迹的手段无疑比抛尸荒野靠谱多了，就算警察发现陆程一失踪了现在也只能是给定行程人口失踪了。<br/>　　<br/>　　    丁武深深的看了他一眼，思索片刻后拿出电话开了机，然后给王惊蛰拨了过去，手机刚接通，听筒里面就传来了王惊蛰的破口大骂。<br/>　　<br/>　　    “你他么在哪呢？好，好，和我玩暗度陈仓是不是？我告诉你咱俩掰了，结束了，从此以后老死不相往来，割袍断义，拜拜了您吧！”王惊蛰一顿咆哮，吐沫星子差点就从手机里喷出来了。<br/>　　<br/>　　    电话那头的王惊蛰确实被气的够呛，早上醒来发现他人不见了，一颗心顿时就提到了嗓子眼上。<br/>　　<br/>　　    丁武瞬间尴尬了，指着电话说道：“可能是他刚醒有点起床气，额，稍微等会，让他顺顺就好了”<br/>　　<br/>　　    王惊蛰破口大骂了几句后，心态确实顺了不少，明显脾气也消了，丁武说道：“我在二火葬附近的树林里，有点小意外，你过来一趟吧，有个人想见你······”<br/>　　<br/>　　    丁武瞥了眼易天逸和冯天良，他俩都报了下名号，丁武又重复了一遍后，就问道：“你要是感觉不妥的话，也可以不用过来”<br/>　　<br/>　　    “行了，等着吧”电话里王惊蛰没好气的说道。<br/>　　<br/>　　    挂了电话后，丁武又看向了林羡鱼，这姑娘跟个受惊的小兔子似的，捂着小脑袋摇的跟个拨浪鼓似的：“我什么也没看见，我失忆了······”<br/>　　<br/>　　    丁武无奈的说道：“你就是看见了也没什么，光凭你一个人的口供警方也不见得会信，而且你也未必会蠢得去报警吧，你不是还想让王惊蛰去给你们家当镇宅神兽么？你把我再送进监狱里去，恐怕那小子以后见了你都得绕着走吧？”<br/>　　<br/>　　    林羡鱼放下手，很认真的说道：“可是，可是你刚刚杀了人啊？”<br/>　　<br/>　　    “他？”丁武指了指地下，说道：“你把他当人那他就是个人，不把他当人，说他是畜生也行，我杀他一个可能会避免以后有很多人少受他的祸害了，我这也算是行善积德了吧？”<br/>　　<br/>　　    易天逸点了点头，丁武这话说的没错，这也是刚才他一点都没犹豫就毁尸灭迹的原因，看见地上的陆程一面相时，易天逸就断定出此人属大恶之人，一辈子没少干什么亏心的事，那一脸的横肉就是最真实的写照了。<br/>　　<br/>　　    说一句不算太夸张的话，生活中人的长相是分善恶的，有很多恶面之人你一看心里就有谱了，知道此人不太好惹，这也算是相面的一种了，远的不说就说前一阵那位被反杀了的宝马龙哥，脑瓜皮贴着一层头发，脸上的肉都是横着长的，塌鼻梁圆脑袋，三岁小孩见了都知道这不是好人。<br/>　　<br/>　　    龙哥死了人人拍手称快，同样的，陆程一就属于这一种，这些年他干建材生意也没少坑蒙拐骗。<br/>　　<br/>　　    在树林里等了不到半个小时，王惊蛰坐车就到了，进到林子里后看见一圈人，他皱眉嗅了嗅鼻子，说道：“死了？”<br/>　　<br/>　　    林子里飘荡着一股淡淡的阴气，那是陆程一死后的冤魂还没有散开，他这种死法太憋屈了，说多了都是眼泪啊，可能陆程一都没想到当初就因为贪那几万块钱，让他今天把命都给丢了。<br/>　　<br/>　　    所以有句话说得挺对的，不是不报，是时候没到，你都不知道自己曾经造过的什么孽，然后会在哪一天找上门来。<br/>　　<br/>　　    王惊蛰的眼睛扫了众人一圈，狠狠的瞪了眼丁武后，就跟易天逸说道：“你找我？”<br/>　　<br/>　　    没等易天逸说话呢，冯天良急巴巴的就走了过来，差一点就老泪纵横了：“冯玉林是家父，不好意思了王先生，您当初找我的时候，我父亲真的没有交代过曾经收过一把菜刀的事，是后我从我母亲那里才打听到的，要是知道有这回事的话，我说什么也不会赖账的，还望您能见谅”<br/>　　<br/>　　    “这事也怪不到你头上去，是当年你爸没交代清楚，或者也有可能是没当回事·······”<br/>　　<br/>　　    冯天良顿时一哆嗦，连忙说道：“当回事了，当回事了，我父亲曾经提到过他碰到了个算命先生才让冯家又再起来的，只是菜刀的事他确实没说过”<br/>　　<br/>　　    “好，你的问题过会再说吧”王惊蛰冲着易天逸问道：“你们相字门这么急巴巴的找我，无非就是为了那点事吧？”<br/>　　<br/>　　    易天逸惊诧的问道：“你知道？”<br/>　　<br/>　　    王惊蛰冷笑着说道：“我家老人临走之前送我一句话······防火防盗，防玄门一脉”
