        龙之卫充耳不闻，就这么坐在餐桌前，对桌上的饭菜丝毫不感兴趣的样子。
<br/>
<br/>        诺筱颖拿起一根烤泥鳅，朝龙之卫的嘴巴前递过去。
<br/>
<br/>        龙之卫伸手便将诺筱颖递过来的烤泥鳅给打到了地上。
<br/>
<br/>        外婆见状，连忙捡起地上的那根烤泥鳅，上上下下地吹了吹，然后自己吃了起来，并让诺筱颖重新拿盘子里的干净的烤泥鳅吃。
<br/>
<br/>        “你不吃的话，我就把它们全部都吃进肚子里去了噢！”诺筱颖稚声稚气地说。
<br/>
<br/>        龙之卫看了诺筱颖一眼，低着头，默不吭声。
<br/>
<br/>        晚餐过后，外婆烧了热水给诺筱颖洗澡，让龙之卫也一起过来洗。
<br/>
<br/>        龙之卫不依，外婆便将他抱了过来，直接扒了他身上的衣服，将他放进了诺筱颖所坐的木盆里。
<br/>
<br/>        “呜哇”一声，龙之卫满脸涨得通红，捂住自己的“小弟弟”，委屈得大哭了起来。
<br/>
<br/>        诺筱颖坐在木盆里，却“哈哈”大笑了起来，还不停地朝龙之卫泼水。
<br/>
<br/>        “羞羞羞！男孩子怎么能和女孩子一起洗澡呢？外婆坏！呜呜——外婆坏！”龙之卫委屈伤心地掉眼泪，好像受到了什么天大的委屈。
<br/>
<br/>        外婆气得笑岔：“筱颍才四岁，有什么大不了的！你两一起洗，我省事！”
<br/>
<br/>        “我要一个人洗澡！我要一个人洗澡！”龙之卫捏着小拳头抗议道。
<br/>
<br/>        外婆不得不先快速地给诺筱颖把澡洗完，然后将诺筱颖抱出木盆，留下龙之卫一个人在那洗澡。
<br/>
<br/>        两个孩子都洗完澡后，这爬到床上，龙之卫又拿起枕头和诺筱颖摆起了“三八线”。
<br/>
<br/>        “这边是我的地盘，你不许越界！听到了没？”龙之卫十分霸道地说。
<br/>
<br/>        诺筱颖完全不理会他，抱着外婆给自己缝制的虎头娃娃，躺在床上自顾自地唱歌说话。
<br/>
<br/>        厨房里，外婆还在忙活。
<br/>
<br/>        一盏油灯，照亮了外婆忙碌的身影。
<br/>
<br/>        和面、揉面，让面团发酵，然后擀面，开始做肉包子。
<br/>
<br/>        这是明早，那两个孩子的早餐。
<br/>
<br/>        筱颍最爱吃大肉包，小卫那孩子应该也爱吃吧！
<br/>
<br/>        家里再穷，外婆也会想办法满足这两个孩子成长所需的营养。
<br/>
<br/>        翌日，早上起来。
<br/>
<br/>        外婆带着两个孩子洗漱完后，将冒着热气的大肉包子端上了桌，然后是两小碗羊奶。
<br/>
<br/>        诺筱颖伸手抓起大肉包，小嘴巴张到最大，大口大口地咬着吃了起来。
<br/>
<br/>        龙之卫看着诺筱颖的吃相，摸了摸已经饿得咕咕叫的肚子，终于不再抗拒，快速地拿起盘子里的一只大肉包，捧在手心里，斯斯文文地吃了起来。
<br/>
<br/>        外婆看到龙之卫肯吃东西了，终于欣慰地笑了。
<br/>
<br/>        诺筱颖吃完肉包子，又将那小碗羊奶喝了个精光，并将羊奶碗舔得干干净净的。
<br/>
<br/>        龙之卫不肯喝羊奶，趁着外婆去喂猪的时候，诺筱颖替龙之卫把那碗羊奶给喝了。
<br/>
<br/>        白天的时候，村里若无人来寻医问药，外婆便会带着诺筱颖和龙之卫上山采药。
<br/>
<br/>        龙之卫第一次上山，回来后，那白花花莲藕般的胳膊和小腿上，就被虫子咬出了过敏性的荨麻疹。
<br/>
<br/>        外婆给龙之卫上了草药，消肿止痒后，就不再带龙之卫上山了，而是让龙之卫和诺筱颖一起在山脚下玩，并叮嘱他两，不许到处乱跑。
<br/>
<br/>        山脚的溪水边，诺筱颖脱了鞋，下水去抓鱼，龙之卫就坐在一旁的大石头上，拿着妈妈留给他的小口琴吹了起来。
<br/>
<br/>        龙之卫吹的曲子很简单，但配合着溪水潺潺，空灵而生动。
<br/>
<br/>        诺筱颖搬起石头，将溪水里的鱼围了起来，然后“噗嗤、噗嗤”地在水中扑棱了起来。
<br/>
<br/>        每次，她也就只能抓到一两条，今天她幸运地抓了四条。
<br/>
<br/>        上岸后，这小丫头的裤腿和衣袖都湿了。
<br/>
<br/>        龙之卫白了诺筱颖一眼，只觉这小丫头片子傻乎乎的。
<br/>
<br/>        诺筱颖才不顾龙之卫的白眼，从自己的小背篓里拿出小刀、盐巴、火柴、竹签，然后开始生火烤鱼。
<br/>
<br/>        龙之卫难以置信地看着诺筱颖。
<br/>
<br/>        诺筱颖却洋洋得意地说：“这只是我的小零食而已！待会儿，外婆还会给我带野果子回来吃！”
<br/>
<br/>        龙之卫将小口琴收好，塞进口袋里，然后从大石头上跳下来，蹦到诺筱颖身边，好奇地看着诺筱颖烤鱼。
<br/>
<br/>        他觉得这个很好玩，就像他们城里的小孩子玩过家家一样。
<br/>
<br/>        鱼烤好后，诺筱颖的裤腿和衣袖也都干了。
<br/>
<br/>        诺筱颖让龙之卫去捧溪水过来把火浇灭，龙之卫不依，坐在一旁的石板上愣着不动。
<br/>
<br/>        “你不把火灭了，我就不给你鱼吃！”诺筱颖嘟着小嘴，得意道。
