　　第684章
<br/>　　程大雷的目光不得不落在高力士身上，见他仍旧一脸平和，仿佛刚才的警世之言并非出此他的口中。
<br/>　　莫看他是个阉人，但委实没有那种猥琐谄媚的气质，模样甚至称得上英俊。如此，倒也打破了程大雷的偏见，真如果见人就一脸精明，上赶着拍马屁，也不会讨人喜欢。真正的高手都是春风化雨，润物无声。
<br/>　　而他口中的话，程大雷当真听进去了。程大雷坐在椅子上，不得不认真寻思高力士的话。
<br/>　　程大雷一直忽略了一件事，在这个世道，生存成本是很高的。一个男人，若生在贫寒人家，只能与人做长工，出苦力，勤勤恳恳，辛辛苦苦，也未必能换来一日三餐。最后逼的走投无路，要么去当兵，要么去当匪。
<br/>　　这还是男人，女人更苦命一些。模样普通的倒好说，但凡有几分姿色，要么是被卖入大户人家府中做丫鬟，成了公子哥的玩物；要么便是被富户恶霸看中，强娶强要。这甚至都是运气好的，卖入青楼乐坊，一双粉唇万人尝，两支藕臂千人枕，这种事都算不得稀罕。
<br/>　　生在这样糟糕世道，美丽本就是原罪。
<br/>　　如程大雷所说，让那些家人健在的丫鬟给她们父母引走，这对她们来说，也未必是好事。多一个人便多一张口，多一张口便要多浪费一份口粮，就要有人去做事。艰辛度日，为人缝缝补补，或者又被父母卖到别家，相比之下，哥嫂的白眼都不算什么了。
<br/>　　算起来，她们还不如成为城主府的丫鬟，每月几钱银子，能够贴补家用。和程大雷说的上话，家中有些不平事，也能够有个后台。若某一日被心血来潮的程大雷看中，对她们来说甚至是福气。
<br/>　　虽然荒唐，但却又是事实。
<br/>　　程大雷胸怀善意不假，可善意未必总能做好事。某句话如何说的，通往地狱的路往往是善意铺成的。
<br/>　　而剩下的那些小姐太太，程大雷可以不限制她们的自由，但她们都是十指不沾阳春水的人物，放出去后也不知如何生活。屋里没个男人照应，必然就会成为恶霸歹徒眼中的肥肉。
<br/>　　她们有的在这次战争中死了兄弟丈夫，有的是被丈夫兄弟抛弃，然后转眼就要成为敌人的床榻玩物。这当然不是令人愉悦的事情，可这就是这个世道，如高力士所说，这是她们的命，她们必须认她们的命。
<br/>　　程大雷唏嘘长叹一番，今次才算对身处的乱世有了更深的认知。
<br/>　　他目光看向高力士，高力士也一直在等待他的答案。程大雷无可奈何道：“择几人送到城主府伺候，剩下的人……”
<br/>　　程大雷无可奈何的挥挥手，苦笑着说出剩下的话：“你来处理吧。”
<br/>　　据高力士所想，事情本该如此，程大雷如此为难，却也不知为了那般。
<br/>　　“那我现在便把她们叫过来，由大当家亲自挑选。”
<br/>　　“罢了。”程大雷摆摆手，本来主意是由高力士随便选几人便好，说到这里，他心中忽然有了主意，道：“等下我让夫人来选，至于剩下的人，你和刘悲一起处理。”
<br/>　　让苏樱挑选，自己也好摆明姿态，显示自己昭昭明月之心，并非好色之徒。而刘悲这个人，似乎对女色并不怎么嗜好，他也能限制张肥、高飞虎、高飞豹这帮人，免得他们贪得无厌。
<br/>　　既在其位，便得适应一个位置的风光，程大雷并无力改变什么，也只好随波逐流。
<br/>　　不过，程大雷也不得不承认，至少随波逐流对自己而言，并不是什么坏事。
<br/>　　想到这里，他心中忽然一动，侧着身子问：“那女人现在在何处？”
<br/>　　高力士立刻明白程大雷说的是谁，他皱起眉头道：“大当家，你快快看她去吧，再不解决掉她，属下便要愁死了。”
<br/>　　程大雷口中说的自然就是崔白玉，这女子性子倔的很，被擒之后，三番五次要自杀，也是高力士日夜派人监视，才没让她得逞。但这几日她想起新的花招，开始绝食了。
<br/>　　据高力士所说，她已有两三天没吃东西，再如此下去，怕一条性命便保不住了。
<br/>　　“走，带我去看看。”
<br/>　　崔白玉和其他女人不同，她与随身的三个丫鬟被关在单独的房间，日夜有女兵守着，免得这几人寻短见。而如果换了男人，怕是没办法在崔白玉的美貌面前撑住场子。
<br/>　　程大雷推开门进去的时候，看到四人被绑在椅子上，第一时间，程大雷的视线便被崔白玉夺走。其实，犬鹿刀三个女孩也有些姿色，但在崔白玉身边，会令人觉得，她们天生就该是当丫鬟的命。
<br/>　　几日不见，崔白玉瘦了一圈，婴儿肥的脸显得有些憔悴。不过她明媚时如明月皎皎，憔悴时若梨花带雨，一举一动皆是风情。
<br/>　　就连高力士也不得不感慨：“生得如此祸国殃民，天生便是做贵妃的命呐。”
<br/>　　贵妃？程大雷身边不缺贵妃，他已经有个杨小妹了，虽然说杨家有女未长成，但程大雷并不着急。
<br/>　　见程大雷进来，崔白玉并没有开口，她冷着一张脸，也没有表现得特别愤怒。只是用冷漠表示自己对程大雷的不屑。
<br/>　　程大雷步步靠近，居高临下的俯视着这人间尤物。崔白玉将头扭到一边，似乎不想与程大雷的目光碰撞。
<br/>　　程大雷握住她的下巴，硬生生将她的脸扳过来，俯下身，与其目光对视。
<br/>　　崔白玉不得不盯着程大雷的眼睛，动弹不得。
<br/>　　“听说你在寻死？”
<br/>　　崔白玉保持沉默，冷漠如冰。
<br/>　　程大雷直起身，忽地拔出匹夫剑，寒芒一闪，崔白玉眼中也闪过一道光。
<br/>　　一剑便割开崔白玉身上的绳索，几日不可动弹，早已筋疲力尽，崔白玉不由自主的瘫倒在地上。
<br/>　　锵地一声，程大雷将剑插在地面上，剑柄轻轻颤动。
<br/>　　“暴遣天物，鬼神共愤。可惜本当家本就是个山贼，不敬鬼神，你若不想成为本当家的掌中玩物，就去死吧。”
<br/>(本章完)
