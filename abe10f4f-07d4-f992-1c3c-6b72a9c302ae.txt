        那个新上任的CEO的名字是叫裴金澈吧！
<br/>
<br/>        童筱颖的思绪拉回到起初，他救了她的那件事情上。
<br/>
<br/>        她明明有看到，他护着她的时候，手肘撞到地面上擦出了一片血肉模糊的伤口。
<br/>
<br/>        而且，他居然连眉头也没皱一下。
<br/>
<br/>        更令她疑惑不解的是，那伤口竟然那么快就愈合了，就好像刚刚什么事也没发生过一样。
<br/>
<br/>        童筱颖确定不是自己眼花，但是又无法解释自己起初所看到的一切。
<br/>
<br/>        裴金澈这个男人到底是什么来历？
<br/>
<br/>        大哥能放心地将自己的集团公司交给他管理的话，代表着大哥十分信任他。
<br/>
<br/>        不知不觉中，电梯抵达了设计部所在的楼层。
<br/>
<br/>        童筱颖连忙从最里头挤了出来，去了设计部报到后，又和部长孙莉莉以及同项目组的三个同事一起去了甲方公司会议室参加图纸答疑。
<br/>
<br/>        只是，令她没料到，来参加这次图纸答疑的第三方造价公司，既然是季氏集团旗下的子公司，而且，傅昼景作为他们的代表，亲力亲为。
<br/>
<br/>        大哥不希望她见傅昼景，但是现在这种情况，是公事，应该不打紧。
<br/>
<br/>        会议室内，傅氏集团和季氏集团的人马各占长桌两端，明明只是图纸答疑，一方是设计方，一方是造价方，却隐隐地透着一股浓郁的火药味。
<br/>
<br/>        听完傅氏集团这边设计部部长孙莉莉对图纸的讲解后，傅昼景静默了半天，终于启唇，醇厚如大提琴的嗓音，吐出来的却是不冷不热的话：“为了控制造价成本，既然这些结构只是为了美观，那还是去掉好了。因为，你们的这设计，其实看起来并不美观，反而让人觉得是在故意提高工程造价，好从政府财政手中多拿些钱出来。”
<br/>
<br/>        傅昼景本身就是建筑设计师，自然能看透这些。
<br/>
<br/>        体育馆的结构图，并非她童筱颖设计，而是她最后做的审批，傅昼景看出来的那些问题，其实她在审批图纸的时候，也看出了些猫腻，只是孙莉莉不同意删除。
<br/>
<br/>        孙莉莉是部长，部长说了算，所以，童筱颖没有做太大的强制要求，最后的审批结果，只要保证体育馆的工程质量安全即可。
<br/>
<br/>        为了此事，孙莉莉和傅昼景在会议桌上争论了起来。
<br/>
<br/>        虽然在和孙莉莉争辩，傅昼景的目光却总是有意无意地落在童筱颖的身上。
<br/>
<br/>        早上他是去了一趟医院后，才来这里参加图纸答疑的。
<br/>
<br/>        医生说，他可以不用吃抗排斥的药了，因为移植在他体内的那颗肾，已经跟他的身体完全融合，成为了他体内的一部分了，简直就像是“原装货”一样。
<br/>
<br/>        傅昼景进一步地想要询问原因，医生却只是说，这样的复查结果令他们也觉得很不可思议，甚至觉得有些不大科学。
<br/>
<br/>        而且，医生还给出了他一个最不想听到的揣测，那就是，说不定，给他捐肾的人，跟他存在着某种血缘关系，才会导致百分百地配对成功，以及肾移植后的百分百融合。
<br/>
<br/>        这些，只是医生的个人揣测，但不管真相是什么，总得来说，结果是好的。
<br/>
<br/>        他傅昼景可以不用再吃抗排斥的药物，从今往后，除了少了一颗肾以外，不管哪方面都是正常人。
<br/>
<br/>        孙莉莉最终还是没有争赢傅昼景，他们傅氏集团的设计部被迫回去改图。
<br/>
<br/>        散会后，孙莉莉带着一肚子的火气离开。
<br/>
<br/>        童筱颖打算去坐公交车，结果被傅昼景开车拦住了去路。
<br/>
<br/>        “筱颍，上车，我送你。”副驾驶座的车窗落下，傅昼景温和地说道。
<br/>
<br/>        童筱颖站在路边瞥了傅昼景一眼，淡漠地谢绝：“谢谢，傅总，您还是去忙您的吧！我自己坐车就行了。”
<br/>
<br/>        “筱颍。”傅昼景将车停稳，从驾驶座上下来后，踱步来到童筱颖的跟前，伸手便抓住了童筱颖的手腕，“让我送你。”
<br/>
<br/>        “真的不需要！”童筱颖拧着眉头，扭动着手腕，却没有傅昼景的力气大，完全挣脱不了他的束缚。
<br/>
<br/>        傅昼景几乎是连拖带拽地将童筱颖塞进了副驾驶座内，替她绑好了安全带。
<br/>
<br/>        童筱颖刚想趁机逃走，他大手一挥，捧住了她的脑袋，跟他的额头抵在了一起。
<br/>
<br/>        “筱颍，我去医院复查了。难道你就不想知道我的复查结果如何吗？难道你就不想知道，我还能活多久吗？”傅昼景黯然神伤道。
<br/>
<br/>        童筱颖微微垂了垂眼帘，冷冷地回答道：“你的事，跟我没关系。”
<br/>
<br/>        “就算是我死了，你也不会再为我心痛了吗？筱颍，我只是身体上背叛了你，而感情上，是你背叛了我。”傅昼景压低了嗓音，深呼吸着，努力抑制住自己胸腔里那股猛烈的悸动。
