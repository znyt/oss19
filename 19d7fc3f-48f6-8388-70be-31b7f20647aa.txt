　　在梳理永恒岛地脉龙脉之时，元始发现它与另外两个地方有牵连，元始猜测，另外两个地方是东海三仙岛中的另外两岛——方丈与瀛台！
<br/>　　循着三岛之中的那一丝联系，元始开启了前往另外两岛的路途。
<br/>　　跨过重重海洋，元始来到另外一处岛屿，它同样不与洪荒在同层一空间内，而是与蓬莱岛在同一层空间。进去很方便，元始并没有花多大劲就进去了，因为三仙岛的阵法是连在一起了，三个仙岛按三才之阵排列，都具有遮掩天机之效。
<br/>　　轻易破开大阵，元始进入其中，入眼所见，便是同样的一块石碑，石碑之上同样的平凡中带有不平凡，上面刻有“方丈”二字。元始知道，此仙岛应该就是方丈岛了。
<br/>　　岛内混沌之气弥漫，一样的浓郁，元始想了想，还是祭出因果莲台，将其中的混沌之气吸收殆尽，同时，也显现出了这里面的浓郁灵气。元始布下阵法，接引洪荒之中的先天灵气，进入其中。
<br/>　　进入岛中，与蓬莱一样，同样景色怡人，奇花异草遍地是，怪石神树随处见，灵鹤飞舞，翠竹欲滴，松鼠跳跃……
<br/>　　元始并没有收集这其中的灵根灵草，而是同样绕着仙岛查看一遍。让元始惊讶的是，这岛中央居然同样有顶级灵宝。而且居然还是白色莲花。
<br/>　　方丈岛中央，有一个池塘，池塘之中有巨大莲花，莲花通体白色，而且最中央还有一座十二品白莲。
<br/>　　“净世白莲？！”元始难以置信道。
<br/>　　要说是造化青莲、灭世黑莲、功德金莲与业火红莲，元始就算再惊讶也不会如此惊叹，因为混沌青莲破碎本来就是化为这四种莲台，但净世白莲又是什么鬼？
<br/>　　元始忍住心中的疑惑，继续查探，十二品白莲的四周还有两朵十一品白莲以及许多九品、六品、三品等级的白莲，莲台之下全部是三光神水。
<br/>　　这些可全部都是宝贝啊，十二品莲台是极品先天灵宝，十一品与九品为上品先天灵宝，而六品莲台为中品先天灵宝，就算是三品莲台，都是下品先天灵宝。这些可都是好东西啊，就算自己用不到，拿来炼器也是极好的。
<br/>　　而三光神水更是好东西，三光神水为日、月、星三光，单一则为毒药，若是合在一起，那就是造化啊。三光神水蕴含浓郁的生机，是培育灵根灵植最好的神水，也是水系神通者最爱好的神水，还可以疗伤，创造神通，妙用多多啊！它虽不为十大真水之一，但更胜十大真水。
<br/>　　十大真水分别为：
<br/>　　玄冥真水、天一真水、一元重水、万化雷水、血河真水、宙光真水、太阴真水、五行神水、无定黑水、三千弱水。
<br/>　　十大真水各有妙用，不可多得。
<br/>　　元始再仔细探查，终于发现了不一样的东西，在三光神水深处，元始发现了一具残破的尸体。
<br/>　　“光明魔神？！”元始再次吓一大跳。
<br/>　　“原来如此，原来如此！”紧接着，元始仿佛明白了什么。
<br/>　　“原来这不是混沌青莲子所化，而是残破的混沌青莲本源吸收了光明魔神残破的本源，进化成了净世白莲。”元始恍然大悟。
<br/>　　不管怎么样，先收起来再说，元始将净世白莲连同三光神水全部收了，就连光明魔神残破的尸体也不放过，虽然其本源已失，但毕竟曾经是混沌魔神，尸体之中还是很有价值的。
<br/>　　收完之后，元始再次施展神通，将此地填平，再移植一些顶级灵根过来，将此地布置完美，相信经过时间的沉淀，没有人能够看出什么。
<br/>　　之后，元始来到最后一座仙岛，不出所料，这最后一座仙岛为瀛台岛，与前两座仙岛一样，美轮美奂，仙气纵横，灵气四溢，花香满岛。
<br/>　　元始在其中发现了一颗先天五行果树，这又是一棵不下于先天十大灵根的顶级灵根，蕴含五行之道，不过，元始这一次并没有将它移走，而是任由它继续在仙岛上。
<br/>　　元始布下大阵守护，并不怕被别人取走。
<br/>　　之后元始再次回到永恒岛玉虚宫，进入闭关状态，忙了这么久，是时候参悟生死石印了。
<br/>　　来到悟道台，元始最先取出的并不是生死石印，而是花费千年时间，将十二品净世白莲炼化。白莲化为一道流光钻入元始的太上无极元始庆云之中，元神之中的种种欲望，渐渐消散，心中涌起无限温馨，再也没有之前的压抑，心底彻底平静下来，发挥出真正的作用。
<br/>　　这十二品净世白莲不仅可以净化心中杂念，助人心境平和悟道，还能破除一切邪恶之念。对付黑暗法则，邪恶之人，这净世白莲最好不过了，就算准圣强者都会被克制，发挥不出全力。
<br/>　　元始心中恢复到朦朦胧胧、无欲无求，如同婴儿一般的状态，取出生死石印，用元神慢慢炼化。
<br/>　　一边炼化，一边参悟这其中的生死奥妙。全身心的投入到炼化之中，心底彻底平和，如同一面镜子，炼化速度加快，元神境界也在快速提高，生死的奥妙，尽显心中。
<br/>　　时间流逝，这样的状态，一直持续了数十万年，元始大多状态是在参悟法则，至于炼化生死石印，那只是顺带的。
<br/>　　“嗯？”
<br/>　　元始睁开双眼，眼中闪过黑白两色玄光，黑色玄光所至，树木快速凋零衰老，白色玄光所至，树木又渐渐生出芽孢，恢复生机。
<br/>　　“生死的奥妙，果然令人沉醉。”元始眯上双眼，感慨道。
<br/>　　恢复常态后，元始看着已经被自己炼化的生死石印，呢喃道：
<br/>　　“原来这是一个破碎的世界所化的啊，难怪昆吾老祖那个混蛋没有炼化，这还得多感谢他啊，要是他完全炼化了，那个时候死的就是我了。
<br/>　　不过，就算他想炼化估计也没有办法吧，要不是此次我有净世白莲相助，使心境完全平衡，要不然我也是没有办法炼化它。生死完全平衡，差之毫厘，就会遭到反噬啊。生死法则所化的世界，威力还真是强悍啊。”
<br/>　　“不过，这次收获也非常巨大。阴阳生死印！”
<br/>　　随着元始一声低喝，虚空顿时出现一个黑白太极印，与之前的阴阳印在外观上并没有太多区别，但其实质他早已变了。一个是阴阳二气，一个是生死二气，早已不同。
<br/>　　阴阳生死印缓缓旋转，仿佛能够吸收天地间一切生机，转化为死气，而周围树木，一旦接近死气，便会化为飞灰，生死互转，玄妙无比。
<br/>　　“呼——”
<br/>　　散了阴阳生死印，元始长呼一口气，站了起来，看向远处。
<br/>　　“出来这么久了，是该回去一趟了！”
<br/>　　元始轻声呢喃，语气中充斥着一丝淡淡的思念。
