　　叶少阳仔细看过去，见慧能貌不惊人，看上去就是个普通的老和尚，站在距离台阶最近的地方，低着头，双手合十，好像在念经。
<br/>　　在他面前，又不少几个和尚盘膝而坐，应该都是他的弟子。
<br/>　　老和尚拉着他，又给他介绍了好几个牛逼人物，叶少阳走着看着，不觉中就到了宫殿前面，突然一个藏僧挡住去路，冲老和尚合十说道：“老师兄这是来做什么？”
<br/>　　“带个人去见尊师。”老和尚也很随意。
<br/>　　那藏僧打量了叶少阳一眼，皱眉道：“道士？”
<br/>　　叶少阳也打量他，见他有三十来岁，生的唇红齿白，十分俊俏，身上带着浓浓的仙气，一看就不是普通人。
<br/>　　“是是，尊师召他来的，你别管了。”
<br/>　　那藏僧退到一边去。
<br/>　　叶少阳快步追上老和尚，低声道：“这高僧是谁？”
<br/>　　“他是仓央。”
<br/>　　仓央……
<br/>　　叶少阳猛然想到了一个名字，忍不住回头看了他一眼，那藏僧也回头看他，微微一笑。
<br/>　　果然是帅得很啊，不枉他到今天还有那么多粉丝迷妹。
<br/>　　都是传奇人物啊！
<br/>　　台阶一共九十九层，叶少阳上去的时候特意数了，猜测应该是隐含着九九归真的象征意义，走完台阶，大雄宝殿就在面前了。
<br/>　　叶少阳回头看了一眼，这法诀寺的范围巨大，两边几乎一眼望不到边，要真是比的话，人家不可能有这么大的寺庙，甚至一百个加起来也没这么大，而且从台阶上俯瞰下去，万头攒动，人山人海，心中感到强烈的震撼，心想能来这里的，在人间至少也都是一流高手了，更不要说慧能、仓央这样的顶级强者，而且，这还只是法诀寺的入室弟子……
<br/>　　叶少阳隐约有些明白，为什么地藏菩萨可以在阴司与酆都大帝平起平坐了，说什么道佛合作，不存在的，道佛自古就不是一家，合作是可以合作，但根本上还是互相看不对眼的，因为牵涉到信仰问题。
<br/>　　地藏菩萨能跟酆都大帝掰腕子，大概靠的就是实力，自己的实力，还有宗门的实力。只不过相对大帝那一门来说，他大部分弟子都在这无量界法诀寺，给人的感觉好像没什么实力似的，要不是今天亲眼见到，叶少阳也不敢相信法诀寺的弟子居然这么多……
<br/>　　不过，问题是，为什么自己会在这里？地藏菩萨找自己又有什么事情？
<br/>　　想到自己之前被魑魅魍魉暗算的事，叶少阳很有点担心，但反过来一想，这都到法诀寺了，别说自己这样的，就算是证道强者也未必逃得出去，干脆就什么都不管了。
<br/>　　抬头看着大雄宝殿，两侧的立柱上也有衣一副对联：
<br/>　　佛法无边广渡众生须渡我，世海多波不念地藏更念谁。
<br/>　　这对联……口气比之前那一幅还大，有点舍我其谁的意思了。边上还有一副对联，叶少阳刚要细看，被老和尚拉拉扯扯地拽进了大雄宝殿。
<br/>　　当面就是一尊巨大的装了金的佛像，双手并拢，俯瞰着下方。
<br/>　　叶少阳急忙撕开袖子，在头顶围了一圈，一块布从面前垂下来，这是道士无可奈何之下进了寺庙、见到佛像的应急手段，将天庭裹起来，不被佛像的眼睛看到，也就不会被佛光入侵。
<br/>　　“你呀，哈哈！”老和尚笑嘻嘻地点着他，自己去上了香，带着他绕过大雄宝殿，往前还是一段汉白玉的台阶。
<br/>　　道路两边长的都是菩提树，枝繁叶茂，层层叠叠，叶少阳心中奇怪，不是要带自己找地藏菩萨吗？为啥又从大雄宝殿出来？
<br/>　　一直走到山峰前面，叶少阳看到了一幅奇景：
<br/>　　山峰前面、正对着大雄宝殿的方向，有一尊巨大的佛，至少有上百米高，脑袋与山峰平齐，一只手抬起来，另一只手放在膝盖上，眼睛闭着。
<br/>　　叶少阳抬头辨认了一下，这尊大佛，应该不是三位佛祖，而且有头发的，在脑后结了个发髻，看表情慈中带严，悲天悯人。
<br/>　　又因为佛像巨大，人站在下面，不免有一种自己十分渺小的感觉。
<br/>　　“大师，这佛像……是地藏菩萨？”叶少阳收回目光，寻找老和尚，这才发现老和尚不见了，四下正在张望，突然身后响起一声佛号：“我佛慈悲，叶少阳，你却是来到了这里。”
<br/>　　叶少阳猛然转身，见是一个长眉细目的老和尚，长得跟寿星似的，身穿五色斑斓的袈裟，头戴莲台法冠，一只手拿着佛珠，另只手竖在胸前，面带微笑地望着自己。
<br/>　　“你是……”叶少阳一下子懵了。
<br/>　　大师笑道：“你来了我的道场，却不知道我是谁么？”
<br/>　　地藏菩萨！
<br/>　　叶少阳惊呆，转头看着佛像，又看看老和尚，一点也不像，正要询问，老和尚似乎明白他的心意，笑道：“人之相貌，只是虚妄之观，皮囊之相，何必在意。”
<br/>　　果然是地藏菩萨了……
<br/>　　叶少阳猜测，自己看到的应该是他的一个分身，心跳的厉害，当下躬身一拜，道：“见过菩萨。”
<br/>　　自己还是第一次见到这种超级大佬，说不紧张那是假的。
<br/>　　地藏菩萨一笑，只是看着他，也不说话。
<br/>　　叶少阳被他看得心里有点发毛，正要看口，地藏菩萨转过身，对叶少阳招手，叶少阳只好过去，走到他身边，顺着他的目光看去，从这里可以直接看到山下的法诀寺，和远处的平原风光，脑海中浮起两个字：壮观。
<br/>　　也只有这个词可以形容眼前看到的画面了。
<br/>　　“叶施主，你觉得我这地方如何？”地藏菩萨声音苍劲雄浑，语调倒是正常。
<br/>　　“好地方啊。人间没有这样的好地方。”叶少阳诚恳地说。
<br/>　　“既如此，叶施主可愿留在这里？”
<br/>　　叶少阳大惊，摸不准他这话是什么意思，想了一下，道：“我是道士。”
<br/>　　“道士……什么是道？”
<br/>　　跟自己盼论道？
<br/>　　叶少阳这么想着，嘴上说道：“道可道，非常道。”
<br/>　　地藏菩萨点头，“既如此，那道佛又有什么区别？”
<br/>(本章完)
