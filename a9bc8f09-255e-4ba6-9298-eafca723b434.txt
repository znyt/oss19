夏夕绾以为自己会开心的，但是她一点都不开心，她环顾着这偌大的夏家别墅，这里有她最美好的儿时时光，那时的这里还充满了明媚的阳光和欢声笑语，现在却成为了冷清的祭奠场。
<br/>
<br/>        她想，没有人是赢家。
<br/>
<br/>        夏夕绾推开了夏妍妍的房间，如果可以的话，她永远不想涉足这里，因为她的陆先生曾经跟夏妍妍在这里滚过床单。
<br/>
<br/>        只要想到这个，她的心里还隐隐的刺痛。
<br/>
<br/>        不过，她必须来，因为她要找回妈咪的医典。
<br/>
<br/>        夏夕绾打开了夏妍妍的抽屉，在最下方的抽屉里，她找到了那本医典。
<br/>
<br/>        坐在椅子上，夏夕绾伸出小手翻开了医典的第一页，瞬间湿润了眼眶，因为她看到了妈咪的字，这是…妈咪留下来的文字。
<br/>
<br/>        噼啪一声，一滴晶莹的泪珠砸落了下来，迅速在纸张上晕染开。
<br/>
<br/>        绾绾，慢点跑，小心摔跤。
<br/>
<br/>        我家小绾绾真漂亮，真是妈咪的小公主。
<br/>
<br/>        夏夕绾的耳畔传到了很久远的声音，妈咪抱着她，她在妈咪又香又软的怀抱里肆意翻滚，尽情欢笑。
<br/>
<br/>        夏夕绾捧着这本医典，抱在自己的怀里，她低着脑袋，在妈咪的身边抽泣哽咽，妈咪，你在哪里，绾绾想你了。
<br/>
<br/>        夏夕绾太伤心了，以至于她没有再去翻抽屉，其实在抽屉的最里面放着那块她曾经丢失的玉佩。
<br/>
<br/>        ……
<br/>
<br/>        夏老爷子葬入了墓岭，那天天空都下起了小雨，夏夕绾穿着一身黑衣打着黑伞，静静的在老爷子的墓碑前站了一天。
<br/>
<br/>        这时身后传来了脚步声，有人将一束白花献到了墓碑前，夏夕绾抬头，竟然是久未露面的苏希。
<br/>
<br/>        夏夕绾很久没有看到苏希了，其实这些年苏希也很少露面，好像她回海城之后，苏希才紧跟着回来海城的。
<br/>
<br/>        这些年，他好像都不在海城。
<br/>
<br/>        苏希站直了身，风雨之中，他撑着一把黑伞跟她并排而立，“绾绾，你做到了，十一年前的事情完美落下了帷幕。”
<br/>
<br/>        夏夕绾戴着黑手套，清纯的乌发用皮带低低的束着，耳畔插了一朵小白花，绝丽的小脸安静而伤悲，她没有看苏希，只是小声道，“我还有一个疑惑，十一年前你为什么站出来做伪证，说是我推爷爷下楼的，你跟夏妍妍她们是一伙的？可是，我派人调查过了，你跟她们私下毫无联系，你的手脚未免也太干净了。”
<br/>
<br/>        苏希勾唇，“绾绾，我做这些不过都是听命行事。”
<br/>
<br/>        “听命？你听谁的命？”夏夕绾看向苏希。
<br/>
<br/>        苏希也看着她，“当然是听你妈咪的命。”
<br/>
<br/>        夏夕绾拧眉，“我不懂。”
<br/>
<br/>        “你妈咪出事之前曾经留话给我，如果她有任何意外，让我斩断你与海城的一切联系，把你即刻送往乡下，所以我顺水推舟，帮了李玉兰她们一把，然后又找了一个算命的先生，告诉夏振国你命硬，其实，夏家的事情我毫无兴趣，我只听命于你的妈咪。”
<br/>
<br/>        夏夕绾惊住了，她从来没想过真相是这样的。
<br/>
<br/>        她以为一切水落石出了，但是现在竟然更加扑朔迷离。
<br/>
<br/>        夏夕绾发现自己从来没有看懂过自己的妈咪，甚至是眼前这个英俊的男人苏希。
