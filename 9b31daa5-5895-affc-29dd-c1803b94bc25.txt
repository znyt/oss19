　　这第五个视频就不再是之前那些18X限制级内容了，而是仙岛夫进行“超级士兵”活体试验的过程记录。
<br/>　　虽然这段视频中没有先前那些别说是对儿童不宜，就是一些成人都难以接受的大尺度限制级内容，可是视频内容却堪比这些大尺度限制级内容。
<br/>　　视频应该是多段拍摄后重新剪辑而成，它记录了一个正常人从接受“超级士兵”试验开始，到接受实验失败，最后被仙岛夫活体解剖取出内脏进行研究的全过程。
<br/>　　看到这段视频，让云熙不由自主的想到了那个臭名昭著的“731部队”。云熙没有经历过那段黑暗的岁月，可是从自己爷爷口中，还有军方所保存的一些资料中，还是对“731部队”有一定的了解，炭疽、鼠疫、真空试验、冻伤试验，那些资料中很多东西实在太触目惊心，让当时的云熙在看过后，一度没有任何胃口。
<br/>　　这段视频总共有2个小时，仙岛夫在视频中对所有试验体都做了编号，视频最后一个试验体的编号是“10”号。
<br/>　　但罗昊知道，被用作“超级士兵”试验的试验体绝对不止视频中的10个，剩余的那些试验体，要么没有被拍摄下来，要么被拍摄下来了，可是却被删除了，没有被恢复过来。
<br/>　　视频文件一共就这5个，剩下的是一些用霓虹国文字加上繁体汉字命名的文件夹，罗昊没有费力的去猜测这些文件夹名称是什么，反正都是要打开的，猜不猜名字并没有什么关系。
<br/>　　罗昊所打开的这个文件夹中，有好几个文件，同样的，全都是用霓虹国文字进行命名，这次连繁体中文都不见了，罗昊对扭扭曲曲跟蚯蚓一样的霓虹国文字那是一脸懵，根本看不懂那是什么意思。
<br/>　　“潜能激发药水。”
<br/>　　旁边，云熙突然把文件名字给翻译了出来。
<br/>　　“你懂霓虹国文！？”
<br/>　　罗昊一脸诧异地看着云熙。
<br/>　　“我从没有说过我不懂啊。”
<br/>　　云熙表现的一脸无辜，从罗昊手里拿过鼠标，把文件打开，直接看了起来，说道：“你又没有问过我。”
<br/>　　我靠！
<br/>　　罗昊眨巴着眼睛，更加懵逼了。自己这老婆也太强大了吧，你英语专业八级也就算了，居然还懂霓虹国文，你怎么就不上天呢？
<br/>　　“你霓虹国文什么水平？”罗昊对云熙问道。
<br/>　　“正常听写、阅读、翻译没有任何问题。”
<br/>　　云熙的眼睛依旧盯着电脑上的文件，仔细地阅读着。
<br/>　　“翻译能到什么水平？”罗昊问道：“能不能达到同步翻译？”
<br/>　　云熙美眸卸了罗昊一眼，又把目光回到电脑的文件上，直接打消了罗昊心中的邪恶念头，说道：“同步翻译是不存在问题，但是你别指望我能帮你做那事儿！不可能！没商量！”
<br/>　　“我也就随便问问。”罗昊小声嘀咕了一句，看向电脑上的文件。
<br/>　　云熙“嘁”了一声，显然是对罗昊的说辞不相信，结婚那么多年，云熙还能不知道自己身边这货是什么样子的？正经起来的时候，的确无愧于“龙刺”这个称号；可一旦不正经起来，那就是个流氓，简直就是给他的“龙刺”徽章抹黑。
<br/>　　那份名为“潜能激发药水”的文件，通篇都是霓虹国文字，中间偶尔夹杂着几个繁体汉字，可是这并不能帮助罗昊弄明白文件内容，除了能看懂文件上那几个化学式外，罗昊觉得自己就像是在看天书一样。
<br/>　　尽管看不懂文件内容，可是通过对文件名的猜测，罗昊也是对这份文件的内容有了一些猜测。这份文件很有可能跟“暗月”研发的“恶魔药剂”有关系，罗昊跟“恶魔药剂”打过好几次交道，如果不是因为它那种可怕的后遗症，这的确是种能在短时间内提升人战斗力的神奇药剂，就算是一个普通人，在使用了“恶魔药剂”都能变成一个大力士，当年的钱天磊在使用了“恶魔药剂”后，借助单兵身甲能跟罗昊和云熙两个人打得难分难解。
<br/>　　“这个潜能激发药水，就是之前我们所接触过的‘恶魔药剂’。”云熙看完了文件内容后，告诉罗昊说道：“但是它跟原本的‘恶魔药剂’又有所不同。仙岛夫是参与到‘恶魔药剂’研究的人员之一，他知道使用了‘恶魔药剂’后的弊端，所以他针对‘恶魔药剂’强大的后遗症进行改进，而且也取得了一定的成果。这种经过他改进，被他命名为潜能激发药水的东西，消除了‘恶魔药剂’近百分之八十的副作用，但同样的，这也减少了药剂本身能带给使用者的效果，它的实际效果只有原来‘恶魔药剂’的百分之四十。”
<br/>　　“文件上还有仙岛夫做的记录。”云熙用鼠标指着一段红色字体，说道：“仙岛夫称，‘暗月’高层对他改造后的‘恶魔药剂’并不满意，他们认为仙岛夫把‘恶魔药剂’效果削弱过多。但是上面并没有提及仙岛夫有没有把这种药剂交给霓虹国方面。”
<br/>　　“应该没有。”
<br/>　　罗昊摇头道：“在仙岛夫逃回霓虹国后，我和蛰龙跟他们交手过，但是并没有发现他们使用这种激发人体潜能的药水，如果霓虹国方面掌握了这种药水，不可能不会使用。”
<br/>　　“这倒是有些奇怪。”云熙摇头道：“仙岛夫居然会把这样的东西藏着没有拿出来，仙岛夫不是应该把这种东西上交给他的国家吗？”
<br/>　　“当时仙岛夫带走了‘超级士兵’的研究数据，以及被改造过的权天使，仙岛夫认为自己是可以完成‘超级士兵’研究的，所以并不需要这种药水。”罗昊说道：“当然，还有一种可能，那就是仙岛夫自己也对这种把效果削弱了百分之六十的药水感到不满意，或许他想要把药水完善后再进行使用，只是他再也不会有机会完善他的研究了。”
<br/>　　罗昊盯着电脑上的文件，陷入了思考中……
